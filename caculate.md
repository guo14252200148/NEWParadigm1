# 基于充足理由的逻辑操作门

## 1. 导言

本文在 `充足理由单子论.md` 已建立的逻辑基础上，抽取一组最小逻辑操作门，并对其进行形式说明、有限枚举与结构解释。本文的任务不是另立一套独立门论，而是在不引入额外连续结构、不诉诸集合论与实无穷的前提下，把该基础中的若干第一性要求压缩为有限可操作的门网络。

本文的目标可以概括为四点：

- 说明门层为何能够从 `充足理由单子论.md` 的充足理由基底中被抽取出来；
- 说明四个基本门为何具有现在这一确定形式，而不是任意给定；
- 说明这组门为何在固定元数约束下仍然足以形成函数完备与有限演算；
- 说明这组门如何联结全局决定结构与局部截面读出，并由此解释其与莱布尼茨式“让我们计算吧”的关系。

本文不把“截面”与“概率”直接写入门定义。门层只处理判词状态的离散变化；权重层负责把全局闭合结构与局部读出位置联结起来；概率只出现在局部读出后的有限归一阶段，不属于门层自身。

本文固定下列对象语言：

```text
Sec(E,κ)       : 结果 E 在角度 κ 下的完整理由截面
Path(E,γ)      : 允许回溯路径词序
Head(E)        : 候选头部词
δ_R(p)         : 路径 p 是否被理由头部 R 吸纳
Ω_{E,γ}(R)     : 理由头部 R 的理由级支持量
Read_Σ(z)      : 复支撑的实读出
Q_{i,k}, Q_i   : 轴层净支持与总净支持
P(R_i|E,γ)     : 候选头部的有限归一读数
```

本文只讨论这条链中的门层位置，而不重写后三层定义。

本文中的 `1/0` 也不表示普通布尔真假，而表示判词在充足理由语义下的两种状态：

- `1` 表示判词处于 `Full` 可接入态，即该判词当前理由充足；
- `0` 表示判词处于 `Zero` 可接入态，即该判词当前理由不充足。

因此，全文中关于 `OR`、`NOR`、`NOT` 等术语的使用，仅表示输出表层面的同型关系，而不表示此处语义退化为普通布尔代数。

### 1.1 基底约束

门层之所以可以从 `充足理由单子论.md` 中抽取出来，是因为其中已经给出了若干足以约束门层的第一性条件。对本文而言，最关键的是以下四组约束。

第一，判词约束。`充足理由单子论.md` 已把所有 `0-1` 输出统一解释为判词状态，因此门层中的输入与输出都必须理解为判词状态，而不是抽象真假。

第二，第一性极简核约束。`充足理由单子论.md` 将系统压缩为 `K1-K4` 四条根式：

```text
K1 : 存在 = 被充足规定
K2 : 同一 = 有效理由结构同一
K3 : 生成 = 权词沿正贡献结构汇聚
K4 : 守恒 = 相关闭组中的满权不失
```

就门层而言，这四条根式至少带来以下要求：

- 门层不能把结果理解为脱离理由结构的任意翻动；
- 门层必须区分“正贡献仍可继续”与“整体已经合法闭合”这两类不同动作；
- 门层中的局部离散变化必须能够回接到全局权词汇聚与相关闭组守恒。

第三，层级约束。`充足理由单子论.md` 已明确指出：

- 截面不是第一性原符；
- 概率不是第一性原符；
- 局部动态不是另一种本体，而是全局结构在有限展开中的读出次序。

因此，门层不能把截面与概率直接塞入门定义，而只能先写判词变换规则，再通过更高层的权重与读出机制将门层接回全局结构。

第四，复制约束。`充足理由单子论.md` 已把复制理解为概念层等价位的引入，而不是本体对象的机械增殖。因此本文中的复制必须满足：

- 复制保留同一性；
- 复制引入新的可操作位置；
- 复制动作自身具有充足理由；
- 复制不以输入对象已经 `Full` 为成立前提。

### 1.2 门层抽取原则

在上述基底约束下，门层的抽取必须遵守以下原则。

第一，固定元数原则。单个门的输入数在操作过程中不得改变；一元门只能把单枝送入双枝位，二元门只能把双枝收回单枝位。

第二，判词闭包原则。门层的输入与输出必须都保持在 `Full/Zero` 桥上的有限判词词表之内。

第三，非退化原则。若一个候选门在所有输入上都塌缩为常态输出，则它不应被视为最小原门，而应视为可由更基本门复合导出的退化情形。

第四，传播与闭合分离原则。由 `K3` 与 `K4` 可知，正贡献的继续延展与多枝的合法闭合不是同一件事，因此门层至少必须保留两类不可混同的二元动作。

第五，显式复制原则。同一对象若要进入多个输入位，这种多位调用不能由书写习惯暗中替代，而必须由系统内部一次显式操作给出。

在这些原则下，本文抽取如下四个最小门：

- `L_s` : 单枝到双枝的对偶分裂；
- `P_d` : 单枝到双枝的同态复制；
- `L_m` : 双枝到单枝的严格闭合；
- `P_f` : 双枝到单枝的可达传播。

## 2. 基本记号与基本门族

### 2.1 判词与反值记号

本文固定如下判词值：

```text
1 : 判词处于 Full 可接入态
0 : 判词处于 Zero 可接入态
```

并定义反值记号：

```text
opp(1) = 0
opp(0) = 1
```

### 2.2 一元门的推出

先考察一元门。设一元门接收单一输入 `x`，并把结果送入双枝位。根据固定元数与充足理由约束，该门必须满足两点：

- 它不能凭空制造新的对象差别；
- 在只有一个输入源的情况下，左右两枝之间没有额外理由被区分。

因此，一元门不可能直接输出反对角双枝：

```text
(0,1) 或 (1,0)
```

因为这两类输出要求门在左右两位之间额外制造不对称，而单输入条件下不存在支撑这种不对称的充足理由。

若进一步要求该门对输入 `0/1` 仍保持响应，而不退化为常双枝，则非退化的一元门只有两类基本可能：

```text
(x,x)
(opp(x), opp(x))
```

常双枝

```text
(0,0)
(1,1)
```

虽然也满足左右同态，但它们不再保留输入差别，因此不应被列为最小原门，而应被视为后续复合中的退化常态输出。

第一类 `(x,x)` 表示复制后保持原判词态。根据 `充足理由单子论.md` 对复制的约束，复制不是机械重写，而是为同一对象引入一个概念层等价位。因此最小非退化复制门必须定义为：

```text
P_d(x) = (x,x)
```

第二类 `(opp(x), opp(x))` 表示把单枝送入其对偶双枝。它对应如下结构要求：整体态一旦被拆入局部分枝，每一枝都不再自动保留原整体的充足性；反之，原整体若不充足，则在对偶分裂下需要通过若干正枝去承担对“不充分”的证成压力。因此最小非退化分裂门必须定义为：

```text
L_s(x) = (opp(x), opp(x))
```

于是，在一元到二元这一层，满足充足理由、固定元数与非退化原则的最小一元门恰好是 `P_d` 与 `L_s`。

### 2.3 二元门的推出

再考察二元门。根据 `K3`，结果的形成依赖正贡献结构的汇聚；根据 `K4`，相关且闭合的整体满足满权守恒。因此，双枝到单枝的门至少必须区分以下两种情形：

- 双枝中的正贡献是否仍足以继续延展；
- 双枝是否已经达到可回收为单枝规定的合法闭合。

这就分别推出传播门与闭合门。

先看传播门。若一个二元门仅用于刻画“正贡献是否仍可继续”，则：

- 输入 `(0,0)` 时，两枝都不提供可继续的正贡献，因此输出只能为 `0`；
- 其余三种输入中至少有一枝仍可继续，因此输出必须为 `1`。

于是传播门被确定为：

```text
P_f(a,b) =
  0, 若 (a,b) = (0,0)
  1, 其余情形
```

再看闭合门。若一个二元门用于刻画“多枝是否能够回收成一个合法单枝规定”，则必须满足：

- 异态对 `(0,1)` 与 `(1,0)` 没有统一的充足理由头部，因此不能构成合法闭合，输出只能为 `0`；
- 同态对 `(u,u)` 可以视为同一判词态的双枝闭合，但其闭合结果不能停留在原双枝层位，而必须回收到与该双枝相对偶的单枝态。

第二点不是附加修辞，而是由最基本的分裂—闭合配对关系强制出来的。因为若某个同态双枝是由单枝经 `L_s` 送入双枝位所得，则闭合门必须把它收回原单枝，否则分裂与闭合就不能构成最基本的回收关系。由

```text
L_s(x) = (opp(x), opp(x))
```

立刻得到闭合门必须满足：

```text
L_m(u,u) = opp(u)
```

故闭合门被唯一压定为：

```text
L_m(a,b) =
  opp(a), 若 a = b
  0,      若 a != b
```

### 2.4 四门定义汇总

综上，本文固定以下四个基本门：

```text
L_s(x) : 分裂门
L_m(a,b) : 闭合门
P_f(a,b) : 传播门
P_d(x) : 复制门
```

其正式定义为：

```text
L_s(x) = (opp(x), opp(x))

L_m(a,b) =
  opp(a), 若 a = b
  0,      若 a != b

P_f(a,b) =
  0, 若 (a,b) = (0,0)
  1, 其余情形

P_d(x) = (x,x)
```

这四个门不是经验性并置，而是四类最小动作的代表：

- 同态引入；
- 对偶引入；
- 传播汇聚；
- 闭合回收。

## 3. 输出表与直接推论

### 3.1 输出表

四个基本门的输出表如下：

```text
L_s(0) = (1,1)
L_s(1) = (0,0)
```

```text
L_m(0,0) = 1
L_m(0,1) = 0
L_m(1,0) = 0
L_m(1,1) = 0
```

```text
P_d(0) = (0,0)
P_d(1) = (1,1)
```

```text
P_f(0,0) = 0
P_f(0,1) = 1
P_f(1,0) = 1
P_f(1,1) = 1
```

从输出表层面可直接读出以下同型关系：

- `L_m` 与经典 `NOR` 同型；
- `P_f` 与经典 `OR` 同型；
- `L_s` 是“反值后复制”的同型结构；
- `P_d` 是“保持后复制”的同型结构。

但这里的“同型”仅限于输出表，不表示语义退化为普通布尔门。此处 `1/0` 始终解释为判词在 `Full/Zero` 桥上的状态。

### 3.1.1 双枝输入省写

为避免重复书写，本文约定如下省写规则。若

```text
y = (y_1,y_2)
```

是某个双枝输出词，则对任一二元门 `G(a,b)`，记：

```text
G(y) := G(y_1,y_2)
```

因此：

```text
L_m(L_s(x))
```

表示

```text
L_m(opp(x), opp(x))
```

而

```text
P_f(P_d(x))
```

表示

```text
P_f(x,x)
```

下文凡二元门直接作用于某个双枝输出词时，都按此规则理解。

### 3.2 命题一：两组直接复原

由定义立即得到：

```text
L_m(L_s(x)) = x
P_f(P_d(x)) = x
```

这两条式子表明：

- 分裂后再按对应闭合门收回，可恢复原判词；
- 复制后再按传播门汇聚，可恢复原判词。

因此：

- `L_s` 与 `L_m` 构成一组带有对偶翻转的配对门；
- `P_d` 与 `P_f` 构成一组保持极性的配对门。

### 3.3 命题二：两组导出反值

同样由定义可得：

```text
P_f(L_s(x)) = opp(x)
L_m(P_d(x)) = opp(x)
```

于是可导出：

```text
NOT(x) = L_m(x,x) = P_f(L_s(x))
```

这说明反值不是额外原符，而是门族内部的直接推论。

### 3.4 推论：常用逻辑门的导出

由前述命题可继续导出：

```text
OR(a,b) = P_f(a,b)
NOR(a,b) = L_m(a,b)
NOT(x) = L_m(x,x)
AND(a,b) = L_m(L_m(a,a), L_m(b,b))
```

因此，仅凭这组门就足以形成一套有限可计算的基本门族。其中 `L_m` 作为 `NOR` 同型门，已经提供一条函数完备的生成核。

## 4. 枚举验证与完备性结果

本文用有限枚举对四门的复合闭包进行验证。枚举不引入新的理论前提，只用于确认由定义已给出的结构结论在有限层数上如何展开。

### 4.1 单目层闭包

枚举所有由单一输入 `x` 出发且仅经由本门族复合得到的单目输出，可得恰好四种输出表：

```text
(0,0) : 常零
(0,1) : 恒等
(1,0) : 反值
(1,1) : 常一
```

其最小出现深度为：

```text
恒等 depth 0
反值 depth 1
常零 depth 2
常一 depth 2
```

因此，单目层恰好闭成四态。这意味着：

- `NOT` 可以在一层内自产生；
- `0/1` 两个常态可以由门族内部在二层内构造；
- 单输入层已经具备完整而有限的演算闭包。

### 4.2 双目标准形、十六类定理与深度分层

先固定本文的分类原则。

### 定义 8：有效逻辑等价

设 `G,H` 为由 `L_s, L_m, P_f, P_d` 组成的有限门网络，且二者都在同一结果位点上输出单一判词。若对任一允许输入词，`G` 与 `H` 在判词层所给出的结果完全一致，则记

```text
G =_eff H
```

并称二者有效逻辑等价。

该定义不是附加约定，而是 `充足理由单子论.md` 中 `K2` 的门层对应写法。
因为 `K2` 要求：

```text
Eff(alpha) = Eff(beta) <=> alpha = beta
```

故凡在结果位点上给出同一有效理由结构的门网络，不应再被计作两个不同逻辑种类。

### 定义 9：双目标准形

对任一有限门网络 `G`，若存在某个门网络 `N` 使得：

```text
G =_eff N
```

且 `N` 可写为

```text
<u0(b), u1(b)>
```

其中 `u0, u1` 都是一变量标准形，则称 `N` 为 `G` 的双目标准形。

这里的“双目”不表示系统只能处理两个输入口，而表示：在有效分类下，门网络最终是按两个判词位的作用型被计类的。

### 命题 6：一变量标准形完备

一变量门网络在有效等价下恰好只有四个标准形：

```text
0
1
b
opp(b)
```

证明

由第 2 节已知，一元门只有 `L_s, P_d` 两类最小原门；由第 3 节与第 4.1 节可知：

```text
P_f(P_d(b)) = b
L_m(P_d(b)) = opp(b)
L_m(b, P_f(L_s(b))) = 0
P_f(b, L_m(P_d(b))) = 1
```

故一变量闭包至少含有上述四种结果。另一方面，4.1 的有限枚举已给出：一变量闭包恰好只出现这四种结果，而无第五种新形。故命题成立。

### 命题 7：双目标准形存在且总类为十六

任一有限二值门网络，在有效等价下都可归约为某个双目标准形；且双目标准形总类恰好是

```text
4 × 4 = 16
```

类。

证明

第一步，证明存在性。
由固定元数原则可知，任一门网络都只能由一元与二元复合构成有限二叉树。取最终结果位点，对其最外层二元分支按第一判词位作余因子分解，可写为

```text
F(a,b) = <F(0,b), F(1,b)>
```

其中两项都是一变量残余网络。由命题 6，这两个一变量残余网络都必在有效等价下归约到四个一变量标准形之一。故整个网络必在有效等价下归约为

```text
<u0(b), u1(b)>
```

的形式，即存在双目标准形。

第二步，证明总类数。
一变量标准形只有四类，而双目标准形由两个一变量标准形有序配对组成，因此标准形目录至多为

```text
4 × 4 = 16
```

类。

第三步，证明这 16 类全部可达。
由本门系的有限枚举结果可得，双目标准形恰好出现下列 16 类，且每一类都有具体门式实现：

```text
depth 0 :
  a     = <0, 1>
  b     = <b, b>

depth 1 :
  OR    = <b, 1>
  NOR   = <opp(b), 0>
  NOT a = <1, 0>
  NOT b = <opp(b), opp(b)>

depth 2 :
  0     = <0, 0>
  1     = <1, 1>
  AND   = <0, b>
  NAND  = <1, opp(b)>
  a->b  = <1, b>
  b->a  = <opp(b), 1>
  F     = <b, 0>
  G     = <0, opp(b)>

depth 3 :
  XOR   = <b, opp(b)>
  XNOR  = <opp(b), b>
```

故双目标准形总类既不少于 `16`，也不多于 `16`，从而恰好等于 `16`。命题成立。

### 推论 1：多输入不增类

名义输入口数的增加，不产生新的二值有效逻辑种类；它只产生更长的门网络展开形。按有效分类计，这些展开形仍必归约到命题 7 的 16 类之一。

### 命题 8：深度分层的紧凑性

双目标准形的最小出现深度分布为：

```text
depth 0 : 2
depth 1 : 4
depth 2 : 8
depth 3 : 2
```

这一分层是自然且紧凑的。

证明

先看计数骨架。由命题 6，一变量标准形目录为

```text
<b>
<b, opp(b)>
<b, opp(b), 0, 1>
```

按最小深度逐层放开，其累积基数依次为：

```text
1, 2, 4
```

双目标准形则是两个一变量标准形的有序配对。故在前三级中，可出现的新类数依次表现为：

```text
2 × 1 = 2
2 × 2 = 4
2 × 4 = 8
```

这就解释了 `2,4,8` 的幂次来源。

再看为何末层只余 `2` 类。定义：若某个双目标准形存在某一输入位和某一取值，使得该输入一旦被固定，输出立刻塌为常态，则称其具有有有单侧决定性有有。对 16 类双目标准形逐一检查可得：

```text
depth 0 到 depth 2 的 14 类均具有单侧决定性
depth 3 的 2 类不具有单侧决定性
```

最后两类恰是：

```text
XOR   = <b, opp(b)>
XNOR  = <opp(b), b>
```

因此，前三级把全部可单侧决定的标准形无空洞放满；末层只剩不能被单侧决定骨架吸收的比较型余项。故分层既非偶然，也不存在浅层空缺，命题成立。

### 4.2.3 归一冗余度与输入口增长

在命题 7 的分类下，二值有效逻辑总类固定为 `16`。因此，任一阶段实际可调用的有效逻辑词表 `E` 都满足：

```text
1 <= |E| <= 16
```

若记：

```text
x := |E|
```

则识别 `E` 中某一逻辑所需的最小二进制描述长度至多为：

```text
log2(x) <= 4
```

另一方面，若某一结果位点有 `n` 个输入口，则其完整输出表长度为：

```text
2^n
```

其中必有

```text
n >= 1
```

据此定义归一冗余度：

```text
R_n(x) := (2^n - log2(x)) / n
```

它表示：在有效逻辑词表固定为 `x` 类时，每个输入口平均对应多少尚未被有效逻辑真正占满的输出容量。

### 命题 9：输入口增加时归一冗余严格上升

对固定的 `x`，只要 `n >= 2`，便有：

```text
R_(n+1)(x) > R_n(x)
```

证明

直接计算：

```text
R_(n+1)(x) - R_n(x)
=
[ 2^n (n - 1) + log2(x) ] / [ n (n + 1) ]
```

由于

```text
n >= 2
1 <= x <= 16
```

故

```text
2^n (n - 1) > 0
log2(x) >= 0
```

于是分子严格大于 `0`，分母也严格大于 `0`，从而

```text
R_(n+1)(x) > R_n(x)
```

命题成立。

### 推论 2：十六类固定时的冗余结论

在二值有效逻辑总类固定为 `16` 的前提下，名义输入口数越多，输出—输入整体信息冗余度越高。

这不是经验印象，而是命题 9 的直接结果。因为：

- 输入口增加时，完整输出表容量按 `2^n` 扩张；
- 有效逻辑总类不增加，故其可区分复杂度至多保持在 `4` 位；
- 因此，增加输入口只会带来更长展开形与更大的完整输出空间，而不会带来新的有效逻辑种类。

### 4.3 若干直接可导的复合式

枚举结果可直接给出以下公式：

```text
0 = L_m(a, L_m(a,a))
1 = P_f(a, L_m(a,a))
AND(a,b) = L_m(L_m(a,a), L_m(b,b))
IMP(a->b) = P_f(b, L_m(a,a))
IMP(b->a) = P_f(a, L_m(a,b))
XOR(a,b) = L_m(L_m(a,b), L_m(L_m(a,a), L_m(b,b)))
XNOR(a,b) = P_f(L_m(a,b), L_m(L_m(a,a), L_m(b,b)))
```

这些公式说明：

- 常零与常一可从一个输入及其反值关系内部导出；
- `AND` 不是原始门，而是更深一层的闭合结果；
- 两个方向的蕴含都在第二层出现，因此方向性是复合层涌现的；
- `XOR / XNOR` 需要比较更深层的闭合链，因此最晚出现。

### 4.4 复制与可演算性

双目标准形下全部 16 类有效逻辑可达，只说明这组门具有表达完备性；它还不足以说明这组门为何适于计算。使完备性进入可演算状态的关键，是复制门。

因为：

- `L_s / L_m / P_f` 决定了哪些有效逻辑标准形可达；
- `P_d` 决定了这些可达标准形能否被反复调用、重新编排并嵌入更长推导。

因此，枚举实验给出的是双重结论：

1. 从表达能力看，门族是完备的；
2. 从演算能力看，复制使这种完备性成为有限编排中的实际能力。

若没有复制，许多复合式虽然形式上可写，却不能在系统内部被系统性复用；一旦有了复制，同一对象便可被带入概念层等价位，进而参与多路传播、比较与闭合。

### 4.5 复合层中的非交换涌现

原始二元门本身满足：

```text
P_f(a,b) = P_f(b,a)
L_m(a,b) = L_m(b,a)
```

因此，原始传播门与原始闭合门都满足交换性。但复合层并不自动保留这种对称。

枚举结果表明：

- 并非所有双目标准形都满足交换对称；
- 非交换性不来自原始门本身，而来自变量在复合树中的位置不对称；
- 哪一个对象先被反值、先被闭合筛选、先被送入传播门，都会改变最终输出。

最直接的方向性例子是：

```text
P_f(b, L_m(a,a)) = a -> b
P_f(a, L_m(a,b)) = b -> a
```

另一个例子是：

```text
F(a,b) = L_m(a, L_m(a,b))
G(a,b) = L_m(b, L_m(a,a))
```

其输出表分别为：

```text
F = (0,1,0,0)
G = (0,0,1,0)
```

由此可见：

- 原始门层有交换性；
- 复合门层未必交换；
- 方向性不是外加假设，而是由复合排列自然涌现的。

## 5. 两条函数完备路线及其层级解释

### 5.1 两条完备路线

若进一步收紧“计算”的含义，就会看到这里并不只有一条函数完备路线，而是存在两条。但这两条路线不是两套互不相关的门理论，而是同一门网络在不同层面上的两种读法。

第一条路线是：

```text
L_m + P_d
```

即：

- 以 `L_m` 作为 `NOR` 同型门；
- 以 `P_d` 保证同一对象能够进入多个输入位；
- 再由

```text
NOT(x) = L_m(P_d(x))
```

从内部导出反值。

于是：

- `NOT` 能由 `L_m` 与 `P_d` 内生；
- `AND / OR / XOR / XNOR / IMP` 都可继续导出；
- 这是一条以 `NOR` 为核心的函数完备路线。

第二条路线是：

```text
P_f + NOT + P_d
```

即：

- 以 `P_f` 作为 `OR` 同型门；
- 以已导出的 `NOT` 作为一元翻转；
- 以 `P_d` 保证固定元数下的多位调用。

例如：

```text
AND(a,b) = NOT(P_f(NOT(a), NOT(b)))
```

由此可以继续导出其余有效逻辑标准形。因此，这是一条以 `OR + NOT` 为核心的函数完备路线。

### 5.2 两条路线的不同重心

两条路线虽然都完备，但其结构重心并不相同。

在 `L_m + P_d` 路线中，核心问题是：

- 哪些输入枝能够形成合法闭合；
- 一旦闭合，结果为何翻到对偶态；
- 不同闭合次序为何不能被随意抹平；
- 深层复合为何自然长出方向性。

因此这条路线更接近“结果规定”路线。

在 `P_f + NOT + P_d` 路线中，核心问题是：

- 当前哪些理由枝仍然可达；
- 多条可达枝如何并接；
- 局部翻转如何改写传播链；
- 哪些局部组合能够稳定重排。

因此这条路线更接近“局部读出”路线。

### 5.3 复制为何必须并入两条路线

若仅在形式上写出

```text
L_m(x,x)
P_f(a, P_f(a,b))
```

似乎同一对象可以被直接重复调用。但若坚持本文的语义约束：

- 单个操作的输入数不能改变；
- 同一对象进入多个输入位必须有明确操作理由；

则复制门不是可有可无的附属件，而是演算得以展开的必要条件。

`P_d` 的作用不是改变真假，而是为同一对象创造概念层等价位，使其能够合法进入多个输入口。因此，说函数完备时更准确的表述应是：

- `NOR + 复制` 构成一条完备路线；
- `OR + NOT + 复制` 构成另一条完备路线。

## 6. 全局、截面与桥

### 6.1 两个视角的层级关系

上述两条完备路线分别对应系统中的两个观察尺度，但这两个尺度不是同阶并列的本体，而是同一闭合网络的两种读法：

- 全局视角处理“什么真正规定了结果”；
- 局部截面视角处理“在当前读出位置还剩哪些可见理由线”。

因此二者不是并列世界，而是前后有序的两层：

- 全局在前，给出决定性闭合结构；
- 局部在后，给出该闭合结构的有限读出与运算次序。

### 6.2 全局视角为何更接近 `NOR + 复制`

从全局理由网络看，重点不在“是否仍有一枝可继续”，而在：

- 哪些枝真正能够闭合；
- 闭合如何把整体翻到对偶态；
- 树形结构如何进入结果规定。

这正对应 `L_m` 的性质：

```text
L_m commutative = 1
L_m associative = 0
```

因此 `NOR + 复制` 更贴近全局视角。这里的非结合并非偶发的代数异常，而是在表明：

- 全局理由结构不能被随意摊平；
- 谁先闭合、谁后闭合，会改变结果被规定的方式；
- 闭合树形本身就是决定性内容的一部分。

### 6.3 局部截面视角为何更接近 `OR + NOT + 复制`

退回局部截面时，重心转为：

- 当前还有没有可达理由线；
- 多条理由线并接后是否仍可通；
- 局部组合能否稳定重排。

这更接近 `P_f` 的性质：

```text
P_f commutative = 1
P_f associative = 1
```

因此 `OR + NOT + 复制` 更贴近局部截面视角。这里的结合不表示局部比全局更真实，而表示：

- 截面只读出仍可见的理由枝；
- 对这些尚未熄灭的理由枝，传播次序可以在局部被压平；
- 因而在读出层上，括号可以弱化为可重排的并接。

### 6.4 全局—截面的桥的严格位置

全局与截面之所以能够接通，不是因为它们都恰好能写成 `0/1`，而是因为它们读的是同一门网络上的同一批理由枝。由此可知，桥不可能是“截面”这个词本身，也不可能是“概率”这个词本身。

能够承担桥接功能的，只能是：

```text
Sec(E,κ) 上同一全局闭合门网络的权重承载关系
Path(E,γ) 上该承载结构的有限可读出性
```

因此，四层关系应写成：

```text
全局层  : 门网络在 Sec(E,κ) 中给出决定性闭合结构
桥      : 由 δ_R(p) 与 Ω_{E,γ}(R) 把闭合结构压到 Path(E,γ) 上
读出层  : 由 Read_Σ、Q_{i,k}、Q_i 把可见承载写成有限支持词
概率层  : 对 Head(E) 上的 Q_i 作有限归一，得到 P(R_i|E,γ)
```

更具体地说，这条桥接链必须按下式分层：

```text
Sec(E,κ)
-> Head(E)
-> δ_{R_i}(p) on Path(E,γ)
-> Ω_{E,γ}(R_i)
-> Read_Σ
-> Q_{i,k}
-> Q_i
-> P(R_i|E,γ)
```

这里的桥既不是把全局压扁成局部，也不是把局部抬升成全局。它真正承担的是：

- 保持同一性；
- 允许读出；
- 不改变决定性；
- 只改变可见次序。

并且必须严格区分：

- 门层直接改写的是判词结构、分裂闭合关系与可传播关系；
- `δ_R(p)` 的变化只在门网络改写后，经路径吸纳才显现；
- `Read_Σ`、`Q_{i,k}`、`Q_i` 与 `P(R_i|E,γ)` 都属于门层之后的读出与归一层，不进入门定义本身。

换言之：

- 没有 `L_m`，全局没有严格规定点；
- 没有 `P_f`，局部没有稳定读出链；
- 没有 `P_d`，同一对象不能跨位置复用；
- 没有 `L_s`，整体与子枝之间的结构张力无法被展开。

因此，桥不是第五个门，而是四个门先改写判词结构与可传播关系，再由 `δ_R -> Ω -> Q -> P` 这条既有对象链完成层间配合。

### 6.5 为什么这种结构可能实现莱布尼茨的理想

莱布尼茨式“让我们计算吧”要求的不只是“逻辑可写成符号”，而是至少要求：

- 对象能够被压成有限可操作形式；
- 同一对象能够被反复调用与比较；
- 推理步骤能够被拆成明确规则；
- 结果能够回收到可检查的规定点。

若没有全局—截面的桥，这一理想无法落实：

- 只有全局，没有截面，则只有被决定的整体，没有可操作读出次序；
- 只有截面，没有全局，则只有局部运算链，不知其是否对应真实决定结构；
- 只有传播，没有闭合，则计算只能扩散，不能判定；
- 只有闭合，没有传播，则结构虽可规定，却不能形成连续演算链；
- 只有形式复写，没有复制，则同一对象无法多位调用，计算不能展开。

因此，这一理想之所以在这里成为可能，不是因为出现了若干形似布尔门的式子，而是因为这里同时把以下四件事压到了同一有限语法之中：

- 全局决定；
- 局部读出；
- 同一对象的可重复调用；
- 结果的可回收判定。

## 7. 固定输入数下的结构性质

### 7.1 单输入层只产生对角双枝

对单输入门而言：

```text
L_s(0) = (1,1)
L_s(1) = (0,0)
P_d(0) = (0,0)
P_d(1) = (1,1)
```

因此，单输入对象一旦被送入双枝位，只会得到两类对角双枝：

- `(0,0)`；
- `(1,1)`。

它不会直接产生 `(0,1)` 与 `(1,0)`。这说明：

- 单输入层不会自行制造左右不一致；
- `L_s` 与 `P_d` 都保持两枝同态骨架；
- 二者的差别只在于是否翻转极性。

### 7.2 单输入标量闭包恰好四态

若保持单输入，只允许先由 `L_s` 或 `P_d` 把单枝送入双枝位，再由 `L_m` 或 `P_f` 收回单枝位，则可得到：

```text
L_m(L_s(x)) = x
P_f(P_d(x)) = x
P_f(L_s(x)) = opp(x)
L_m(P_d(x)) = opp(x)
L_m(x, P_f(L_s(x))) = 0
P_f(x, L_m(P_d(x))) = 1
```

因此在固定单输入下，所有可达标量态恰好闭成：

- 恒等 `x`；
- 反值 `opp(x)`；
- 常零 `0`；
- 常一 `1`。

这表明单输入层已经自足地产生保持、翻转、常零与常一四类基本结果，不需要再额外引入别的一元原门。

### 7.3 双输入层在深度 3 饱和

若固定双输入为 `a,b`，只允许二元复合继续回到单枝输出，则穷举结果表明：

```text
depth 0 新增 2 张
depth 1 新增 4 张
depth 2 新增 8 张
depth 3 新增 2 张
```

到深度 3 为止，16 类二值有效逻辑标准形全部出现，此后再加深也不再产生新类。因此：

- 双输入表达能力在很浅的层数就已饱和；
- `XOR / XNOR` 最深，并不表示不可表达，只表示其需要更长闭合链；
- 双输入层的复杂性是在有限浅层中迅速完成，而不是无限拖延出来的。

### 7.4 对角与反对角的基本分裂律

在双输入层里，同态输入与异态输入会被两类门截然不同地处理：

```text
P_f(x,x) = x
L_m(x,x) = opp(x)
P_f(x, opp(x)) = 1
L_m(x, opp(x)) = 0
```

这四条式子构成双输入层的基本分裂律。它们说明：

- 对角输入 `(x,x)` 保留可判别信息；
- 反对角输入 `(x,opp(x))` 会直接被压到边界态；
- `P_f` 把“任一正枝仍在”理解为可继续传播；
- `L_m` 把“必须同态”理解为严格闭合门槛。

### 7.5 `P_f` 的传播半群性质

有限计算验证表明：

```text
P_f commutative = 1
P_f associative = 1
P_f monotone     = 1
P_f(x,x) = x
P_f(a,0) = a
P_f(a,1) = 1
P_f(a, P_f(a,b)) = P_f(a,b)
```

因此，`P_f` 可视为一个稳定的传播半群：

- 它交换；
- 它结合；
- 它单调；
- `0` 是中性位；
- `1` 是饱和值；
- 重复传播不会累加新内容。

所以 `P_f` 的本质不是闭合门，而是可达性保持门。

### 7.6 `L_m` 的严格闭合与树形敏感性

有限计算验证表明：

```text
L_m commutative = 1
L_m associative = 0
L_m monotone     = 0
L_m(x,x) = opp(x)
L_m(a,0) = opp(a)
L_m(a,1) = 0
```

因此，`L_m` 比传播严格得多：

- 它要求两枝同态；
- 即使同态，也不保留原态，而翻到对偶态；
- 它既无中性位，也无饱和值；
- 它不结合，因此闭合树形本身进入语义；
- 它不单调，因此“增加一个正枝”并不必然更容易闭合。

所以 `L_m` 的本质不是“并”，而是对偶闭合检验门。

### 7.7 混合复合中的透明律、筛选律与方向性

在更深一层复合中，可得到：

```text
P_f(a, L_m(a,b)) = b -> a
P_f(b, L_m(a,a)) = a -> b
L_m(a, P_f(a,b)) = L_m(a,b)
P_f(a, P_f(a,b)) = P_f(a,b)
L_m(a, L_m(a,b)) = (0,1,0,0)
L_m(b, L_m(a,a)) = (0,0,1,0)
```

这些式子说明：

- 两个相反方向的蕴含在第二层复合中出现；
- 对 `L_m` 而言，某些传播层是透明的；
- 对 `P_f` 而言，重复传播不会新增内容；
- 左向与右向筛选在更深闭合中分化出来。

因此，真正的方向性出现于第二层复合之后，而不属于原始单输入层或原始二元门本身。

## 8. 与相关理论的关系

### 8.1 与函数完备分类思想的关系

经典门论讨论函数完备性时，并不关心某个门是否常见，而关心有限原门经过有限复合后能否生成全部输出表。就此而言，本体系与经典完备理论的深层对应点不在于“真值表长得像”，而在于它同样研究有限原操作的生成能力。

这里的不同之处在于：

- 经典门论通常把问题表述为布尔函数生成；
- 本体系则把问题重写为充足理由结构的有限生成。

因此，本体系不是把经典完备理论机械搬入，而是在自己的基底上重新解释“完备”。

### 8.2 与非结合结构理论的关系

一旦某个二元运算不满足结合律，公式就不应再被视为可随意加括号的平面串，而应被视为树状复合结构。这一思想在非结合 Lambek 演算、bunched logic 以及相关语义理论中都有明确体现。

这与 `L_m` 的非结合性高度一致。因为：

```text
L_m(a, L_m(b,c))
!=
L_m(L_m(a,b), c)
```

真正改变的不是书写习惯，而是：

- 哪一枝先被视为局部整体；
- 哪一次闭合先发生；
- 哪个局部结果再进入下一轮闭合；
- 因而哪一条规定链先冻结为结果。

因此，`L_m` 的非结合性表明全局语义天然要求结构性复合，而不是平面串语义。

### 8.3 非结合比结合更原初

从函数完备问题本身看，完备性所讨论的是：有限原操作经过有限复合后，能够生成多大的输出族。这里的“复合”在未假定结合律之前首先天然是结构性复合。

因此：

- 树先于串；
- 非结合先于结合；
- 结合不是完备性的前提，而是某些局部情形下额外获得的塌缩对称。

这一点可进一步表述为：

- `NOR` 虽然非结合，但仍然函数完备；
- 因此函数完备性根本不以结合律为公设；
- 结合性不能反过来被提升为完备门的先验要求。

故更强的判断是：

```text
完备性先问树能生成什么
结合性才问这些树能否被遗忘
```

### 8.4 线性逻辑中的复制只是这里的受限边界情形

线性逻辑最重要的启发，在于它指出复制不能被视为免费动作：是否允许复制、何时允许复制、复制后的东西是否仍算同一项，都会改变计算模型。

但线性逻辑通常通过上下文规则管理复制问题，即：

- 默认不允许自由 contraction；
- 只有在受控模态下才允许重复使用；
- 因而它首先讨论的是“某个假设能不能多用一次”。

本文中的 `P_d` 与此有本质差别：

```text
P_d(x) = (x,x)
```

`P_d` 不仅表示允许多用一次，而表示系统内部执行了一次显式复制操作，由此为对象开出概念层等价位。换言之：

- 在线性逻辑中，复制问题首先是使用许可问题；
- 在本体系中，复制问题首先是对象级生成问题。

因此，线性逻辑可被理解为本体系中“复制问题”的一个受限边界情形：它抓住了“复制不能免费”这一点，但尚未把复制本身提升为对象语言内部的独立门操作。

### 8.5 与证明网络和规约语义的关系

如果把这套门系统从更宽的计算视角来观察，那么它更接近一种门网络上的规约语义，而不只是新的真值表库。其关注点不在于某个公式如何纸面推出，而在于：

- 对象如何进门；
- 如何分裂、闭合、传播、复制；
- 最终怎样在网络中规约到结果。

因此，本体系自然与 proof nets、cut elimination、图重写等思想存在潜在接口，但其自身的解释仍必须保持在充足理由基底之内。

## 9. 充足理由语义下的解释

### 9.1 `L_s` 的语义

`L_s` 不是普通拆分，而是把一个判词沿 `Full/Zero` 对偶桥送入其对偶双枝。

```text
L_s(1) = (0,0)
```

表示：一个作为整体已具有充足理由的命题，一旦被拆入两个孤立子枝，这两个子枝都不再单独拥有足够理由。

```text
L_s(0) = (1,1)
```

则表示：一个不充足的整体，在对偶分裂下被拉成两个正枝，以承担对该不充分性的相对独立证成压力。

因此，`L_s` 的语义核心不是“真变假”或“假变真”，而是整体与子枝之间充足性张力的显式展开。

### 9.2 `L_m` 的语义

`L_m(a,b)` 的规则是：

- 相同，才允许回收为一个对偶单枝；
- 不同，直接落零。

因此：

```text
L_m(0,1) = 0
L_m(1,0) = 0
```

表示异态对没有统一的充足理由头部，不能构成合法闭合。故 `L_m` 的本质不是宽松并合，而是严格闭合检验。

### 9.3 `P_f` 的语义

`P_f` 的规则是：

```text
P_f(0,0) = 0
P_f(0,1) = 1
P_f(1,0) = 1
P_f(1,1) = 1
```

因此它的语义不是闭合，而是可达性传播：只要仍有一条可行理由线，传播就继续；只有当两条输入都无理由时，传播才关闭。

### 9.4 `P_d` 的语义

`P_d` 不改变极性，但它也不是对对象作机械重写。更准确地说，`P_d` 表示：

- 对输入对象创造一个概念层等价概念；
- 该等价概念保持原对象当前判词态；
- 复制后的两位不是两个彼此无关的副本，而是“对象位 + 概念等价位”。

因此，`P_d` 的充足理由在复制动作本身，而不在输入对象原先是否已经理由充足。

### 9.5 四门的协同关系

若将四门的语义协同起来看，则可得：

- 分裂给出理由结构的展开；
- 闭合给出理由结构的回收；
- 传播给出理由线的延续；
- 复制给出理由项的可重复调用。

四者合起来，门层就不再是单纯的真值变换器，而是一个最小理由动力学。

## 10. 结论

本文在 `充足理由单子论.md` 的逻辑基础上，抽取并说明了四个最小逻辑操作门：

```text
L_s : 把一个判词送入其对偶双枝
L_m : 只允许同态双枝闭合，异态双枝落零
P_f : 只要仍有正枝，传播就继续
P_d : 为输入对象创造一个概念层等价位，并保持其判词态
```

就其最紧的结构角色而言，可以压缩为：

```text
分裂 = 对偶化
闭合 = 闭合检验
传播 = 可达保持
复制 = 概念层等价引入
```

在此基础上，全文得到以下结论：

- 这组门不是任意选定，而是由充足理由基底、固定元数、判词闭包、传播与闭合分离、复制显式化共同迫出的最小门族；
- 单目层恰好闭成四态，双目标准形完整覆盖 16 类有效逻辑，因此门族在有限层数内达到完备；
- `NOR + 复制` 与 `OR + NOT + 复制` 构成同一门网络在全局决定层与局部截面层中的两条完备路线；
- 全局与截面之间的桥不在概率词或截面词本身，而在同一门网络上的权重承载关系及其局部可读出性；
- `P_d` 把复制从元语言习惯压回对象语言中的显式操作，因此比线性逻辑中的受控复制更一般；
- `L_m` 的非结合性说明全局语义天然要求树形规定，因此结合性不应被视为完备性的公设。

因此，若从充足理由逻辑的角度看，最底层的逻辑核并不是普通真假运算，而是：

- 分裂；
- 闭合；
- 传播；
- 以及使这三者进入有限演算编排的显式复制。
