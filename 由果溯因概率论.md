# 由果溯因概率论

## 1. 目的

本文处理以下问题：

1. 严格由因推果为何不属于概率层。
2. 由果溯因的概率读数为何必须写成有限净支持的归一投影。
3. 前向支撑为何先表现为复支撑，进入概率层前为何必须经过实读出与量子化。
4. 精确溯因分布的基本表达式是什么。
5. 概率分布的最基础性质与差分性质是什么。
6. 经典分布为何只是附加结构下的收缩型。

全文只允许有限路径、有限权步、有限归一、有限判词与有限项运算。

## 2. 继承基底

### 2.1 有限对象

只允许：

```text
有限单子词
有限连接式
有限路径词
有限路径词序
有限权重词
有限判词词序
```

### 2.2 第一性根式

保留：

```text
K1 规定式 :
  Live(w(alpha)) = 1 => 存在某个 R 使得 R => alpha

K2 同一式 :
  Eff(alpha) = Eff(beta) <=> alpha = beta

K3 传权式 :
  结果权词由正贡献理由汇聚给出

K4 闭组守恒式 :
  相关闭组中的满权守恒
```

### 2.3 截面对象

给定结果单子 `E` 与背景 `γ`，保留：

```text
Sec(E,κ)       : 结果 E 在角度 κ 下的完整理由截面
Path(E,γ)      : 允许回溯路径词序
ω(p)           : 路径 p 的路径权
δ_R(p)         : 路径 p 是否被理由头部 R 吸纳
Ω_{E,γ}(R)     : 理由头部 R 的理由级支持量
```

### 2.4 有限权步

对任一完整理由截面，固定有限总权步数 `M`。若单子 `alpha` 占有 `k_alpha` 个权步，则：

```text
w(alpha) := k_alpha : M
```

其中 `k_alpha : M` 只是有限归一比例词，不是连续实值。

本文排除：

测度空间

## 4. 支撑方向与概率层

### 定义 4.1：支撑偏序图

因果有向图 `G=(V,E_d)` 的有向性只表示理由支撑方向，不表示经验时间先后。

### 命题 4.2：时序非本体性

若仅改变经验时序标注而不改变边方向、路径可达与连接式，则由果溯因读数不变。

### 证明

由果溯因读数只依赖：

```text
Path(E,γ)
ω(p)
δ_R(p)
有效轴判词
局部相对权
实读出
量子化单位
```

这些对象都由支撑结构给出，不由经验时序标注给出。

### 定义 4.3：严格由因推果

若理由头部 `R` 对结果单子 `E` 构成合法充足理由，则记：

```text
R => E
```

### 定理 4.4：前向概率无本体意义

若讨论对象是真正的 `R => E`，则不存在独立的前向概率问题。

### 证明

`R => E` 属于规定关系。规定关系的本体值是“被规定”或“未被规定”，不是“以多大概率发生”。若再写 `P(E|R)`，则要么只是退化读数，要么对象已不再是被规定项 `E` 本身，要么 `R` 并不充足。

### 推论 4.5：概率只属于由果溯因层

严格前向因果不属于概率层；概率只属于结果已给定后的回溯归一读数。

## 5. 自由轴与路径判词

### 定义 5.1：剩余自由有效轴

对结果 `E` 与背景 `γ`，把当前尚未闭合锁定的有效结构规约为不可约正交轴：

```text
Free(E,γ) = <ξ_1,...,ξ_s>
```

满足：

```text
1. ξ_k 对当前回溯问题有效
2. 任意 ξ_i, ξ_j 在当前切面下正交
3. ξ_k 在当前切面下不可再约
```

### 定义 5.2：结果要求方向

对每条自由轴 `ξ_k`，赋予结果端的要求方向：

```text
τ_k ∈ {+1,-1}
```

### 定义 5.3：路径-轴判词

对路径 `p` 与自由轴 `ξ_k`，定义：

```text
K_γ(p)          ∈ {0,1}
Eff_k(p|γ)      ∈ {0,1}
Orth_k(p|γ)     ∈ {0,1}
σ_k(p)          ∈ {-1,0,+1}
```

## 6. 局部相对权与复支撑

### 定义 6.1：局部相对权

对边 `e:u->v`，定义：

```text
ρ_v(e)
=
|e| / Σ_{f∈In(v)} |f|
```

### 命题 6.2：局部归一性

对任一节点 `v`：

```text
0 < ρ_v(e) ≤ 1
Σ_{f∈In(v)} ρ_v(f) = 1
```

### 定义 6.3：路径保留率

若路径 `p=e_1...e_r`，且每条边都写成 `e_t:u_t->v_t`，定义：

```text
R(p) := Π_{t=1}^r ρ_{v_t}(e_t)
```

### 定义 6.4：边的复支撑提升

对每条边 `e:u->v`，定义复数 `α(e)` 满足：

```text
|α(e)|^2 = ρ_v(e)
```

若同时记录对偶转移，则定义 `β(e)` 满足：

```text
|β(e)|^2 = 1 - ρ_v(e)
```

### 命题 6.5：局部复支撑守恒

```text
|α(e)|^2 + |β(e)|^2 = 1
```

### 定义 6.6：路径复支撑振幅

若路径 `p=e_1...e_r`，定义：

```text
A(p) := Π_{t=1}^r α(e_t)
```

### 命题 6.7：路径保留率是模平方

```text
|A(p)|^2 = R(p)
```

### 证明

```text
|A(p)|^2
= Π_t |α(e_t)|^2
= Π_t ρ_{v_t}(e_t)
= R(p)
```

## 7. 实读出与量子化

### 定义 7.1：路径-轴复支撑项

固定当前完整相关截面 `Sec(E,κ)`。对每个相关路径-轴对 `(p_j,ξ_k)`，定义复支撑项 `z_{j,k}`，满足：

```text
|z_{j,k}|^2
=
Take(K_γ(p_j)·Eff_k(p_j|γ)·Orth_k(p_j|γ), ω(p_j))
```

### 命题 7.2：复支撑项存在且不唯一

对每个相关 `(p_j,ξ_k)`，至少存在一个 `z_{j,k}` 满足定义 7.1。若 `|u|=1`，则 `uz_{j,k}` 给出同一实读出。

### 定义 7.3：实读出算子

```text
Read_Σ(z) := |z|^2
```

### 命题 7.4：相位盲性

若 `|u|=1`，则：

```text
Read_Σ(uz) = Read_Σ(z)
```

### 命题 7.5：相位与符号分层

`Read_Σ(z_{j,k})` 只给出非负可见量；路径方向仍由

```text
σ_k(p_j) ∈ {-1,0,+1}
```

单独记录。

### 推论 7.6：净支持量子数的相位不变性

固定全部 `Read_Σ(z_{j,k})`、`δ_i(p_j)`、`σ_k(p_j)` 与 `τ_k`，任意改变各 `z_{j,k}` 的相位，都不改变全部净支持量子数与归一读数。

### 定义 7.7：非零实读出词

把全部非零实读出按任意固定次序写成有限词：

```text
<r_1,...,r_u>
```

其中每个 `r_h` 都是某个非零 `Read_Σ(z_{j,k})`。

### 定理 7.8：最小损耗单位存在

存在一个最大的正有理单位 `ε_Σ`，使得上述有限词中每一项都是 `ε_Σ` 的整数倍。

### 证明

`Path(E,γ)` 有限，`Free(E,γ)` 有限，`ω(p_j)` 由有限权步与有限乘法权给出，故该读出词是有限有理词。有限有理词必存在公共最大单位。

### 推论 7.9：量子化实读出

对每个相关 `(p_j,ξ_k)`，存在唯一整数 `m_{j,k} ∈ N`，使：

```text
Read_Σ(z_{j,k}) = m_{j,k} ε_Σ
```

### 推论 7.10：与路径保留率的一致性

若某路径 `p_j` 的乘法权部分完全由局部相对权链乘给出，则可取：

```text
z_{j,k} = c_{j,k} A(p_j)
```

使：

```text
|c_{j,k}|^2
=
Take(K_γ(p_j)·Eff_k(p_j|γ)·Orth_k(p_j|γ), ω(p_j)) / R(p_j)
```

### 定义 7.11：前读出转移差异

对结果 `E` 的相关路径-轴对 `(p_j,ξ_k)`，称在进入

```text
Read_Σ(z_{j,k})
```

之前仍须保留、且既未压成实读出值也未并入

```text
σ_k(p_j)
```

方向记录的正贡献转移差异为该对的前读出转移差异。

本段中的“充足性有效信息”一词专指此前读出转移差异。

### 命题 7.12：前读出载体存在

对每个相关 `(p_j,ξ_k)`，在 `Read_Σ(z_{j,k})` 之前必须存在一个不同于实读出值本身的前读出载体层，用以承载前读出转移差异。

### 证明

由 `K3`，结果权词由正贡献理由汇聚而成；由 `R5`，合法使用必须沿全部正贡献链继续展开，不得遗漏正贡献因子；由普通单子的充足理由不可逆返，已进入结果链的正贡献不能逆返其来源；由 `R9`，除非当前切面已经满足闭合退化条件，否则不得把当前截面强行压成终点满值；由 `K4`，相关闭组中的满权守恒。故在进入 `Read_Σ` 之前，前读出转移差异既不能被提前读成终点实值，也不能无载体失落，因而必须有独立于实读出值本身的前读出载体层。

### 命题 7.13：非负实提升把前读出载体压成读出层

若把前向支撑只写成非负实数 `x_{j,k} ≥ 0`，并仍要求：

```text
Read_Σ(x_{j,k}) := x_{j,k}^2
```

且

```text
Read_Σ(x_{j,k})
=
Take(K_γ(p_j)·Eff_k(p_j|γ)·Orth_k(p_j|γ), ω(p_j))
```

则每个可见读出只对应唯一的前向支撑值：

```text
x_{j,k}
=
sqrt(
  Take(K_γ(p_j)·Eff_k(p_j|γ)·Orth_k(p_j|γ), ω(p_j))
)
```

故非负实提升不能承担命题 7.12 所要求的前读出载体层。

### 证明

非负实数上 `x^2 = r` 对每个 `r ≥ 0` 只有唯一解 `x = sqrt(r)`。故一旦固定 `Read_Σ(x_{j,k})`，前向值便被唯一锁定，前读出载体与终点读出合并为同一层。

### 命题 7.14：纯实提升把前读出载体塌缩到方向层

若把前向支撑只写成实数 `x_{j,k} ∈ R`，并仍取

```text
Read_Σ(x_{j,k}) := x_{j,k}^2
```

则对固定读出 `r > 0`，全部前向支撑只有两种可能：

```text
x_{j,k} = +sqrt(r)
x_{j,k} = -sqrt(r)
```

这两种差别仅是符号差别。由于路径方向已经由

```text
σ_k(p_j) ∈ {-1,0,+1}
```

单独记录，故纯实提升留下的全部剩余自由度都可被吸收到方向层，不能形成独立于方向层之外的前读出载体层。

### 证明

实数上方程 `x^2 = r` 的全部解恰为 `±sqrt(r)`。而本文已把可见量与方向符号分层，故纯实提升保留的唯一额外差别只是符号差别；该差别已由 `σ_k(p_j)` 承担，不再给出独立新层位。

### 定理 7.15：复支撑的绝对必要性

若要求前向支撑仍以数值载体表达，且前读出转移差异在进入 `Read_Σ` 之前既不被压成实读出、也不被并入 `σ_k(p_j)` 的方向记录，则前向支撑不能只取实数，必须取复数。

### 证明

由命题 7.12，前读出载体层必须存在。由命题 7.13，非负实提升把该载体唯一压成终点读出。由命题 7.14，纯实提升把该载体全部塌缩到方向层。故所求域必须同时满足：

```text
1. 固定 Read_Σ 后仍保留非唯一提升；
2. 这类提升不等同于 σ_k(p_j) 的方向记录。
```

对复数前向支撑，取任意 `z_{j,k} != 0`。若 `u` 满足 `|u| = 1`，则

```text
Read_Σ(u z_{j,k}) = Read_Σ(z_{j,k})
```

故在固定实读出后，复数仍保留单位模提升。再取 `u` 既不等于 `+1` 也不等于 `-1`，则 `u z_{j,k}` 与 `z_{j,k}` 的差别不再只是实数符号差别，因而不能被

```text
σ_k(p_j) ∈ {-1,0,+1}
```

吸收。于是复数前向支撑在不改变 `Read_Σ` 的前提下，确实保留了独立于方向层之外的前读出载体。结合命题 7.13 与命题 7.14，实数层全部失效而复数层满足要求，故前向支撑不能只取实数，必须取复数。

### 推论 7.16：复数不是概率本身，而是前读出载体

复数只承担前读出载体功能。真正进入概率层的仍然只是：

```text
Read_Σ
-> m_{j,k} ε_Σ
-> Q_{i,k}
-> Q_i
-> P(R_i|E,γ)
```

## 8. 精确溯因分布

### 定义 8.1：候选理由头部

设当前结果 `E` 在背景 `γ` 下的候选理由头部为：

```text
Head(E) = <R_1,...,R_n>
```

### 定义 8.2：路径吸纳指示词

对每个 `R_i` 与路径 `p_j`，定义：

```text
δ_i(p_j) := δ_{R_i}(p_j) ∈ {0,1}
```

### 定义 8.3：单轴净支持量子数

```text
Q_{i,k}
=
max(
  0,
  τ_k · Σ_{j=1}^m δ_i(p_j) σ_k(p_j) m_{j,k}
)
```

### 定义 8.4：总净支持量子数

```text
Q_i := Σ_{k=1}^s Q_{i,k}
```

### 定义 8.5：支持词

```text
Q := <Q_1,...,Q_n>
Q_tot := Σ_{i=1}^n Q_i
```

### 定义 8.6：有限投影算子

对任意非零非负词

```text
q = <q_1,...,q_n>,   q_tot := Σ_i q_i > 0
```

定义其有限投影：

```text
Π(q) := <Π_1(q),...,Π_n(q)>
Π_i(q) := q_i / q_tot
```

### 定义 8.7：精确溯因分布

若 `Q_tot > 0`，定义：

```text
P(R_i|E,γ)
=
Π_i(Q)
=
Q_i / Σ_l Q_l
```

### 命题 8.8：与理由级支持量一致

定义 8.7 与

```text
Norm(Ω_{E,γ}(R_i) ; <Ω_{E,γ}(R_1),...,Ω_{E,γ}(R_n)>)
```

一致。

### 证明

`Ω_{E,γ}(R_i)` 由路径权按 `δ_i(p_j)` 聚合而成；相关路径-轴贡献经 `Read_Σ` 量子化后写成 `m_{j,k} ε_Σ`；再由有效轴筛选与符号方向合成为 `Q_i ε_Σ`；有限归一时公共因子 `ε_Σ` 消去。

## 9. 概率分布的基本性质

### 定理 9.1：有限支撑性

精确溯因分布的支撑有限。

### 证明

候选头部、路径与自由轴都有限。

### 定理 9.2：非负性与归一性

若 `Q_tot > 0`，则：

```text
P(R_i|E,γ) ≥ 0
Σ_i P(R_i|E,γ) = 1
```

### 定理 9.3：正比例不变性

对任意 `c > 0`，

```text
Π(cq) = Π(q)
```

### 证明

```text
Π_i(cq) = cq_i / Σ_j cq_j = q_i / Σ_j q_j = Π_i(q)
```

### 定理 9.4：零位保持

若 `q_tot > 0`，则：

```text
Π_i(q) = 0  <=>  q_i = 0
```

### 证明

分母为正，故 `q_i/q_tot = 0` 当且仅当 `q_i = 0`。

### 定理 9.5：两点次序判据

若 `q_tot > 0`，则：

```text
Π_i(q) > Π_j(q)  <=>  q_i > q_j
Π_i(q) = Π_j(q)  <=>  q_i = q_j
Π_i(q) < Π_j(q)  <=>  q_i < q_j
```

### 推论 9.6：两点差分公式

```text
Π_i(q) - Π_j(q) = (q_i - q_j) / q_tot
```

### 定理 9.7：正坐标比值保持

若 `q_i > 0` 且 `q_j > 0`，则：

```text
Π_i(q) / Π_j(q) = q_i / q_j
```

### 定理 9.8：投影等价刻画

对任意两个非零非负词 `q,r`，下列两条等价：

```text
1. Π(q) = Π(r)
2. 存在 c > 0 使 q = cr
```

### 证明

若 `q = cr`，则由定理 9.3 立即得 `Π(q)=Π(r)`。

反之，若 `Π(q)=Π(r)`，记：

```text
q_tot := Σ_i q_i
r_tot := Σ_i r_i
```

则对每个 `i`，

```text
q_i / q_tot = r_i / r_tot
```

故

```text
q_i = (q_tot/r_tot) r_i
```

于是 `q = cr`，其中 `c := q_tot/r_tot > 0`。

### 定理 9.9：两两概率差的支持读出

记：

```text
D_{ij}(q) := q_i - q_j
W_{ij}(q) := Π_i(q) - Π_j(q)
```

则：

```text
W_{ij}(q) = D_{ij}(q) / q_tot
```

### 定理 9.10：均衡基线分解

对每个 `i`，有：

```text
Π_i(q)
=
1/n + (1/n) Σ_j W_{ij}(q)
```

### 证明

```text
Σ_j W_{ij}(q)
=
Σ_j (Π_i(q) - Π_j(q))
=
nΠ_i(q) - Σ_j Π_j(q)
=
nΠ_i(q) - 1
```

整理即得。

### 推论 9.11：支持词的基线分解

对每个 `i`，有：

```text
q_i
=
q_tot / n + (1/n) Σ_j D_{ij}(q)
```

### 证明

```text
Σ_j D_{ij}(q)
=
Σ_j (q_i - q_j)
=
nq_i - q_tot
```

整理即得。

### 定理 9.12：轴分解混合式

记：

```text
T_k := Σ_i Q_{i,k}
```

对每个满足 `T_k > 0` 的轴，定义：

```text
λ_k := T_k / Q_tot
π_{i|k} := Q_{i,k} / T_k
```

则：

```text
P(R_i|E,γ)
=
Σ_{k:T_k>0} λ_k π_{i|k}
```

### 证明

```text
P(R_i|E,γ)
= Q_i / Q_tot
= Σ_{k:T_k>0} Q_{i,k} / Q_tot
= Σ_{k:T_k>0} (T_k/Q_tot)(Q_{i,k}/T_k)
```

### 推论 9.13：轴锁定退化

若存在固定非负常数 `c_{i,k}` 与正标量 `s`，使：

```text
Q_{i,k} = s c_{i,k}
```

则分布由固定轴内份额唯一决定；整体尺度 `s` 不进入归一结果。

## 10. 差分性质

### 定义 10.1：两词比较

取两个非零非负支持词：

```text
q = <q_1,...,q_n>
q' = <q'_1,...,q'_n>
```

记：

```text
P := Π(q)
P' := Π(q')
Δ_i := q'_i - q_i
Δ_tot := Σ_i Δ_i = q'_tot - q_tot
S_{ij} := Δ_i - Δ_j
```

### 命题 10.1.1：一步合法性

两词比较保持在概率层中的充要条件是：

```text
q_i + Δ_i >= 0
```

对全部 `i` 成立，并且：

```text
q_tot + Δ_tot > 0
```

### 证明

由定义 10.1，

```text
q'_i = q_i + Δ_i
q'_tot = q_tot + Δ_tot
```

而 `q'` 仍须为非零非负支持词，故结论成立。

### 命题 10.1.2：原始增量的符号补偿

1. 若 `Δ_tot >= 0` 且存在某个 `i` 满足 `Δ_i < 0`，则存在某个 `j` 满足 `Δ_j > 0`。
2. 若 `Δ_tot <= 0` 且存在某个 `i` 满足 `Δ_i > 0`，则存在某个 `j` 满足 `Δ_j < 0`。

### 证明

若 `Δ_tot >= 0` 且存在某个 `i` 满足 `Δ_i < 0`，但对全部 `j` 都有 `Δ_j <= 0`，则

```text
Δ_tot = Σ_j Δ_j < 0
```

矛盾。第一条成立。第二条同理。

### 定义 10.2：有效信息贡献

对每个坐标 `i`，定义：

```text
E_i := Δ_i - P_i Δ_tot
```

### 定理 10.3：增量基线分解

对每个 `i`，有：

```text
Δ_i
=
Δ_tot / n + (1/n) Σ_j S_{ij}
```

### 证明

```text
Σ_j S_{ij}
=
Σ_j (Δ_i - Δ_j)
=
nΔ_i - Δ_tot
```

整理即得。

### 定理 10.4：精确变化公式

```text
P'_i - P_i = E_i / q'_tot
```

### 证明

```text
P'_i - P_i
= q'_i/q'_tot - q_i/q_tot
= (q_tot q'_i - q'_tot q_i) / (q_tot q'_tot)
= (q_tot(q_i+Δ_i) - (q_tot+Δ_tot)q_i) / (q_tot q'_tot)
= (q_totΔ_i - q_iΔ_tot) / (q_tot q'_tot)
= (Δ_i - (q_i/q_tot)Δ_tot) / q'_tot
= E_i / q'_tot
```

### 命题 10.4.1：一步仿射更新公式

对每个 `i`，有：

```text
P'_i
=
(q_tot / q'_tot) P_i
+ Δ_i / q'_tot
```

### 证明

由 `q'_i = q_i + Δ_i`，

```text
P'_i
=
q'_i / q'_tot
=
q_i / q'_tot
+ Δ_i / q'_tot
```

再代入

```text
q_i = q_tot P_i
```

即得。

### 命题 10.4.2：原始增量与概率变化的分离判据

对每个 `i`，有：

```text
P'_i > P_i  <=>  Δ_i > P_i Δ_tot
P'_i = P_i  <=>  Δ_i = P_i Δ_tot
P'_i < P_i  <=>  Δ_i < P_i Δ_tot
```

### 证明

由定义 10.2，

```text
E_i = Δ_i - P_i Δ_tot
```

再与推论 10.5 联立即得。

### 推论 10.4.3：原始负增量与概率上升相容

对每个 `i`，下列两条成立：

1. 若

```text
Δ_tot < 0
```

且

```text
P_i Δ_tot < Δ_i < 0
```

则

```text
P'_i > P_i
```

2. 若

```text
Δ_tot > 0
```

且

```text
0 < Δ_i < P_i Δ_tot
```

则

```text
P'_i < P_i
```

### 证明

两条都直接由命题 10.4.2 得到。

### 推论 10.5：提高与压低判据

```text
P'_i > P_i  <=>  E_i > 0
P'_i = P_i  <=>  E_i = 0
P'_i < P_i  <=>  E_i < 0
```

### 定理 10.6：零和律

```text
Σ_i E_i = 0
```

### 证明

```text
Σ_i E_i = Σ_i Δ_i - Δ_tot Σ_i P_i = Δ_tot - Δ_tot = 0
```

### 命题 10.6.1：有效信息的正负补偿

若 `E != 0`，则存在正分量当且仅当存在负分量。特别地，若某个 `i` 满足

```text
E_i < 0
```

则存在某个 `j` 满足

```text
E_j > 0
```

### 证明

由定理 10.6，

```text
Σ_i E_i = 0
```

若存在负分量而不存在正分量，则全部分量都不大于零，且至少有一个严格小于零，故总和严格小于零，矛盾。反向同理。

### 定理 10.7：径向-切向分解

若记：

```text
Δ := <Δ_1,...,Δ_n>
E := <E_1,...,E_n>
```

则：

```text
Δ = Δ_tot · P + E
```

### 定理 10.8：纯缩放判据

下列两条等价：

```text
1. 存在 c > 0 使 q' = cq
2. 对所有 i, E_i = 0
```

### 证明

若 `q' = cq`，则 `Δ_i = (c-1)q_i`，`Δ_tot = (c-1)q_tot`，故

```text
E_i = (c-1)q_i - (q_i/q_tot)(c-1)q_tot = 0
```

反之，若 `E_i = 0`，则

```text
Δ_i = P_i Δ_tot = (q_i/q_tot)Δ_tot
```

故

```text
q'_i = q_i + (Δ_tot/q_tot) q_i = c q_i
```

其中 `c := 1 + Δ_tot/q_tot > 0`。

### 定义 10.8.1：和项与分项

在分解

```text
Δ = Δ_tot · P + E
```

中，称

```text
Δ_tot · P
```

为该次变化的和项，称

```text
E
```

为该次变化的分项。

### 命题 10.8.2：和项只改总量

和项满足：

```text
Σ_i (Δ_tot P_i) = Δ_tot
```

并且：

```text
Π(q + Δ_tot P) = Π(q)
```

### 证明

由 `Σ_i P_i = 1`，立得

```text
Σ_i (Δ_tot P_i) = Δ_tot
```

又因

```text
q_i + Δ_tot P_i
= q_i + (q_i/q_tot)Δ_tot
= (1 + Δ_tot/q_tot) q_i
```

故

```text
q + Δ_tot P = (1 + Δ_tot/q_tot) q
```

由定理 10.8，归一投影不变。

### 命题 10.8.3：分项只改结构

分项满足：

```text
Σ_i E_i = 0
P' - P = E / q'_tot
```

### 证明

第一式即定理 10.6。第二式逐坐标由定理 10.4 给出。

### 推论 10.8.4：相对商式分解

若 `Δ_tot != 0`，定义：

```text
G_i := Δ_i / Δ_tot
ρ_i := E_i / Δ_tot
```

则对每个 `i`，有：

```text
G_i = P_i + ρ_i
```

### 证明

由定义 10.2，

```text
E_i = Δ_i - P_i Δ_tot
```

两边同除以 `Δ_tot` 即得。

### 命题 10.8.5：归一化混合条件

若

```text
Δ_tot > 0
```

并且对全部 `i` 都有

```text
Δ_i >= 0
```

则

```text
G = <G_1,...,G_n>
```

是概率词，并且：

```text
P'
=
(q_tot/q'_tot) P
+
(Δ_tot/q'_tot) G
```

其中：

```text
0 < Δ_tot/q'_tot < 1
q_tot/q'_tot + Δ_tot/q'_tot = 1
```

### 证明

由 `Δ_tot = Σ_i Δ_i`，得

```text
Σ_i G_i
=
[1/Δ_tot] Σ_i Δ_i
=
1
```

又由 `Δ_i >= 0` 与 `Δ_tot > 0`，得对全部 `i` 都有

```text
0 <= G_i <= 1
```

故 `G` 是概率词。再把

```text
Δ_i = Δ_tot G_i
```

代入命题 10.4.1 即得。又由

```text
q'_tot = q_tot + Δ_tot
```

以及 `q_tot > 0`, `Δ_tot > 0`，得

```text
0 < Δ_tot/q'_tot < 1
```

并且

```text
q_tot/q'_tot + Δ_tot/q'_tot
=
(q_tot + Δ_tot) / q'_tot
=
1
```

故该式确为一步归一化混合。

### 定理 10.9：两点差分变化公式

```text
E_i - E_j
=
(Δ_i - Δ_j) - ((q_i - q_j)/q_tot) Δ_tot
```

### 证明

由定义 10.2 直接相减，并代入 `P_i - P_j = (q_i-q_j)/q_tot`。

### 定理 10.10：E 的平均两两差分公式

对每个 `i`，有：

```text
E_i
=
(1/n) Σ_j (E_i - E_j)
```

故结合定理 10.9，得：

```text
E_i
=
(1/n) Σ_j
[
  (Δ_i - Δ_j)
  -
  ((q_i-q_j)/q_tot) Δ_tot
]
```

### 证明

由定理 10.6，

```text
Σ_j (E_i - E_j)
=
nE_i - Σ_j E_j
=
nE_i
```

### 推论 10.11：E 的均衡基线分解

对每个 `i`，有：

```text
E_i
=
(Δ_i - Δ_tot/n)
-
(P_i - 1/n) Δ_tot
```

### 证明

由定义 10.2，

```text
E_i = Δ_i - P_iΔ_tot
```

加减 `Δ_tot/n` 即得。

### 推论 10.12：E 的两两行和公式

记：

```text
W_{ij} := P_i - P_j
```

则：

```text
E_i
=
(1/n) Σ_j S_{ij}
-
(Δ_tot/n) Σ_j W_{ij}
```

### 定理 10.13：概率层两点差分演化

记：

```text
D_{ij}(q) := q_i - q_j
```

则：

```text
P'_i - P'_j
=
(D_{ij}(q) + S_{ij}) / q'_tot
```

### 证明

```text
P'_i - P'_j
=
(q'_i - q'_j) / q'_tot
=
(q_i - q_j + Δ_i - Δ_j) / q'_tot
=
(D_{ij}(q) + S_{ij}) / q'_tot
```

### 推论 10.14：两点排序翻转判据

若初始时

```text
D_{ij}(q) > 0
```

则变化后：

```text
P'_i > P'_j  <=>  Δ_i - Δ_j > -D_{ij}(q)
P'_i = P'_j  <=>  Δ_i - Δ_j = -D_{ij}(q)
P'_i < P'_j  <=>  Δ_i - Δ_j < -D_{ij}(q)
```

### 定理 10.15：边际增益比判据

若 `Δ_tot != 0`，定义：

```text
G_i := Δ_i / Δ_tot
```

则：

```text
Δ_tot > 0  =>  [P'_i > P_i  <=>  G_i > P_i]
Δ_tot > 0  =>  [P'_i = P_i  <=>  G_i = P_i]
Δ_tot > 0  =>  [P'_i < P_i  <=>  G_i < P_i]

Δ_tot < 0  =>  [P'_i > P_i  <=>  G_i < P_i]
Δ_tot < 0  =>  [P'_i = P_i  <=>  G_i = P_i]
Δ_tot < 0  =>  [P'_i < P_i  <=>  G_i > P_i]
```

### 证明

由

```text
E_i = Δ_tot (G_i - P_i)
```

及推论 10.5，按 `Δ_tot` 的符号分别判断即得。

### 定义 10.15.1：下标块

固定长度 `n`。取一个互不重复的非空真下标块：

```text
A = <a_1,...,a_r>
```

把其余下标按任意固定次序写成补块：

```text
bar(A) = <b_1,...,b_{n-r}>
```

### 定义 10.15.2：块和

对任意词 `x = <x_1,...,x_n>`，定义：

```text
x_A := Σ_{h=1}^r x_{a_h}
```

特别地，记：

```text
q_A := Σ_{h=1}^r q_{a_h}
P_A := Σ_{h=1}^r P_{a_h}
Δ_A := Σ_{h=1}^r Δ_{a_h}
E_A := Σ_{h=1}^r E_{a_h}
```

### 命题 10.15.3：块有效信息公式

```text
P_A = q_A / q_tot
E_A = Δ_A - P_A Δ_tot
```

### 证明

第一式由 `P_{a_h} = q_{a_h}/q_tot` 逐项求和即得。第二式由定义 10.2 逐项求和即得。

### 定理 10.15.4：块精确变化公式

```text
P'_A - P_A = E_A / q'_tot
```

### 证明

对定理 10.4 在 `a_1,...,a_r` 上逐项求和即得。

### 定理 10.15.5：块支持基线分解

```text
q_A - (r/n) q_tot
=
(1/n) Σ_{h=1}^r Σ_{l=1}^{n-r} D_{a_h b_l}(q)
```

### 证明

对推论 9.11 在 `a_1,...,a_r` 上逐项求和，得：

```text
q_A
=
(r/n) q_tot
+
(1/n) Σ_{h=1}^r Σ_j D_{a_h j}(q)
```

块内项满足：

```text
Σ_{h=1}^r Σ_{u=1}^r D_{a_h a_u}(q) = 0
```

故只剩跨块项。

### 定理 10.15.6：块增量基线分解

```text
Δ_A - (r/n) Δ_tot
=
(1/n) Σ_{h=1}^r Σ_{l=1}^{n-r} S_{a_h b_l}
```

### 证明

对定理 10.3 在 `a_1,...,a_r` 上逐项求和，并与定理 10.15.5 同理消去块内项，即得。

### 定理 10.15.7：块有效割和公式

```text
E_A
=
(1/n)
Σ_{h=1}^r Σ_{l=1}^{n-r}
(
  E_{a_h} - E_{b_l}
)
```

### 证明

由定理 10.6，

```text
Σ_{l=1}^{n-r} E_{b_l} = -E_A
```

故

```text
Σ_{h=1}^r Σ_{l=1}^{n-r} (E_{a_h} - E_{b_l})
=
(n-r)E_A + rE_A
=
nE_A
```

两边同除以 `n` 即得。

### 推论 10.15.8：块边际增益比判据

若 `Δ_tot != 0`，定义：

```text
G_A := Δ_A / Δ_tot
```

则：

```text
Δ_tot > 0  =>  [P'_A > P_A  <=>  G_A > P_A]
Δ_tot > 0  =>  [P'_A = P_A  <=>  G_A = P_A]
Δ_tot > 0  =>  [P'_A < P_A  <=>  G_A < P_A]

Δ_tot < 0  =>  [P'_A > P_A  <=>  G_A < P_A]
Δ_tot < 0  =>  [P'_A = P_A  <=>  G_A = P_A]
Δ_tot < 0  =>  [P'_A < P_A  <=>  G_A > P_A]
```

### 证明

由命题 10.15.3，

```text
E_A = Δ_tot (G_A - P_A)
```

再与定理 10.15.4 联立，并按 `Δ_tot` 的符号分别判断即得。

### 定义 10.15.9：块内有效信息

若 `q_A > 0`，则对每个 `h=1,...,r`，定义：

```text
P_{a_h|A} := q_{a_h} / q_A
E_{a_h|A} := Δ_{a_h} - P_{a_h|A} Δ_A
```

### 定理 10.15.10：层化分解

若 `q_A > 0`，则对每个 `h=1,...,r`，有：

```text
E_{a_h} = E_{a_h|A} + P_{a_h|A} E_A
```

### 证明

由定义 10.15.9，

```text
E_{a_h|A} + P_{a_h|A} E_A
=
Δ_{a_h} - P_{a_h|A} Δ_A + P_{a_h|A} (Δ_A - P_A Δ_tot)
=
Δ_{a_h} - (P_{a_h|A} P_A) Δ_tot
```

又

```text
P_{a_h|A} P_A
=
(q_{a_h}/q_A)(q_A/q_tot)
=
q_{a_h}/q_tot
=
P_{a_h}
```

故上式等于

```text
Δ_{a_h} - P_{a_h} Δ_tot = E_{a_h}
```

### 定义 10.16：双下标差分词

固定长度 `n`。把一组双下标分量

```text
M_{ij}
```

写成一个双下标差分词 `M`。

后续分别代入：

```text
D_{ij} := q_i - q_j
W_{ij} := P_i - P_j
S_{ij} := Δ_i - Δ_j
T_{ij} := E_i - E_j
```

### 推论 10.16.1：块量的差分写法

对定义 10.15.1 中的任意下标块 `A = <a_1,...,a_r>`，有：

```text
q_A - (r/n) q_tot
=
(1/n) Σ_{h=1}^r Σ_{l=1}^{n-r} D_{a_h b_l}

Δ_A - (r/n) Δ_tot
=
(1/n) Σ_{h=1}^r Σ_{l=1}^{n-r} S_{a_h b_l}

E_A
=
(1/n) Σ_{h=1}^r Σ_{l=1}^{n-r} T_{a_h b_l}
```

### 证明

前两式分别即定理 10.15.5 与定理 10.15.6。第三式由 `T_{a_h b_l} = E_{a_h} - E_{b_l}` 与定理 10.15.7 立即得到。

### 定理 10.17：双下标差分词的一致性律

若存在有限词

```text
x = <x_1,...,x_n>
```

使

```text
M_{ij} = x_i - x_j
```

则必有：

```text
M_{ii} = 0
M_{ij} = -M_{ji}
M_{ij} + M_{jk} = M_{ik}
```

### 证明

```text
M_{ii} = x_i - x_i = 0
M_{ij} = x_i - x_j = -(x_j - x_i) = -M_{ji}
M_{ij} + M_{jk} = (x_i-x_j) + (x_j-x_k) = x_i-x_k = M_{ik}
```

### 定理 10.18：一致性律的反向重建

若双下标差分词 `M` 满足：

```text
M_{ii} = 0
M_{ij} = -M_{ji}
M_{ij} + M_{jk} = M_{ik}
```

则必存在有限词

```text
x = <x_1,...,x_n>
```

使：

```text
M_{ij} = x_i - x_j
```

### 证明

定义

```text
x_i := (1/n) Σ_j M_{ij}
```

则

```text
x_i - x_k
=
(1/n) Σ_j (M_{ij} - M_{kj})
```

由三点可加律，

```text
M_{kj} = M_{ki} + M_{ij}
```

故

```text
M_{ij} - M_{kj}
=
M_{ij} - (M_{ki} + M_{ij})
=
-M_{ki}
=
M_{ik}
```

与 `j` 无关，因此

```text
x_i - x_k
=
(1/n) Σ_j M_{ik}
=
M_{ik}
```

即得

```text
M_{ij} = x_i - x_j
```

### 推论 10.19：行和重建

若 `M_{ij} = x_i - x_j`，则对每个 `i`，

```text
x_i
=
(1/n) Σ_j M_{ij}
+
c
```

其中 `c` 是共同平移常数。

### 证明

由

```text
(1/n) Σ_j M_{ij}
=
(1/n) Σ_j (x_i - x_j)
=
x_i - (1/n) Σ_j x_j
```

记

```text
c := (1/n) Σ_j x_j
```

即得

```text
x_i = (1/n) Σ_j M_{ij} + c
```

### 推论 10.20：四点恒等式

若 `M` 满足定理 10.18 的条件，则对任意 `i,j,k,l`，

```text
M_{ij} + M_{kl} = M_{il} + M_{kj}
```

### 证明

由 `M_{ij} = x_i - x_j`，

```text
M_{ij} + M_{kl}
=
(x_i-x_j) + (x_k-x_l)
=
(x_i-x_l) + (x_k-x_j)
=
M_{il} + M_{kj}
```

### 定义 10.21：基准位差分词

固定一个基准下标 `a`。对任意满足一致性律的双下标差分词 `M`，定义其基准位差分词：

```text
u_i^(a) := M_{ia}
```

### 定理 10.22：基准位生成式

对任意 `i,j`，有：

```text
M_{ij} = u_i^(a) - u_j^(a)
```

### 证明

由三点可加律，

```text
M_{ij} + M_{ja} = M_{ia}
```

故

```text
M_{ij} = M_{ia} - M_{ja} = u_i^(a) - u_j^(a)
```

### 推论 10.23：换基准位变换律

若改取基准下标 `b`，则：

```text
u_i^(b) = u_i^(a) - u_b^(a)
```

### 证明

由定理 10.22，

```text
u_i^(b)
=
M_{ib}
=
M_{ia} - M_{ba}
=
u_i^(a) - u_b^(a)
```

### 推论 10.24：概率差分词的基准位重建

对

```text
W_{ij} := P_i - P_j
```

固定基准下标 `a`，记

```text
u_i^(a) := W_{ia} = P_i - P_a
```

则对每个 `i`，

```text
P_i = u_i^(a) + (1 - Σ_h u_h^(a)) / n
```

### 证明

由定理 9.10，

```text
P_i = 1/n + (1/n) Σ_j W_{ij}
```

再由定理 10.22，

```text
W_{ij} = u_i^(a) - u_j^(a)
```

故

```text
Σ_j W_{ij}
=
Σ_j (u_i^(a) - u_j^(a))
=
n u_i^(a) - Σ_j u_j^(a)
```

代回即得。

### 推论 10.25：支持差分词的基准位重建

对

```text
D_{ij}(q) := q_i - q_j
```

固定基准下标 `a`，记

```text
v_i^(a) := D_{ia}(q) = q_i - q_a
```

则对每个 `i`，

```text
q_i = v_i^(a) + q_a
```

并且

```text
P_i = v_i^(a)/q_tot + q_a/q_tot
```

### 证明

由定义

```text
v_i^(a) = q_i - q_a
```

故

```text
q_i = v_i^(a) + q_a
```

两边同除以 `q_tot` 即得

```text
P_i = v_i^(a)/q_tot + q_a/q_tot
```

### 定理 10.26：概率基本分布表达式的完全等价族

对每个 `i`，下列诸式完全等价：

```text
1. P_i = q_i / q_tot

2. P_i = 1/n + (1/n) Σ_j W_{ij}

3. P_i = 1/n + (1/(n q_tot)) Σ_j D_{ij}(q)

4. P_i = u_i^(a) + (1 - Σ_h u_h^(a)) / n

5. P_i = v_i^(a)/q_tot + q_a/q_tot
```

其中

```text
W_{ij} = P_i - P_j
D_{ij}(q) = q_i - q_j
u_i^(a) = W_{ia}
v_i^(a) = D_{ia}(q)
```

### 证明

`1 <=> 2` 由定理 9.10。

`2 <=> 3` 由定理 9.9：

```text
W_{ij} = D_{ij}(q) / q_tot
```

`2 <=> 4` 由推论 10.24。

`1 <=> 5` 由推论 10.25 两式同除以 `q_tot` 即得。

### 定义 10.27：最小平移后的非负基准位词

设 `u^(a)` 是概率差分词 `W` 的某个基准位差分词，并记：

```text
m := min_i u_i^(a)
g_i := u_i^(a) - m
```

则：

```text
g_i >= 0
min_i g_i = 0
```

### 定理 10.28：最小合法性判据

基准位差分词 `u^(a)` 能生成一个合法概率分布，当且仅当：

```text
Σ_i g_i <= 1
```

其中 `g_i := u_i^(a) - min_h u_h^(a)`。

### 证明

由推论 10.24，设

```text
c := (1 - Σ_h u_h^(a)) / n
```

则

```text
P_i = u_i^(a) + c
```

合法性即对所有 `i` 有 `P_i >= 0`。这等价于最小坐标满足：

```text
min_i u_i^(a) + c >= 0
```

记 `m := min_i u_i^(a)`，则

```text
m + (1 - Σ_h u_h^(a))/n >= 0
```

等价于

```text
1 - Σ_h u_h^(a) + n m >= 0
```

又因为

```text
Σ_i g_i
=
Σ_i (u_i^(a) - m)
=
Σ_i u_i^(a) - n m
```

故合法性等价于

```text
Σ_i g_i <= 1
```

### 推论 10.29：规范概率重建式

在定理 10.28 的条件下，定义

```text
λ := (1 - Σ_i g_i) / n
```

则：

```text
P_i = g_i + λ
```

### 证明

由 `u_i^(a) = g_i + m` 与

```text
c = (1 - Σ_h u_h^(a))/n
```

可得

```text
P_i
=
u_i^(a) + c
=
g_i + m + (1 - Σ_h u_h^(a))/n
=
g_i + (1 - Σ_h g_h)/n
=
g_i + λ
```

### 推论 10.30：严格正性判据

全部概率严格为正，当且仅当：

```text
Σ_i g_i < 1
```

### 证明

由推论 10.29，最小概率位正是 `λ`，故全体严格为正当且仅当 `λ > 0`。

### 推论 10.31：零读数判据

存在某个 `i` 使 `P_i = 0`，当且仅当：

```text
Σ_i g_i = 1
```

并且此时：

```text
P_i = g_i
```

对所有 `i` 成立。

### 证明

由推论 10.29，

```text
P_i = g_i + λ
```

又 `g_i >= 0` 且 `min_i g_i = 0`，故最小概率位为 `λ`。存在零位当且仅当 `λ = 0`，即

```text
(1 - Σ_i g_i)/n = 0
<=> Σ_i g_i = 1
```

此时 `P_i = g_i`。

## 12. 收缩型

### 命题 12.1：超几何收缩

若附加：

```text
1. 总量子数固定为 T
2. 某类支持量子共有 K 个
3. 当前读出固定抽取 n 个量子
4. 抽取不放回
5. 可容抽取配置等权
```

则微配置平均收缩为：

```text
P(X=x)
=
C(K,x) C(T-K,n-x) / C(T,n)
```

### 命题 12.2：二项收缩

若附加：

```text
1. 有 n 个同型局部分流块
2. 每块只给两类结果
3. 每块局部支持比例同为 p
4. 各块在当前切面下独立复制
```

则收缩为：

```text
P(X=x) = C(n,x) p^x (1-p)^{n-x}
```

### 命题 12.3：泊松收缩

在命题 12.2 的结构上再取：

```text
n -> ∞
p -> 0
np -> λ
```

则得：

```text
P(X=x)=e^{-λ} λ^x / x!
```

### 命题 12.4：格点正态收缩

若总量子数来自大量弱相关或独立量子贡献之和，且单个贡献方差有限、无单通道支配整体，则总量子数的大尺度格点分布收缩为局部中心极限定理给出的格点正态包络。

### 命题 12.5：收缩型的从属地位

命题 12.1 至 12.4 都不是第一性对象；它们只是微配置结构在附加条件下的收缩型。

## 14. 总定理

在 `充足理由单子论.md` 的有限结构、有限权步、有限路径与有限归一前提下：

1. 严格由因推果属于规定层，不属于概率层。
2. 概率只属于由果溯因的截面归一读数。
3. 由 `K3`、`K4`、`R5`、`R9` 与普通单子的充足理由不可逆返可知：正贡献理由沿不可逆链向结果汇聚，在未满足闭合退化条件时不得被提前压成终点满值，且其满权不得失落；因此在进入 `Read_Σ` 之前必须保留独立的前读出载体层。非负实提升会把该层唯一压成终点读出，纯实提升又只把它并入方向层；故若前向支撑仍以数值载体表达，则它不能只取实数，必须取复数；进入概率层前仍必须经过实读出与量子化。
4. 精确溯因分布是净支持词的有限投影：

```text
P(R_i|E,γ) = Π_i(Q) = Q_i/Σ_lQ_l
```

5. 概率分布的最基本性质包括：

```text
非负性
归一性
正比例不变性
零位保持
两点次序判据
两点差分公式
轴分解混合式
```

6. 差分的最基本性质包括：

```text
E_i := Δ_i - P_iΔ_tot
P'_i - P_i = E_i/q'_tot
Σ_i E_i = 0
Δ = Δ_tot·P + E
P'_i > P_i  <=>  Δ_i > P_iΔ_tot
Δ_tot != 0  =>  G_i = P_i + ρ_i
```

7. 一切经典分布都只是该离散溯因结构在附加条件下的收缩型，不是其第一性基础。
