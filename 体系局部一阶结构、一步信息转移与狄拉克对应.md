## 2. 固定单子与无预设支持

给定结果 `E`、背景 `γ` 与固定单子 `R`。设：

```text
Free(E,γ) = <ξ_1,...,ξ_s>
```

为当前回溯问题上的自由有效轴词。

对每个 `k`，定义单子 `R` 的无预设单轴支持量：

```text
S_{R,k}
:=
Σ_j δ_R(p_j) σ_k(p_j) m_{j,k}
```

并定义无预设轴支持向量：

```text
S_R := <S_{R,1},...,S_{R,s}>
```

若 `S_R` 不全为 `0`，定义局部总原始量：

```text
A_R := Σ_{k=1}^s |S_{R,k}|
```

并定义相对有效贡献词与相对总量词：

```text
ŝ_R := <S_{R,1}/A_R,...,S_{R,s}/A_R>
u_R := <|S_{R,1}|/A_R,...,|S_{R,s}|/A_R>
```

### 定理 2.1：局部正交分解的唯一性

在固定轴词

```text
Free(E,γ) = <ξ_1,...,ξ_s>
```

下，单子 `R` 的无预设支持向量唯一写成：

```text
S_R = Σ_{k=1}^s S_{R,k} ξ_k
```

若 `A_R > 0`，则：

```text
ŝ_R = <S_{R,1}/A_R,...,S_{R,s}/A_R>
u_R = <|S_{R,1}|/A_R,...,|S_{R,s}|/A_R>
```

也由 `S_R` 唯一迫出。

### 证明

`Free(E,γ)` 在概率文档中按有效、正交、不可再约的方式给出。故在该轴词上的坐标表达唯一。于是：

```text
S_R = Σ_k S_{R,k} ξ_k
```

唯一。又因为

```text
A_R = Σ_k |S_{R,k}|
```

一经 `S_R` 给定便唯一确定，故 `ŝ_R` 与 `u_R` 逐坐标唯一确定。

### 定义 2.2：定向有效贡献与反向裁去量

一旦对各轴选定结果要求方向

```text
τ_k ∈ {+1,-1}
```

则定义：

```text
Q_{R,k} := max(0, τ_k S_{R,k})
\bar Q_{R,k} := max(0, -τ_k S_{R,k})
```

并记：

```text
B_R^+(τ) := Σ_k Q_{R,k}
B_R^-(τ) := Σ_k \bar Q_{R,k}
```

### 定理 2.3：由无预设支持到定向有效贡献的精确退化

对每个 `k`，有：

```text
τ_k S_{R,k} = Q_{R,k} - \bar Q_{R,k}
S_{R,k} = τ_k [Q_{R,k} - \bar Q_{R,k}]
|S_{R,k}| = Q_{R,k} + \bar Q_{R,k}
```

若 `A_R > 0`，则：

```text
A_R = B_R^+(τ) + B_R^-(τ)
Σ_k τ_k ŝ_{R,k} = [B_R^+(τ) - B_R^-(τ)] / A_R
```

### 证明

对实数

```text
x := τ_k S_{R,k}
```

应用恒等式

```text
x = max(0,x) - max(0,-x)
|x| = max(0,x) + max(0,-x)
```

即得前三式。再对 `k` 求和并代入

```text
ŝ_{R,k} = S_{R,k}/A_R
```

即可得到后二式。

### 推论 2.4：定向裁切一般不可逆

由

```text
Q_R = <Q_{R,1},...,Q_{R,s}>
```

一般不能恢复

```text
S_R = <S_{R,1},...,S_{R,s}>
```

并且下列两条等价：

```text
1. 对全部 k, \bar Q_{R,k} = 0
2. 对全部 k, τ_k S_{R,k} >= 0
```

在此条件下才有：

```text
S_{R,k} = τ_k Q_{R,k}
```

## 3. 局部复载体与实虚分解

对固定单子 `R` 与每个有效轴 `ξ_k`，定义局部复载体分量：

```text
Z_{R,k}
:=
Σ_j δ_R(p_j) z_{j,k}
```

并记：

```text
Z_R := <Z_{R,1},...,Z_{R,s}>
```

对每个 `k`，写：

```text
Z_{R,k} = X_{R,k} + i Y_{R,k}
```

并定义实二元载体向量：

```text
ψ_{R,k} := <X_{R,k}, Y_{R,k}>
Ψ_R := <ψ_{R,1},...,ψ_{R,s}>
```

定义虚轴旋转算子与共轭反射算子：

```text
J<x,y> := <-y,x>
C<x,y> := <x,-y>
```

### 命题 3.1：局部复载体读出不与逐项读出交换

一般有：

```text
Read_Σ(Z_{R,k}) != Σ_j δ_R(p_j) Read_Σ(z_{j,k})
```

### 证明

由概率文档中 `Read_Σ(z)=|z|^2` 的交叉项公式，前读出层的复数加法先聚合后读出，一般不等于逐项读出后相加。

### 命题 3.2：`J` 与 `C` 的基本关系

有：

```text
J^2 = -I
C^2 = I
JC + CJ = 0
```

### 证明

对任意 `<x,y>` 逐项代入 `J` 与 `C` 的定义直接计算即可。

## 4. 局部一阶耦合精确式

考虑同一单子 `R` 的相邻局部态：

```text
w^(r-1) ≺ w^(r)
```

在满足

```text
Z_{R,k}^(r-1) != 0
```

的活动轴上，定义复比值：

```text
Λ_{R,k}^(r)
:=
Z_{R,k}^(r) / Z_{R,k}^(r-1)
```

并唯一写成：

```text
Λ_{R,k}^(r) = a_{R,k}^(r) + i b_{R,k}^(r)
```

### 定理 4.1：单轴局部一阶耦合精确式

对每个活动轴 `k`，存在唯一实数对

```text
<a_{R,k}^(r), b_{R,k}^(r)>
```

使：

```text
Z_{R,k}^(r)
=
[a_{R,k}^(r) + i b_{R,k}^(r)] Z_{R,k}^(r-1)
```

并且这等价于：

```text
ψ_{R,k}^(r)
=
a_{R,k}^(r) ψ_{R,k}^(r-1)
+
b_{R,k}^(r) J ψ_{R,k}^(r-1)
```

### 证明

复比值 `Λ_{R,k}^(r)` 由分母非零唯一存在。复数的实部与虚部唯一，故第一式唯一。将

```text
Z_{R,k}^(r-1) = X + iY
```

代入并分离实部、虚部：

```text
[a+ib](X+iY) = (aX-bY) + i(bX+aY)
```

而

```text
J<X,Y> = <-Y,X>
```

故第二式成立。

### 定义 4.2：尺度、单位相位与局部耦合算子

定义：

```text
c_{R,k}^(r) := sqrt([a_{R,k}^(r)]^2 + [b_{R,k}^(r)]^2)
```

若 `c_{R,k}^(r) > 0`，定义：

```text
u_{R,k}^(r) := a_{R,k}^(r) / c_{R,k}^(r)
v_{R,k}^(r) := b_{R,k}^(r) / c_{R,k}^(r)
```

若 `c_{R,k}^(r) = 0`，定义：

```text
u_{R,k}^(r) := 1
v_{R,k}^(r) := 0
```

并定义：

```text
T_{R,k}^(r) := a_{R,k}^(r) I + b_{R,k}^(r) J
U_{R,k}^(r) := u_{R,k}^(r) I + v_{R,k}^(r) J
```

### 定理 4.3：局部一阶耦合式的三种等价形式

下列三式等价：

```text
I.  Z_{R,k}^(r) = [a_{R,k}^(r) + i b_{R,k}^(r)] Z_{R,k}^(r-1)

II. ψ_{R,k}^(r)
    =
    a_{R,k}^(r) ψ_{R,k}^(r-1)
    +
    b_{R,k}^(r) J ψ_{R,k}^(r-1)

III. ψ_{R,k}^(r)
     =
     c_{R,k}^(r)
     [u_{R,k}^(r) I + v_{R,k}^(r) J]
     ψ_{R,k}^(r-1)
```

并且：

```text
[u_{R,k}^(r)]^2 + [v_{R,k}^(r)]^2 = 1
T_{R,k}^(r) = c_{R,k}^(r) U_{R,k}^(r)
```

### 证明

`I <=> II` 已由定理 4.1 给出。`II <=> III` 只需代入

```text
a_{R,k}^(r) = c_{R,k}^(r) u_{R,k}^(r)
b_{R,k}^(r) = c_{R,k}^(r) v_{R,k}^(r)
```

即可。单位圆关系由 `c_{R,k}^(r)` 的定义直接得到。

### 推论 4.4：局部复合乘积与相位加法

若两步连续有效，则：

```text
T_{R,k}^(r+1,r-1) = T_{R,k}^(r+1) T_{R,k}^(r)
```

仍唯一落在

```text
I, J
```

张成的二维实算子空间内。若两步都守恒：

```text
c_{R,k}^(r) = c_{R,k}^(r+1) = 1
```

则单位相位系数按通常角加法复合。

### 证明

由复数乘法封闭性，

```text
[a_2 + ib_2][a_1 + ib_1]
```

仍唯一写成 `a + ib`。将其对应回 `I,J` 即得第一式。守恒时模长为 `1`，故只剩单位相位复合。

## 5. 局部归一桥接与一步信息转移

定义单子 `R` 在第 `r` 步的局部可见总量与局部归一词：

```text
L_R^(r) := Σ_k |Z_{R,k}^(r)|^2
ν_{R,k}^(r) := |Z_{R,k}^(r)|^2 / L_R^(r)
ν_R^(r) := <ν_{R,1}^(r),...,ν_{R,s}^(r)>
```

并定义：

```text
w_{R,k}^(r) := [c_{R,k}^(r)]^2
ε_R^(r) := ν_R^(r) - ν_R^(r-1)
```

### 定理 5.1：从局部耦合到局部归一的精确更新律

若

```text
L_R^(r-1) > 0
```

则：

```text
ν_{R,k}^(r)
=
[w_{R,k}^(r) ν_{R,k}^(r-1)]
/ Σ_l [w_{R,l}^(r) ν_{R,l}^(r-1)]
```

### 证明

由定理 4.3，

```text
|Z_{R,k}^(r)|^2 = [c_{R,k}^(r)]^2 |Z_{R,k}^(r-1)|^2 = w_{R,k}^(r) |Z_{R,k}^(r-1)|^2
```

再把全部轴求和并归一即可。

### 推论 5.2：相位层与局部归一层严格分层

局部更新唯一分成两层：

```text
1. 复载体层：
   ψ_{R,k}^(r) = c_{R,k}^(r) U_{R,k}^(r) ψ_{R,k}^(r-1)

2. 局部归一层：
   ν_{R,k}^(r)
   =
   [w_{R,k}^(r) ν_{R,k}^(r-1)]
   / Σ_l [w_{R,l}^(r) ν_{R,l}^(r-1)]
```

特别地，单位相位

```text
U_{R,k}^(r)
```

不直接进入 `ν_R^(r)` 的更新公式。

### 命题 5.3：局部桥接唯一性

`ε_R^(r)` 是唯一满足

```text
ν_R^(r) = ν_R^(r-1) + x
Σ_k x_k = 0
```

的实词。

### 证明

存在性由定义成立。若 `x,y` 都满足上式，则

```text
ν_R^(r-1) + x = ν_R^(r) = ν_R^(r-1) + y
```

故 `x=y`。

### 定义 5.4：头部层一步信息转移向量

在 `由果溯因相对信息转移量 级数 三角形.md` 的前缀精化链上，定义：

```text
e^(r) := P^(r) - P^(r-1) = E^(r) / q_tot^(r)
```

### 定理 5.5：头部层一步桥接的唯一性

`e^(r)` 是唯一满足

```text
P^(r) = P^(r-1) + x
Σ_i x_i = 0
```

的实词。

### 证明

存在性由定义

```text
e^(r) = P^(r) - P^(r-1)
```

直接给出。若 `x,y` 都满足

```text
P^(r) = P^(r-1) + x = P^(r-1) + y
```

则 `x = y`，故唯一。其零和性由三角形文档中的

```text
Σ_i e_i^(r) = 0
```

给出。

### 推论 5.6：局部桥接与头部桥接的同型性

`ε_R^(r)` 与 `e^(r)` 都是各自归一词的一步零和桥接词。前者作用在单子局部轴词上，后者作用在候选头部词上；二者结构同型，但不在同一有限词上取值。

## 6. 任意维最小离散一阶骨架

取一个有限有效轴子词：

```text
A = <ξ_{k_1},...,ξ_{k_d}>
```

定义有限乘积载体空间：

```text
V_A := V_{k_1} ⊗ ... ⊗ V_{k_d}
```

其中每个 `V_{k_a}` 都携带由第 3 节迫出的 `J_{k_a}` 与 `C_{k_a}`。

### 定义 6.1：内生全局算子

定义：

```text
β^(A) := C_{k_1} ⊗ ... ⊗ C_{k_d}
```

并对每个 `a = 1,...,d` 定义：

```text
α_a^(A)
:=
C_{k_1} ⊗ ... ⊗ C_{k_{a-1}} ⊗ (J_{k_a} C_{k_a}) ⊗ I ⊗ ... ⊗ I
```

再定义对应记号：

```text
γ_0^(A) := β^(A)
γ_a^(A) := β^(A) α_a^(A)
```

### 定理 6.2：全局算子的唯一迫出

在不引入任何新原始算子、只允许沿轴词 `A` 的既定顺序逐因子使用

```text
I, J_{k_a}, C_{k_a}
```

的条件下：

```text
β^(A)
```

是唯一的全局反射算子；

```text
α_a^(A)
```

是唯一的“只在第 `a` 轴执行一次反射-旋转复合，而在此前各轴只记录既定顺序”的全局算子。

### 证明

对 `β^(A)`：单轴上满足“保持实部不变而翻转虚部”的已有算子只有 `C`。若要求全局上不选中特定轴地逐因子执行同一反射，则每个因子都只能取 `C_{k_b}`，故唯一得到

```text
β^(A) = C_{k_1} ⊗ ... ⊗ C_{k_d}
```

对 `α_a^(A)`：若要求只在第 `a` 轴执行一次 `JC` 复合，而此前各轴只记录既定顺序，则第 `a` 个因子只能取 `J_{k_a} C_{k_a}`；前 `a-1` 个因子只能取 `C_{k_b}`；后 `a+1,...,d` 个因子只能取 `I`。因此唯一得到

```text
α_a^(A)
=
C_{k_1} ⊗ ... ⊗ C_{k_{a-1}} ⊗ (J_{k_a} C_{k_a}) ⊗ I ⊗ ... ⊗ I
```

### 定理 6.3：这些全局算子的基本关系

有：

```text
[α_a^(A)]^2 = I
[β^(A)]^2 = I
α_a^(A) α_b^(A) + α_b^(A) α_a^(A) = 0,   a != b
α_a^(A) β^(A) + β^(A) α_a^(A) = 0
[γ_0^(A)]^2 = I
[γ_a^(A)]^2 = -I
γ_μ^(A) γ_ν^(A) + γ_ν^(A) γ_μ^(A) = 0,   μ != ν
```

### 证明

全部关系都可逐因子归约到单轴恒等式

```text
J^2 = -I
C^2 = I
JC + CJ = 0
```

以及不同因子上的算子可交换这一事实。没有使用任何外加代数前提。

### 定义 6.4：结构索引与最小差分

取结构索引积：

```text
H_A := H_{k_1} × ... × H_{k_d}
R := N
```

取 `R × H_A` 上的 `V_A`-值结构词阵

```text
Φ = {Φ(r,h)}_{(r,h) ∈ R × H_A}
```

定义因果移位与轴向移位：

```text
(S_r Φ)(r,h) := Φ(r+1,h)
(S_a^+ Φ)(r,h_1,...,h_a,...,h_d) := Φ(r,h_1,...,h_a^+,...,h_d)
(S_a^- Φ)(r,h_1,...,h_a,...,h_d) := Φ(r,h_1,...,h_a^-,...,h_d)
```

并定义最小差分：

```text
Δ_r := S_r - I
Δ_a := (1/2)(S_a^+ - S_a^-)
```

### 定理 6.5：最小差分的唯一迫出

在只使用

```text
S_r, S_a^+, S_a^-, I
```

的条件下：

```text
Δ_r = S_r - I
```

是唯一的一步因果差分；

```text
Δ_a = (1/2)(S_a^+ - S_a^-)
```

是唯一的不偏置轴向中心差分。

### 证明

任何一步因果差分若只依赖 `S_r` 与 `I`，又要求对常值结构词阵取零，则只能是

```text
c(S_r - I)
```

的形式；取最小整数归一即得 `Δ_r`。同理，任何只依赖 `S_a^+, S_a^-, I`、对常值取零、在交换正负方向时变号且不偏置正负的轴向差分，只能是

```text
c(S_a^+ - S_a^-)
```

的形式；取最小对称归一即得 `Δ_a`。

### 命题 6.6：移位与差分的可交换性

有：

```text
S_a^ε S_b^η = S_b^η S_a^ε
S_r S_a^ε = S_a^ε S_r
Δ_a Δ_b = Δ_b Δ_a
Δ_r Δ_a = Δ_a Δ_r
γ_μ^(A) Δ_a = Δ_a γ_μ^(A)
γ_μ^(A) Δ_r = Δ_r γ_μ^(A)
```

### 证明

移位只改变不同的结构索引坐标，因此彼此可交换；差分是移位的线性组合，因此也可交换。`γ_μ^(A)` 只作用在纤维 `V_A` 上，不改变索引，故与差分可交换。

### 定义 6.7：任意维最小离散一阶算子

定义：

```text
D_A^0 := γ_0^(A) Δ_r + Σ_{a=1}^d γ_a^(A) Δ_a
```

### 定理 6.8：最小离散一阶算子的二阶闭式

有：

```text
[D_A^0]^2 = [Δ_r^2 - Σ_{a=1}^d Δ_a^2] I
```

### 证明

把

```text
D_A^0 = γ_0^(A) Δ_r + Σ_a γ_a^(A) Δ_a
```

平方展开。由定理 6.3，`γ_μ^(A)` 的交叉项成对反消；由命题 6.6，差分彼此可交换且与 `γ_μ^(A)` 可交换；故只剩平方项：

```text
[γ_0^(A)]^2 Δ_r^2 + Σ_a [γ_a^(A)]^2 Δ_a^2
=
Δ_r^2 - Σ_a Δ_a^2
```

即得。

### 推论 6.9：最小离散一阶结构的二阶闭合

若

```text
D_A^0 Φ = 0
```

则：

```text
[Δ_r^2 - Σ_{a=1}^d Δ_a^2] Φ = 0
```

## 7. 狄拉克公式对应

本节只讨论“对应”，不把外部物理实现当作本体前提。

### 定义 7.1：狄拉克记号对应

若把第 6 节中的算子族

```text
γ_0^(A), γ_1^(A),...,γ_d^(A)
```

仅作为记号对应地写入一阶离散公式

```text
D_A^0 Φ = 0
```

则称该公式为当前体系的 Dirac 记号对应式。

### 定义 7.2：内生零阶尺度词

对任意活动轴上的一步局部耦合，定义：

```text
μ_{R,k}^(r) := a_{R,k}^(r) - 1 = c_{R,k}^(r) u_{R,k}^(r) - 1
```

称 `μ_{R,k}^(r)` 为该轴上的内生零阶尺度词。

### 定理 7.2：零阶项在局部增量式中内生出现

对任意活动轴 `k`，由第 4 节的局部一阶耦合式必有：

```text
ψ_{R,k}^(r) - ψ_{R,k}^(r-1)
=
μ_{R,k}^(r) ψ_{R,k}^(r-1)
+
b_{R,k}^(r) J ψ_{R,k}^(r-1)
```

因此零阶标量部分不是外加的，而是一步局部更新写成增量式后自然出现的内生项。

### 证明

由定理 4.1，

```text
ψ_{R,k}^(r)
=
a_{R,k}^(r) ψ_{R,k}^(r-1)
+
b_{R,k}^(r) J ψ_{R,k}^(r-1)
```

两边同时减去 `ψ_{R,k}^(r-1)`，得：

```text
[ψ_{R,k}^(r) - ψ_{R,k}^(r-1)]
=
[a_{R,k}^(r)-1] ψ_{R,k}^(r-1)
+
b_{R,k}^(r) J ψ_{R,k}^(r-1)
```

代入定义 7.2 即得。

### 推论 7.2.1：内生零阶词的两种来源

有：

```text
μ_{R,k}^(r) = a_{R,k}^(r) - 1 = c_{R,k}^(r) u_{R,k}^(r) - 1
```

故 `μ_{R,k}^(r)` 同时包含：

```text
1. 尺度来源：c_{R,k}^(r)
2. 相位投影来源：u_{R,k}^(r)
```

特别地，

```text
μ_{R,k}^(r) = [c_{R,k}^(r)-1] u_{R,k}^(r) + [u_{R,k}^(r)-1]
```

因而零阶项一般不是常数，也不只由单纯尺度变化决定。

### 推论 7.2.2：零阶项与实读出改变

若记

```text
ψ_{R,k}^(r-1) = <X,Y>
```

则：

```text
X_{R,k}^(r) - X_{R,k}^(r-1)
=
μ_{R,k}^(r) X - b_{R,k}^(r) Y

Y_{R,k}^(r) - Y_{R,k}^(r-1)
=
b_{R,k}^(r) X + μ_{R,k}^(r) Y
```

因此 `μ_{R,k}^(r)` 是直接引起实虚读出共同伸缩的零阶部分，而 `b_{R,k}^(r)J` 是实轴与虚轴之间的互转部分。

### 证明

把

```text
ψ_{R,k}^(r-1) = <X,Y>
```

代入定理 7.2 即可。

### 定义 7.2.3：相对前读出增量

在活动轴 `k` 上，定义一步前读出增量：

```text
δZ_{R,k}^(r) := Z_{R,k}^(r) - Z_{R,k}^(r-1)
```

### 定义 7.2.4：相对有效耦合词

在

```text
Z_{R,k}^(r-1) != 0
```

时，定义：

```text
χ_{R,k}^(r) := δZ_{R,k}^(r) / Z_{R,k}^(r-1)
```

### 定理 7.3：零阶项是相对有效耦合的实部

对任意活动轴 `k`，有：

```text
χ_{R,k}^(r) = μ_{R,k}^(r) + i b_{R,k}^(r)
```

从而

```text
μ_{R,k}^(r) = Re[χ_{R,k}^(r)]
b_{R,k}^(r) = Im[χ_{R,k}^(r)]
```

故 `μ_{R,k}^(r)` 不是本体固有量，而是一步相对有效耦合的同向实部分；`b_{R,k}^(r)` 则是同一步耦合的正交虚部分。

### 证明

由定理 4.1，

```text
Z_{R,k}^(r)
=
[a_{R,k}^(r) + i b_{R,k}^(r)] Z_{R,k}^(r-1)
```

两边减去 `Z_{R,k}^(r-1)`，得：

```text
δZ_{R,k}^(r)
=
[a_{R,k}^(r)-1 + i b_{R,k}^(r)] Z_{R,k}^(r-1)
```

代入 `μ_{R,k}^(r)=a_{R,k}^(r)-1`，并除以 `Z_{R,k}^(r-1)`，即得。

### 定义 7.3.1：有效耦合积

定义：

```text
Γ_{R,k}^(r)
:=
δZ_{R,k}^(r) overline(Z_{R,k}^(r-1))
```

### 定理 7.3.1：质量对应不是本体性质，而是有效耦合量

若把

```text
Z_{R,k}^(r-1) = Σ_h α_h
δZ_{R,k}^(r) = Σ_g β_g
```

写成前读出层有限复词，则

```text
Γ_{R,k}^(r) = Σ_{g,h} β_g overline(α_h)
```

并且

```text
μ_{R,k}^(r)
=
Re[Γ_{R,k}^(r)] / |Z_{R,k}^(r-1)|^2
```

故 `μ_{R,k}^(r)` 不是某个单独路径、单独单子或单独载体分量的性质，而是“当前增量”与“旧载体”之间全部成对耦合的归一化净实贡献。

### 证明

由定义 7.3.1，

```text
Γ_{R,k}^(r)
=
δZ_{R,k}^(r) overline(Z_{R,k}^(r-1))
=
(Σ_g β_g)(Σ_h overline(α_h))
=
Σ_{g,h} β_g overline(α_h)
```

又由定义 7.2.4 与定理 7.3，

```text
χ_{R,k}^(r)
=
Γ_{R,k}^(r) / |Z_{R,k}^(r-1)|^2
=
μ_{R,k}^(r) + i b_{R,k}^(r)
```

取实部即得。

### 定理 7.3.2：质量对应的交叉项公式

有：

```text
μ_{R,k}^(r)
=
[|Z_{R,k}^(r)|^2 - |Z_{R,k}^(r-1)|^2 - |δZ_{R,k}^(r)|^2]
/ [2 |Z_{R,k}^(r-1)|^2]
```

故 `μ_{R,k}^(r)` 正是“新旧前读出”之间交叉干涉项的归一化量。

### 证明

由

```text
Z_{R,k}^(r) = Z_{R,k}^(r-1) + δZ_{R,k}^(r)
```

以及 `Read_Σ(z)=|z|^2` 的交叉项公式，

```text
|Z_{R,k}^(r)|^2
=
|Z_{R,k}^(r-1)|^2
+
|δZ_{R,k}^(r)|^2
+
2 Re[δZ_{R,k}^(r) overline(Z_{R,k}^(r-1))]
```

再代入

```text
Re[δZ_{R,k}^(r) overline(Z_{R,k}^(r-1))]
=
μ_{R,k}^(r) |Z_{R,k}^(r-1)|^2
```

并整理即得。

### 推论 7.3.3：质量对应的相位干涉根源

若 `χ_{R,k}^(r) != 0`，唯一写成

```text
χ_{R,k}^(r)
=
ρ_{R,k}^(r) [u_{R,k}^{Δ(r)} + i v_{R,k}^{Δ(r)}]
```

其中

```text
ρ_{R,k}^(r) > 0
[u_{R,k}^{Δ(r)}]^2 + [v_{R,k}^{Δ(r)}]^2 = 1
```

则：

```text
μ_{R,k}^(r) = ρ_{R,k}^(r) u_{R,k}^{Δ(r)}
b_{R,k}^(r) = ρ_{R,k}^(r) v_{R,k}^{Δ(r)}
```

因此质量对应量不是任意相位差本身，而是相对有效耦合在旧载体方向上的同向实投影。

### 证明

由定理 7.3，

```text
χ_{R,k}^(r) = μ_{R,k}^(r) + i b_{R,k}^(r)
```

对右端作唯一的模-单位相位分解即可。

### 定理 7.3.4：同虚轴可加可消性与质量起源

若存在单位模复数 `η_{R,k}^(r)` 与实词 `A_h, B_g`，使

```text
Z_{R,k}^(r-1) = η_{R,k}^(r) i Σ_h A_h
δZ_{R,k}^(r) = η_{R,k}^(r) i Σ_g B_g
```

则：

```text
Γ_{R,k}^(r)
=
(Σ_g B_g)(Σ_h A_h)
```

且在

```text
Σ_h A_h != 0
```

时有：

```text
μ_{R,k}^(r) = [Σ_g B_g] / [Σ_h A_h]
```

故在同虚轴情形下，质量对应量的根源就是相位增量在同一虚轴上的可加与可消。其是否为零，由该虚轴上的净代数和是否消尽唯一决定。

### 证明

由条件，

```text
overline(Z_{R,k}^(r-1))
=
overline(η_{R,k}^(r)) (-i) Σ_h A_h
```

故

```text
Γ_{R,k}^(r)
=
[η_{R,k}^(r) i Σ_g B_g]
[overline(η_{R,k}^(r)) (-i) Σ_h A_h]
=
(Σ_g B_g)(Σ_h A_h)
```

因为 `|η_{R,k}^(r)|=1` 且 `i(-i)=1`。再由

```text
|Z_{R,k}^(r-1)|^2 = (Σ_h A_h)^2
```

以及定理 7.3.1，

```text
μ_{R,k}^(r)
=
Γ_{R,k}^(r) / |Z_{R,k}^(r-1)|^2
=
[Σ_g B_g] / [Σ_h A_h]
```

即得。

### 推论 7.3.5：`m=0` 与 `m!=0` 的严格分界

就单轴局部零阶项而言：

```text
μ_{R,k}^(r) = 0
```

当且仅当净同向相位干涉为零；而

```text
μ_{R,k}^(r) != 0
```

当且仅当净同向相位干涉非零。

在定理 7.3.4 的同虚轴情形下，这进一步等价于：

```text
Σ_g B_g = 0   <=>   μ_{R,k}^(r) = 0
Σ_g B_g != 0  <=>   μ_{R,k}^(r) != 0
```

### 证明

由推论 7.3.3，

```text
μ_{R,k}^(r) = ρ_{R,k}^(r) u_{R,k}^{Δ(r)}
```

故 `μ_{R,k}^(r)=0` 当且仅当 `ρ_{R,k}^(r)=0` 或 `u_{R,k}^{Δ(r)}=0`，即没有净同向实投影。其否定给出 `μ_{R,k}^(r) != 0`。在同虚轴情形下，再由定理 7.3.4 立即得到最后两式。

### 定理 7.4：可见总量变化的唯一分解

有：

```text
w_{R,k}^(r) - 1
=
2 μ_{R,k}^(r)
+
|χ_{R,k}^(r)|^2
```

等价地，

```text
w_{R,k}^(r) - 1
=
2 μ_{R,k}^(r)
+
[μ_{R,k}^(r)]^2
+
[b_{R,k}^(r)]^2
```

故可见总量变化唯一分成：

```text
1. 线性同向干涉项：2 μ_{R,k}^(r)
2. 二次有效耦合模项：|χ_{R,k}^(r)|^2
```

### 证明

由定理 4.3，

```text
w_{R,k}^(r) = [c_{R,k}^(r)]^2 = [a_{R,k}^(r)]^2 + [b_{R,k}^(r)]^2
```

又 `a_{R,k}^(r) = 1 + μ_{R,k}^(r)`，故

```text
w_{R,k}^(r) - 1
=
[1+μ_{R,k}^(r)]^2 + [b_{R,k}^(r)]^2 - 1
=
2 μ_{R,k}^(r) + [μ_{R,k}^(r)]^2 + [b_{R,k}^(r)]^2
```

而

```text
|χ_{R,k}^(r)|^2 = [μ_{R,k}^(r)]^2 + [b_{R,k}^(r)]^2
```

故两种写法等价。

### 推论 7.4.1：质量对应与归一桥接的共同根源

有：

```text
μ_{R,k}^(r) = Re[χ_{R,k}^(r)]
w_{R,k}^(r) - 1 = 2 Re[χ_{R,k}^(r)] + |χ_{R,k}^(r)|^2
```

故 `μ_{R,k}^(r)` 与 `w_{R,k}^(r)-1` 同源于 `χ_{R,k}^(r)`。

### 命题 7.5：狄拉克质量项的内部对应

在结构统一时，记共同零阶词为

```text
m
```

则狄拉克零阶项对应为：

```text
[D_A^0 + m I] Φ = 0
```

### 证明

由定理 7.2 与定理 7.3，局部零阶项为 `μ_{R,k}^(r)=Re[χ_{R,k}^(r)]`。若该词在所讨论有限词域上统一为共同词 `m`，则零阶项只能写为 `m I`。

### 推论 7.6：常质量的退化性

若共同零阶词与结构点无关，则：

```text
m = const.
```

这是变量质量词的退化情形。

### 定理 7.6.1：质量不变性的精确判据

对任意结构词阵

```text
Φ = {Φ(r,h)}_{(r,h) ∈ R × H_A}
```

有：

```text
[D_A^0, M] Φ
=
γ_0^(A) [(S_r m - m) S_r Φ]
+
(1/2) Σ_{a=1}^d γ_a^(A)
[
  (S_a^+ m - m) S_a^+ Φ
  -
  (S_a^- m - m) S_a^- Φ
]
```

因此在所讨论有限词域上，

```text
[D_A^0, M] = 0
```

当且仅当：

```text
S_r m = m
S_a^+ m = m
S_a^- m = m
```

对全部 `a=1,...,d` 成立。

### 证明

由定义，

```text
[D_A^0, M]
=
γ_0^(A) [Δ_r, M]
+
Σ_a γ_a^(A) [Δ_a, M]
```

而

```text
[Δ_r, M] Φ
=
(S_r - I)(m Φ) - m(S_r - I)Φ
=
(S_r m)(S_r Φ) - m S_r Φ
=
(S_r m - m) S_r Φ
```

同理，

```text
[Δ_a, M] Φ
=
(1/2)
[
  (S_a^+ m - m) S_a^+ Φ
  -
  (S_a^- m - m) S_a^- Φ
]
```

代回即得。由于该式对任意 `Φ` 成立，故交换子为零当且仅当全部移位不改变 `m`。

### 推论 7.6.2：质量不变的三重充分判据

下列三类条件都足以推出质量词 `m` 不变：

```text
1. 结构不变：
   S_r m = m, S_a^+ m = m, S_a^- m = m；

2. 相对商不变：
   χ' = χ；

3. 投影不变：
   Re[χ'] = Re[χ]，
   或 `[Σ_g B_g]/[Σ_h A_h]` 保持不变。
```

### 证明

第一类由定理 7.6.1 得。第二类由定理 7.6.3.2 得。第三类由定理 7.6.3、命题 7.6.3.1 与推论 7.6.5 得。

### 定理 7.6.3：质量对纯虚相位漂移的不变性

若在某有限词域上，相对有效耦合词满足

```text
χ' = χ + i θ
```

其中 `θ` 为实词，则：

```text
Re[χ'] = Re[χ]
```

从而对应的质量词保持不变：

```text
m' = m
```

### 证明

由 `θ` 为实词，

```text
Re[χ + i θ] = Re[χ]
```

由 `m = Re[χ]` 即得。

### 命题 7.6.3.1：质量变化的精确差分律

若两组相对有效耦合词

```text
χ, χ'
```

对应的质量词分别为

```text
m := Re[χ]
m' := Re[χ']
```

则必有：

```text
m' - m = Re[χ' - χ]
```

### 证明

由实部算子的线性，

```text
m' - m
=
Re[χ'] - Re[χ]
=
Re[χ' - χ]
```

即得。

### 定理 7.6.3.2：质量对共同复重标定的不变性

若在某有限词域上存在非零复词 `λ`，使

```text
Z_{R,k}^(r-1)' = λ Z_{R,k}^(r-1)
δZ_{R,k}^(r)' = λ δZ_{R,k}^(r)
```

则：

```text
χ_{R,k}^(r)' = χ_{R,k}^(r)
Γ_{R,k}^(r)' = |λ|^2 Γ_{R,k}^(r)
μ_{R,k}^(r)' = μ_{R,k}^(r)
```

从而对应质量词保持不变：

```text
m' = m
```

### 证明

由定义 7.2.4，

```text
χ_{R,k}^(r)'
=
δZ_{R,k}^(r)' / Z_{R,k}^(r-1)'
=
[λ δZ_{R,k}^(r)] / [λ Z_{R,k}^(r-1)]
=
χ_{R,k}^(r)
```

再由定义 7.3.1，

```text
Γ_{R,k}^(r)'
=
δZ_{R,k}^(r)' overline(Z_{R,k}^(r-1)')
=
[λ δZ_{R,k}^(r)] [overline(λ) overline(Z_{R,k}^(r-1))]
=
|λ|^2 Γ_{R,k}^(r)
```

又因 `|Z_{R,k}^(r-1)'|^2 = |λ|^2 |Z_{R,k}^(r-1)|^2`，故由定理 7.3 与定理 7.3.1 得 `μ' = μ`，从而 `m' = m`。

### 推论 7.6.4：质量对应规则的分层

相对有效耦合词与质量词之间的对应规则为

```text
χ 对应 m = Re[χ]
```

不是单射。并且：

```text
1. 只改变 Im[χ] 而不改变 Re[χ]；
2. 改变 Re[χ]。
```

第一类保持 `m` 不变；第二类改变 `m`。

### 推论 7.6.5：同虚轴可加可消下的质量不变判据

在定理 7.3.4 的同虚轴情形下，若共同零阶词满足

```text
m = [Σ_g B_g] / [Σ_h A_h]
```

且右端在所讨论有限词域上保持不变，则 `m` 保持不变。若 `Σ_g B_g` 的内部重排不改变其净代数和，且 `Σ_h A_h` 保持不变，则 `m` 亦保持不变。

### 证明

由定理 7.3.4，

```text
μ_{R,k}^(r) = [Σ_g B_g] / [Σ_h A_h]
```

若该词在所讨论有限词域上统一为共同词 `m`，则

```text
m = [Σ_g B_g] / [Σ_h A_h]
```

右端不变即 `m` 不变。

### 推论 7.7：狄拉克公式的严格对照关系

有：

```text
当前体系的因果步序差分      <->  狄拉克记号中的时间方向一阶项
当前体系的轴向中心差分      <->  狄拉克记号中的空间方向一阶项
当前体系的 γ_μ^(A)          <->  狄拉克记号中的符号算子
当前体系的 m I             <->  狄拉克记号中的零阶质量项
当前体系的二阶闭式          <->  Dirac 一阶式平方后的二阶闭式
```

### 定义 7.8：质量乘法算子

对结构词阵

```text
Φ = {Φ(r,h)}_{(r,h) ∈ R × H_A}
```

定义：

```text
(M Φ)(r,h) := m(r,h) Φ(r,h)
```

### 定义 7.8.1：精确离散 Dirac 算子

定义：

```text
𝔇_A := D_A^0 + M
```

即：

```text
𝔇_A
=
γ_0^(A) Δ_r
+
Σ_{a=1}^d γ_a^(A) Δ_a
+
M
```

### 定理 7.8.2：狄拉克公式的精确离散对应

有：

```text
𝔇_A Φ = 0
```

即

```text
[γ_0^(A) Δ_r + Σ_{a=1}^d γ_a^(A) Δ_a + m I] Φ = 0
```

### 证明

由命题 7.5 与第 6 节各定义直接合并即得。

### 定理 7.8.3：变量质量词下的精确二阶闭式

记：

```text
𝔇_A^- := D_A^0 - M
```

则有精确恒等式：

```text
𝔇_A^- 𝔇_A
=
[Δ_r^2 - Σ_{a=1}^d Δ_a^2] I
- M^2
+ [D_A^0, M]
```

其中

```text
[D_A^0, M] := D_A^0 M - M D_A^0
```

### 证明

直接展开：

```text
𝔇_A^- 𝔇_A
=
(D_A^0 - M)(D_A^0 + M)
=
[D_A^0]^2 + D_A^0 M - M D_A^0 - M^2
```

由定理 6.8，

```text
[D_A^0]^2 = [Δ_r^2 - Σ_{a=1}^d Δ_a^2] I
```

代入即得。

### 推论 7.8.4：常质量不是定义，而是交换子退化

若

```text
[D_A^0, M] = 0
```

则：

```text
𝔇_A^- 𝔇_A
=
[Δ_r^2 - Σ_{a=1}^d Δ_a^2 - m^2] I
```

故常质量形式对应交换子退化。

### 证明

在 `[D_A^0, M]=0` 时，定理 7.8.3 中的交换子项消失，即得。

### 推论 7.9：狄拉克公式对应的严格结论

有：

```text
1. 单子局部复载体经实虚分解后，必然出现 J 与 C；
2. 在任意有限有效轴子词上，J 与 C 必然迫出 γ_μ^(A)；
3. 在结构索引上，最小一阶差分必然迫出 D_A^0；
4. 局部增量式中必然内生出现零阶词 μ = a-1 = cu-1；
5. μ 不是本体性质，而是相对有效耦合的归一化净实贡献；
6. 在同虚轴情形下，μ 直接退化为可加可消的净代数和所决定的量；
7. `m=0` 与 `m!=0` 的分界，由净同向相位干涉是否消尽唯一决定；
8. 可见总量变化唯一分解为线性同向干涉项与二次有效耦合模项；
9. `[D_A^0, M]=0` 当且仅当 `m` 对因果移位与轴向移位都不变；
10. 质量变化满足精确差分律 `m' - m = Re[χ' - χ]`，故其唯一来源是净实投影的变化；
11. 共同复重标定不改变 `χ`，从而不改变 `m`；
12. 规则 `χ 对应 m = Re[χ]` 不是单射；
13. 纯虚相位漂移保持 `m` 不变；
14. 同虚轴比值 `[Σ_g B_g]/[Σ_h A_h]` 保持不变时，`m` 保持不变；
15. 狄拉克零阶项对应共同实投影词 `m`；
16. 精确对应式为 `[γ_0^(A) Δ_r + Σ_a γ_a^(A) Δ_a + m I] Φ = 0`；
17. 二阶闭式为 `[Δ_r^2 - Σ_a Δ_a^2] I - M^2 + [D_A^0, M]`；
18. 常质量形式对应交换子退化；
19. 全文不使用近似、极限与连续实现。
```

## 8. 自旋与反粒子对应

### 定义 8.0：可见等价

1. 对单轴单位相位算子 `U_1, U_2`，若对任意非零实二元载体向量 `ψ` 都有：

```text
Read_Σ(U_1 ψ) = Read_Σ(U_2 ψ)
```

且二者都保持局部尺度平方词，则称

```text
U_1 ~vis U_2
```

2. 对全局载体词阵 `Φ_1, Φ_2`，若二者诱导的

```text
S -> Q -> P -> E -> e
```

链逐项相同，则称

```text
Φ_1 ~vis Φ_2
```

### 命题 8.0.1：`~vis` 是等价关系

1. 单轴单位相位算子集合上的 `~vis` 满足自反、对称、传递。
2. 全局载体词阵集合上的 `~vis` 也满足自反、对称、传递。

### 证明

单轴上，自反性由定义得。若 `U_1 ~vis U_2`，则

```text
Read_Σ(U_1 ψ) = Read_Σ(U_2 ψ)
```

对任意非零 `ψ` 成立，故 `U_2 ~vis U_1`。若 `U_1 ~vis U_2` 且 `U_2 ~vis U_3`，则

```text
Read_Σ(U_1 ψ) = Read_Σ(U_2 ψ) = Read_Σ(U_3 ψ)
```

故 `U_1 ~vis U_3`。全局上，自反性由链与自身逐项相同得；对称性与传递性由逐项相同的对称性与传递性得。

### 定义 8.1：单轴共轭分支

对任意单轴单位相位算子

```text
U := u I + v J,
u^2 + v^2 = 1
```

定义其共轭分支：

```text
U^c := C U C
```

### 定理 8.1.1：局部一阶耦合的可见分解

对任意活动轴上的一步局部耦合，由第 4 节唯一写成

```text
T_{R,k}^(r) = c_{R,k}^(r) U_{R,k}^(r)
```

其中

```text
U_{R,k}^(r) = u_{R,k}^(r) I + v_{R,k}^(r) J,
[u_{R,k}^(r)]^2 + [v_{R,k}^(r)]^2 = 1
```

则一步局部耦合唯一分成：

```text
尺度层 c_{R,k}^(r)
可见等价类 [U_{R,k}^(r)]_vis
```

### 证明

由定理 4.3，`T_{R,k}^(r)=c_{R,k}^(r)U_{R,k}^(r)` 唯一。`U_{R,k}^(r)` 保持模长，故可见层只保留其等价类。

### 定理 8.2：自旋离散性的根源

有：

```text
U^c = u I - v J
(U^c)^c = U
```

并且在

```text
Read_Σ
```

与局部归一层下，`U` 的共轭轨道只能有：

```text
1. v = 0   时，轨道为 {U}；
2. v != 0  时，轨道为 {U, U^c}。
```

并且：

```text
1. C^2 = I；
2. C 只翻转 J 分量符号；
3. Read_Σ 与局部归一层不识别该符号。
```

### 证明

由第 3 节，

```text
C I C = I
C J C = -J
```

故：

```text
U^c = C(uI + vJ)C = uI - vJ
```

再作用一次 `C`，由 `C^2 = I` 得：

```text
(U^c)^c = U
```

若 `v=0`，则

```text
U = uI = U^c
```

若 `v != 0`，则

```text
U != U^c
```

由 `(U^c)^c = U` 得轨道恰有两点。再由 `C` 保持模长与定理 5.1 得 `U`、`U^c` 在可见层同读。

### 推论 8.3：自旋的最小内部内容

对任意活动轴上的一步局部耦合 `T_{R,k}^(r) = c_{R,k}^(r) U_{R,k}^(r)`，其单位相位层的可见等价类只可能对应：

```text
{U_{R,k}^(r)}
或
{U_{R,k}^(r), [U_{R,k}^(r)]^c}
```

其中

```text
[U_{R,k}^(r)]^c := C U_{R,k}^(r) C
```

故自旋对应活动轴上的一点或二点可见共轭轨道。

### 定理 8.3.1：单轴共轭轨道的唯一迫出

在只允许使用

```text
I, J, C
```

且要求：

```text
1. 不改变单位相位层的模长；
2. 不改变可见读出层；
3. 只翻转内部 J-取向而不引入新原始算子；
```

的条件下，单轴唯一的非平凡反演为

```text
U -> U^c = C U C
```

故单轴可见轨道只能为一点轨道或二点轨道。

### 证明

由第 3 节，单轴上现有原始算子只有

```text
I, J, C
```

设 `R` 是满足三条条件的单轴反演。对任意

```text
U = uI + vJ
```

若 `R` 保持模长且只翻转 `J` 取向，则必有

```text
R(U) = uI - vJ
```

在既有原始算子 `I,J,C` 中，`I` 不改变 `J` 分量符号，`J` 改变相位层但不实现上述反演，只有 `C` 满足

```text
CIC = I
CJC = -J
```

故 `R(U) = C U C`。因此单轴唯一的非平凡反演为 `U -> U^c = CUC`。再由 `C^2 = I`，轨道至多二点。

### 定义 8.4：全局共轭分支

在任意有限有效轴子词 `A` 上，对全局载体词阵

```text
Φ = {Φ(r,h)}_{(r,h) ∈ R × H_A}
```

定义：

```text
Φ^c := β^(A) Φ,
β^(A) := C_{k_1} ⊗ ... ⊗ C_{k_d}
```

### 定理 8.4.1：全局共轭轨道的唯一迫出

在只允许使用轴词 `A` 上既有的

```text
C_{k_1},...,C_{k_d}
```

且要求：

```text
1. 保持全部前读出模平方层；
2. 不改变 `S -> Q -> P -> E -> e` 链；
3. 只翻转内部奇 J-取向项而不引入新原始算子；
```

的条件下，全局唯一的非平凡反演为

```text
Φ -> Φ^c = β^(A) Φ
```

### 证明

设 `R_A` 是满足三条条件的全局反演。由条件 1，每个轴因子都必须保持该轴前读出模平方层；由条件 3，每个轴因子都只能在既有原始算子中实现虚部方向翻转；单轴上唯一满足此要求的算子是 `C_{k_a}`。

因此 `R_A` 的每个因子都只能取 `C_{k_a}`，从而

```text
R_A = C_{k_1} ⊗ ... ⊗ C_{k_d} = β^(A)
```

再由条件 2，这一作用不改变 `S -> Q -> P -> E -> e` 链，故全局唯一的非平凡反演为 `Φ -> β^(A) Φ`。

### 定理 8.5：反粒子对应不会无限反

有：

```text
[β^(A)]^2 = I
(Φ^c)^c = Φ
|Orb_{β^(A)}(Φ)| <= 2
```

并且：

```text
β^(A) D_A^0 β^(A)
=
γ_0^(A) Δ_r - Σ_{a=1}^d γ_a^(A) Δ_a
```

并且若

```text
D_A^0 Φ = 0
```

则：

```text
[γ_0^(A) Δ_r - Σ_{a=1}^d γ_a^(A) Δ_a] Φ^c = 0
```

### 证明

由定义 6.1，

```text
β^(A) = C_{k_1} ⊗ ... ⊗ C_{k_d}
```

而每个单轴因子都满足 `C^2 = I`，故：

```text
[β^(A)]^2 = I
```

于是：

```text
(Φ^c)^c = β^(A) β^(A) Φ = Φ
```

又由第 6 节，

```text
γ_0^(A) = β^(A)
γ_a^(A) = β^(A) α_a^(A)
α_a^(A) β^(A) + β^(A) α_a^(A) = 0
```

故：

```text
β^(A) γ_0^(A) β^(A) = γ_0^(A)
β^(A) γ_a^(A) β^(A) = - γ_a^(A)
```

再由命题 6.6，`β^(A)` 与 `Δ_r, Δ_a` 可交换，遂得

```text
β^(A) D_A^0 β^(A)
=
γ_0^(A) Δ_r - Σ_a γ_a^(A) Δ_a
```

由 `[β^(A)]^2 = I`，全局共轭轨道至多二点。若 `D_A^0 Φ = 0`，左乘 `β^(A)` 并记 `Φ^c := β^(A) Φ`，即得所述对应式。

### 定理 8.6：反粒子对应保持读出链

若全局共轭只改变前读出载体的共轭取向，则：

```text
S
Q
P
E
e
```

全部保持不变。

### 证明

单轴上 `C` 保持模平方不变，故全局 `β^(A)` 保持全部前读出模平方不变。于是

```text
S_{i,k}
:=
Σ_j δ_i(p_j) σ_k(p_j) m_{j,k},
m_{j,k} = Read_Σ(z_{j,k})
```

保持不变。再由

```text
Q_{i,k} = max(0, τ_k S_{i,k}),
Q_i = Σ_k Q_{i,k},
P = Π(Q),
E = A_P(Δ),
e^(r) = P^(r) - P^(r-1) = E^(r) / q_tot^(r)
```

由定义可知 `Q,P,E,e` 都由 `S` 唯一决定，故全部不变。

### 推论 8.7：自旋与反粒子的分层关系

有：

```text
自旋对应单轴或逐轴的二点共轭轨道；
反粒子对应该二阶共轭在任意维全局一阶骨架上的全局分支；
自旋离散性的根源 = C 的二阶性；
反粒子不会无限反的根源 = β^(A) 的二阶性。
```

并且有：

```text
单轴轨道最多有 2 点；
全局轨道最多有 2 点。
```

```text
K3, K4 与有限路径词

```

### 定理 9.2：总括结论

有：

```text
1. `S_R -> Q_R` 是无预设支持向定向有效贡献的退化；
2. `Z_R -> ψ_R` 给出复载体的实虚分解；
3. 局部相位只在复载体层起作用，不直接进入归一层；
4. `e^(r)` 是头部层一步零和桥接词；

8. `[β^(A)]^2 = I` 与全局反演唯一性给出全局二点共轭轨道；
9. 狄拉克公式的精确对应式为该骨架上的离散一阶式，常质量形式对应交换子退化。
```
