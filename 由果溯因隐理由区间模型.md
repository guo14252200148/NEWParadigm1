# 由果溯因隐理由区间模型

## 1. 目的

本文处理以下问题：

1. 隐理由是什么。
2. 区间如何定义。
3. 区间何时退化。
4. 排序何时稳定。
5. 哪些压缩保持精确，哪些压缩丢失端点。
6. 有效轴如何制造区间。

全文只用有限词、有限词序、有限项运算。

## 3. 隐理由

### 定义 3.1：隐理由补全

固定结果 `E`、背景 `γ` 与某个合法截面角度 `κ`。把全部合法补全写成有限词序：

```text
Comp(E,γ) = <η_1,...,η_t>
```

其中每个 `η_a` 都满足：

```text
γ ⊗ η_a
```

给出 `E` 的某个合法更完整理由截面 `Sec(E,κ)`。

本文中，`η_a` 只表示当前背景尚未吸纳、但在更完整理由截面 `Sec(E,κ)` 中必须计入的存在信息；它不是时间隐状态。

### 定理 3.2：有限性

若与 `E` 相关的正贡献因子数、允许路径数、有效轴数、词序分岔数都有限，则 `Comp(E,γ)` 有限。

### 证明

补全只由有限对象按有限兼容规则组成。有限对象的有限写法总数有限。

## 4. 支持与归一

### 定义 4.1：候选头部

记：

```text
Head(E) = <R_1,...,R_n>
```

### 定义 4.2：补全下的路径吸纳

对每个 `η_a`、`R_i`、`p_j`，记：

```text
δ_i[a](p_j) := δ_{R_i}^{γ⊗η_a}(p_j) ∈ {0,1}
```

### 定义 4.3：补全下的单轴净支持

```text
Q_{i,k}[a]
=
max(
  0,
  τ_k · Σ_j δ_i[a](p_j) σ_k[a](p_j) m_{j,k}[a]
)
```

### 定义 4.4：补全下的总净支持

```text
Q_i[a] := Σ_k Q_{i,k}[a]
Q[a]   := <Q_1[a],...,Q_n[a]>
Q_tot[a] := Σ_i Q_i[a]
```

### 定义 4.5：补全下的归一读数

若 `Q_tot[a] > 0`，定义：

```text
P_i[a] := Q_i[a] / Q_tot[a]
P[a]   := <P_1[a],...,P_n[a]>
```

## 5. 精确区间

### 定义 5.1：精确区间

对每个 `R_i`，定义：

```text
underline_P_i(E|γ) := min_a P_i[a]
overline_P_i(E|γ)  := max_a P_i[a]
```

### 定理 5.2：存在性

若 `Comp(E,γ)` 非空且保留的每个下标 `a` 都满足 `Q_tot[a] > 0`，则 `underline_P_i(E|γ)` 与 `overline_P_i(E|γ)` 都存在。

### 证明

`a` 只在有限范围内取值，故极值存在。

### 命题 5.3：支持决定概率

若 `Q[a] = Q[b]`，则 `P[a] = P[b]`。

### 证明

代入同一分子与同一分母即可。

### 推论 5.4：去重原则

求精确区间时，只需保留不同的支持词。

## 6. 投影等价与退化

### 定义 6.1：投影等价

对任意 `a,b`，记：

```text
a ~ b
```

当且仅当存在 `c(a,b) > 0` 使：

```text
Q[a] = c(a,b) Q[b]
```

### 定理 6.2：投影等价刻画

下列两条等价：

```text
1. a ~ b
2. P[a] = P[b]
```

### 证明

若 `Q[a] = cQ[b]`，归一时 `c` 消去。

反之，若 `P[a] = P[b]`，记：

```text
s[a] := Σ_j Q_j[a]
s[b] := Σ_j Q_j[b]
```

则：

```text
Q_i[a] = P_i[a] s[a] = P_i[b] s[a] = (s[a]/s[b]) Q_i[b]
```

故二者只差同一正比例。

### 定理 6.3：区间退化判据

下列两条等价：

```text
1. 对所有 i, P_i[a] 与 a 无关
2. 全部 Q[a] 两两投影等价
```

### 证明

`2 => 1` 由定理 6.2 立即得到。

若 `1` 成立，则对任意 `a,b` 都有

```text
P[a] = P[b]
```

由定理 6.2，得

```text
a ~ b
```

即全部 `Q[a]` 两两投影等价。

### 推论 6.4：区间来源

区间是否出现，只取决于支持方向是否变化，不取决于总量是否单独变化。

## 7. 排序稳定

### 定义 7.1：支持差

```text
D_{ij}[a] := Q_i[a] - Q_j[a]
```

### 命题 7.2：单行排序判据

```text
P_i[a] > P_j[a]  <=>  D_{ij}[a] > 0
P_i[a] = P_j[a]  <=>  D_{ij}[a] = 0
P_i[a] < P_j[a]  <=>  D_{ij}[a] < 0
```

### 证明

`P_i[a]` 与 `P_j[a]` 共用同一正分母 `Q_tot[a]`。

### 定义 7.3：稳定边界

```text
L_{ij} := min_a D_{ij}[a]
U_{ij} := max_a D_{ij}[a]
```

### 定理 7.4：排序稳定判据

```text
L_{ij} > 0  =>  对所有 a, P_i[a] > P_j[a]
U_{ij} < 0  =>  对所有 a, P_i[a] < P_j[a]
L_{ij} <= 0 <= U_{ij}  =>  排序可能持平或翻转
```

### 推论 7.5：第一候选稳定

`R_i` 在全部补全下都保持第一候选，当且仅当对所有 `j != i` 都有：

```text
L_{ij} > 0
```

## 8. 精化与变化方向

### 定义 8.1：两行增量

取两个补全 `η_a,η_b`。记：

```text
Δ_i   := Q_i[b] - Q_i[a]
Δ_tot := Q_tot[b] - Q_tot[a]
```

若 `Δ_tot > 0`，定义：

```text
G_i(a -> b) := Δ_i / Δ_tot
```

### 定理 8.2：提高或压低判据

```text
P_i[b] > P_i[a]  <=>  G_i(a -> b) > P_i[a]
P_i[b] = P_i[a]  <=>  G_i(a -> b) = P_i[a]
P_i[b] < P_i[a]  <=>  G_i(a -> b) < P_i[a]
```

### 证明

直接通分：

```text
P_i[b] - P_i[a]
=
Q_tot[a] Δ_i - Q_i[a] Δ_tot
---------------------------------
Q_tot[a] (Q_tot[a] + Δ_tot)
```

分母为正，故符号由

```text
Δ_i/Δ_tot - Q_i[a]/Q_tot[a]
```

决定。

## 9. 支持边界与闭式外包络

### 定义 9.1：支持边界

若对全部 `a` 都有：

```text
q_i^- <= Q_i[a] <= q_i^+
```

则称 `<q_i^-, q_i^+>` 为 `R_i` 的支持边界。

### 定理 9.2：闭式外包络公式

若相应分母严格为正，即

```text
q_i^- + Σ_{j!=i} q_j^+ > 0
q_i^+ + Σ_{j!=i} q_j^- > 0
```

则：

```text
underline_P_i^{env}
=
q_i^- / (q_i^- + Σ_{j!=i} q_j^+)

overline_P_i^{env}
=
q_i^+ / (q_i^+ + Σ_{j!=i} q_j^-)
```

### 证明

记

```text
f_i(x_1,...,x_n) := x_i / Σ_j x_j
```

并在坐标盒

```text
B := {x : q_j^- <= x_j <= q_j^+}
```

上考察 `f_i`。

先证下界。对任意 `x ∈ B`，有

```text
x_i >= q_i^-
x_j <= q_j^+,  j != i
```

故

```text
f_i(x)
=
x_i / (x_i + Σ_{j!=i} x_j)
>=
x_i / (x_i + Σ_{j!=i} q_j^+)
```

记

```text
S_i^+ := Σ_{j!=i} q_j^+
```

映射

```text
t -> t / (t + S_i^+)
```

在

```text
t >= 0,  t + S_i^+ > 0
```

上单调递增，故

```text
x_i / (x_i + Σ_{j!=i} q_j^+)
>=
q_i^- / (q_i^- + Σ_{j!=i} q_j^+)
```

于是

```text
f_i(x)
>=
q_i^- / (q_i^- + Σ_{j!=i} q_j^+)
```

且当

```text
x_i = q_i^-
x_j = q_j^+,  j != i
```

时取等。

再证上界。对任意 `x ∈ B`，有

```text
x_i <= q_i^+
x_j >= q_j^-,  j != i
```

故

```text
f_i(x)
=
x_i / (x_i + Σ_{j!=i} x_j)
<=
x_i / (x_i + Σ_{j!=i} q_j^-)
<=
q_i^+ / (q_i^+ + Σ_{j!=i} q_j^-)
```

其中最后一步同样由

```text
t -> t / (t + Σ_{j!=i} q_j^-)
```

的单调递增性得到。且当

```text
x_i = q_i^+
x_j = q_j^-,  j != i
```

时取等。故定理成立。

### 命题 9.3：外包络非精确端点

定理 9.2 给出的是严格外包络，不是无条件精确端点。

### 证明

真实支持只来自有限个 `Q[a]`；坐标盒上的极值未必由某个真实 `Q[a]` 取到。

### 定义 9.4：端点可实现

若存在下标 `a_i^-` 与 `a_i^+` 使：

```text
Q_i[a_i^-] = q_i^-
Q_j[a_i^-] = q_j^+,  j != i

Q_i[a_i^+] = q_i^+
Q_j[a_i^+] = q_j^-,  j != i
```

则称 `R_i` 的两端点可实现。

### 推论 9.5：端点重合判据

若 `R_i` 满足端点可实现，则：

```text
underline_P_i(E|γ) = underline_P_i^{env}
overline_P_i(E|γ)  = overline_P_i^{env}
```

## 10. 词序与非对易

### 定义 10.1：隐藏原子词

记最小隐藏原子为：

```text
Z(E,γ) = <z_1,...,z_r>
```

补全一般写成兼容词：

```text
η = w(z_1,...,z_r)
```

### 命题 10.2：最一般形式

对每个 `R_i`，最一般只能写成：

```text
Q_i[a] = F_i(w_a)
```

不能先验写成线性和。

### 定理 10.3：线性化条件

只有当同时满足：

```text
1. 原子词两两兼容
2. 吸纳顺序不改支持值
3. 不同原子词的贡献可分离
```

时，才可写成：

```text
Q_i[a] = q_i + Σ_b c_{b,i} x_b[a]
```

### 定理 10.4：计数压缩失真判据

若存在两个补全词 `w_u,w_v` 满足：

```text
w_u 与 w_v 的原子出现次数逐项相同
但 Q[w_u] != Q[w_v]
```

则只按计数压缩会丢失真实支持；因此真实区间、真实端点或真实排序可能消失。

### 推论 10.5：保真压缩对象

当词序或非对易结构真实存在时，精确对象不是计数，而是兼容词本身。

## 11. 有效轴机制

### 定义 11.1：轴分解

对每个 `a` 与 `R_i`，有：

```text
Q_i[a] = Σ_{k=1}^s Q_{i,k}[a]
```

### 定义 11.2：轴激活词

```text
Act_i[a] := <act_{i,1}[a],...,act_{i,s}[a]>
```

其中：

```text
act_{i,k}[a] = 1  <=>  Q_{i,k}[a] > 0
act_{i,k}[a] = 0  <=>  Q_{i,k}[a] = 0
```

### 定义 11.3：轴配比词

若 `Q_i[a] > 0`，定义：

```text
ρ_i[a]
:=
<Q_{i,1}[a]/Q_i[a],...,Q_{i,s}[a]/Q_i[a]>
```

### 定理 11.4：轴内刚性分解判据

固定某个 `R_i`。下列两条等价：

```text
1. `Act_i[a]` 与 `a` 无关，且对一切满足 `Q_i[a] > 0` 的 `a`，`ρ_i[a]` 也与 `a` 无关
2. 存在固定非负常数 `c_{i,1},...,c_{i,s}`，以及对每个 `a` 的正标量 `s_i[a]`，使：
   Q_{i,k}[a] = s_i[a] c_{i,k}
```

### 证明

`2 => 1`。

若

```text
Q_{i,k}[a] = s_i[a] c_{i,k}
```

且

```text
s_i[a] > 0
```

则

```text
Q_{i,k}[a] > 0  <=>  c_{i,k} > 0
```

故 `Act_i[a]` 与 `a` 无关。

若对某个 `a` 有

```text
Q_i[a] > 0
```

则

```text
Q_i[a] = Σ_k Q_{i,k}[a] = s_i[a] Σ_k c_{i,k}
```

从而

```text
Q_{i,k}[a] / Q_i[a]
=
c_{i,k} / Σ_l c_{i,l}
```

与 `a` 无关，故 `ρ_i[a]` 也与 `a` 无关。

`1 => 2`。

分两种情形。

第一种，公共激活词全为 `0`。则对全部 `a,k` 都有

```text
Q_{i,k}[a] = 0
```

取

```text
c_{i,k} = 0
s_i[a] = 1
```

结论成立。

第二种，公共激活词不全为 `0`。此时存在某个轴下标 `k_0`，使对全部 `a` 都有

```text
Q_{i,k_0}[a] > 0
```

故对全部 `a`，

```text
Q_i[a] = Σ_k Q_{i,k}[a] > 0
```

于是 `ρ_i[a]` 对全部 `a` 都有定义，且按假设与 `a` 无关。取任意固定下标 `a_0`，定义

```text
c_{i,k} := ρ_{i,k}[a_0]
s_i[a] := Q_i[a]
```

则对全部 `a,k`，

```text
Q_{i,k}[a]
=
Q_i[a] ρ_{i,k}[a]
=
s_i[a] c_{i,k}
```

故得 `2`。

### 推论 11.4.1：区间的轴层来源

若对每个 `R_i` 都满足定理 11.4 的条件，则存在固定非负常数

```text
C_i := Σ_k c_{i,k}
```

与正标量 `s_i[a]`，使：

```text
Q_i[a] = s_i[a] C_i
```

因此，在全部轴内结构都冻结之后，区间是否继续出现，只取决于候选间总支持相对尺度

```text
<s_1[a] C_1,...,s_n[a] C_n>
```

是否仍随 `a` 改变。

### 证明

由定理 11.4，

```text
Q_{i,k}[a] = s_i[a] c_{i,k}
```

对 `k` 求和即得

```text
Q_i[a] = s_i[a] Σ_k c_{i,k} = s_i[a] C_i
```

故结论成立。

### 定理 11.5：轴锁定判据

若存在固定非负常数 `c_{i,k}` 与正标量 `s[a]`，使对所有 `a,i,k` 都有：

```text
Q_{i,k}[a] = s[a] c_{i,k}
```

则全部 `Q[a]` 两两投影等价，故全部区间退化为点。

### 证明

有：

```text
Q_i[a] = Σ_k Q_{i,k}[a] = s[a] Σ_k c_{i,k}
```

故全部 `Q[a]` 只差同一正比例。

## 12. 计算步骤

对给定 `E` 与 `γ`：

1. 写出候选头部 `Head(E)=<R_1,...,R_n>`。
2. 写出全部合法补全词序 `Comp(E,γ)=<η_1,...,η_t>`。
3. 对每个 `a,i,k` 计算 `Q_{i,k}[a]`。
4. 合成 `Q_i[a]` 与 `Q[a]`。
5. 去掉重复的 `Q[a]`。
6. 对每个保留下标 `a` 作归一，得到 `P[a]`。
7. 对 `a` 取极值，得到 `underline_P_i(E|γ)` 与 `overline_P_i(E|γ)`。
8. 若只知支持边界，则先算 `underline_P_i^{env}` 与 `overline_P_i^{env}`，再检查端点是否可实现。
9. 若存在词序效应，则不得按计数压缩。

## 13. 总结定理

在 `充足理由单子论.md` 与 `由果溯因概率论.md` 的有限结构理论下，可直接得到：

1. 隐理由是补全，不是时间隐状态。
2. 精确区间是有限归一读数上的真实极值。
3. 支持词决定概率词。
4. 两个概率词相同，当且仅当对应支持词只差同一正比例。
5. 区间退化为点，当且仅当全部支持词两两投影等价。
6. 排序稳定只由支持差符号稳定决定。
7. 边际增益比给出精化后读数提高或压低的充要判据。
8. 在相应分母为正时，闭式公式给出外包络；但它不是无条件精确端点。
9. 词序或非对易存在时，压成计数会失真。
10. 有效轴上，轴激活与轴配比刻画候选内部轴结构；在轴内结构冻结后，区间是否继续出现，还取决于候选间总支持相对尺度是否改变。

故整个模型可压成：

```text
补全词序
-> 支持词
-> 概率词
-> 精确区间
```

这已足以给出正式、简明、可验算、且无集合论外壳的隐理由区间模型。
