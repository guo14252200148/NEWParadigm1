# 充足理由单子论（严格符号化稿）

## 1. 目的

本文把本系统整理为一份严格基于既有基底的符号化稿。全文遵循以下原则：

1. 只使用有限结构、有限序列、排列-复合、关系代数，不诉诸集合论与实无穷。
2. 所有对象先定义，后使用；所有判据先写成 0-1 形式，后进入定理链。
3. 局部切面判断与全局决定性判断严格区分。
4. 因果不以时序为基础，但切面上的回溯闭合与否受时层可达性约束。

## 2. 书写约定

### 2.1 基本符号

```text
alpha, beta, γ, nu, Theta : 单子
R => alpha                    : R 是 alpha 的充足理由头部
alpha -> beta                 : alpha 是 beta 的部分原因
alpha ⊗ beta                  : 结构化复合
alpha ⇝ beta                  : 存在因果路径
w(alpha)                      : 单子 alpha 的权重
P(alpha | γ)                  : 条件权重
1                             : 宇宙总权重常数
```

### 2.2 有限对象

本文只允许以下有限对象：

1. 有限单子词\
   例如 `r_1 ⊗ r_2 ⊗ ... ⊗ r_n`
2. 有限连接式\
   例如 `C_alpha`
3. 有限路径词\
   例如 `x_1 rel_1 x_2 rel_2 ... rel_{l-1} x_l`
4. 有限路径表\
   例如 `<p_1, p_2, ..., p_m>`
5. 有限权重词\
   例如 `<ω(p_1), ..., ω(p_m)>`

因此，所有乘积与求和都默认是有限项。

### 2.3 有限运算记号

为避免把系统重新拖回连续数值背景，全文统一使用以下有限运算记号：

```text
Couple(x,y)                    : 把两个有限权项耦合成一个有限复合权项
Pool(<x_1,...,x_n>)            : 把有限权词压成一个有限汇聚权项
Take(b,x)                      : 若 b=1 则取 x，若 b=0 则取零权
All(<b_1,...,b_n>)             : 若整词全为 1 则回 1，否则回 0
Norm(x ; <x_1,...,x_n>)        : 把 x 在有限权词中归一成一个有限比例词
Live(x)                  : 若 x 不是零权词则回 1，否则回 0
Zero(x)                  : 若 x 是零权词则回 1，否则回 0
Full(x)                  : 若 x 是满权词则回 1，否则回 0
```

其中：

- `Pool` 代替普通加和；
- `Couple` 代替普通乘法型耦合；
- `All` 代替一串 0-1 判据的逐位合闸；
- `Norm` 保留概率语言，但底层只表示有限归一比例，不诉诸连续区间；
- `Live / Zero / Full` 分别给出非零权、零权、满权这三个离散判词。

### 2.4 判词语法表

全文中所有 0-1 判据统一视作**判词**。判词只输出两种离散值：

```text
1 : 判词成立的理由充足
0 : 判词成立的理由不充足
```

其统一语法写为：

```text
Pred ::=
  Live(x)
| Zero(x)
| Full(x)
| Past_τ(L)
| Now_τ(L)
| Future_τ(L)
| Len(p,q)
| Typ(p,q)
| Lay_τ(p,q)
| Card_E(a,b)
| Eq_i^E(a,b)
| Com_τ(a,b | E,γ)
| Par_i^E(a,b;τ)
| Par_E(a,b;τ)
| EqW_E(a,b;τ)
| Com_E(a,b;τ)
```

按角色分组，可写成下表：

```text
权态判词 :
  Live(x), Zero(x), Full(x)

层位判词 :
  Past_τ(L), Now_τ(L), Future_τ(L)

路径骨架判词 :
  Len(p,q), Typ(p,q), Lay_τ(p,q)

并行判词 :
  Card_E(a,b), Par_i^E(a,b;τ), Par_E(a,b;τ)

权比判词 :
  Eq_i^E(a,b), EqW_E(a,b;τ)

截面判词 :
  Com_τ(a,b | E,γ)

全局判词 :
  Com_E(a,b;τ)
```

约定如下：

- 判词表只负责给出离散判定，不负责给出概率。
- `Com_τ` 是截面单子上的判词，因此只在截面权重分布已经被读出时才有意义。
- `Com_E` 是局部因果网络整体上的决定性判词，它给出结构是否通约，但它本身不携带概率。
- 概率只属于截面单子的权重分布；对全局决定性结构，不引入概率概念。

### 2.5 第一性极简核

提取最接近第一性的基础逻辑，那么原符只需保留：

```text
alpha          : 被规定的有限位点
R => alpha     : 充足规定
⊗              : 结构复合
w(alpha)       : 权词
1              : 满权常词
```

在这组原符下，可把整个系统压缩为四条根式：

```text
K1 规定式 :
  Live(w(alpha)) = 1 => R => alpha

K2 同一式 :
  Eff(alpha) = Eff(beta) <=> alpha = beta

K3 传权式 :
  若 R_alpha := (r_1 ⊗ ... ⊗ r_n ; C_alpha),
  则
  w(alpha)
  =
  Pool(<Take(Con_alpha(r_1),w(r_1)), ..., Take(Con_alpha(r_n),w(r_n))>)

K4 闭组守恒式 :
  若 G_κ := <g1, ..., gn> 是同一组单子及其视角单元下
  相关且闭合的有限权词，
  则
  Full(Pool(<w(g1), ..., w(gn)>)) = 1
```

这四条的角色分别是：

- `K1` 给出存在根据：凡实显者，必被充足规定。
- `K2` 给出同一根据：同一性不来自名字，而来自有效理由结构。
- `K3` 给出权重根据：结果不是凭空出现，而是由正贡献理由的权词汇聚而成。
- `K4` 给出守恒根据：对同组、相关、闭合的权词整体，满权守恒成立。

在这个极简核里：

- 不把截面直接写成第一性原符；
- 不把概率直接写成第一性原符；
- `->` 不是原符，而是 `K3` 中正贡献关系的派生记号；
- 局部动态不是另一种本体，而是全局权词在有限理由展开中的读出次序；
- 概率不是全局性质，而只是这种局部读出在有限归一之后的表示。

因此，若要把它作为最接近第一性的基础逻辑，可以简写为：

```text
存在 = 被充足规定
同一 = 有效理由结构同一
生成 = 权词沿正贡献结构汇聚
守恒 = 相关闭组中的满权不失
```

## 3. 原始对象

### 定义 1：单子

单子不是脱离关系独立存在的对象，而是某个有限对象、事件或概念规则在充足理由律实际使用中被规定出来的有限位点。

形式条件：

```text
Live(w(alpha)) = 1 => 存在某个 R 使得 R => alpha
```

故：

- 不存在独立单子；
- 单子至少涉及理由项与被规定项两个关系位置。

### 定义 2：有限性边界

只有有穷对象、事件或概念规则，才可能被充足理由完整遍历并成为单子。

因此，全部对象及其在给定连接式下的排列组合虽然可以极大，但在宇宙的决定性闭合结构中仍必然总量有限。

```text
无限域不是合法单子
```

### 定义 3：理由展开式

对任意单子 `alpha`，取一个有限理由展开式：

```text
R_alpha := (r_1 ⊗ r_2 ⊗ ... ⊗ r_n ; C_alpha)
```

其中：

- `r_1, ..., r_n` 是理由因子；
- `C_alpha` 是连接式，记录复合、传递、并列和层级关系。

### 定义 4：贡献判据

对理由因子 `ri`，定义它对 `alpha` 的贡献判据：

```text
Con_alpha(ri) = 1, 当且仅当 删除 ri 后，R_alpha 不再足以推出 alpha
Con_alpha(ri) = 0, 其余情形
```

### 定义 5：规约算子与规范形

定义规约算子：

```text
Red_alpha(R_alpha)
= 删除所有满足 Con_alpha(ri)=0 的因子，
  并把保持 C_alpha 不变的纯排列同值式压到同一规范次序
```

定义单子 `alpha` 的有效理由结构：

```text
Eff(alpha) := NF_alpha(R_alpha) := Red_alpha(R_alpha)
```

于是：

- `Eff(alpha)` 是有限结构规范形；
- 完全相同的 `Eff(alpha)` 推出同一单子；
- 无贡献因子不改变同一性。

### 定义 6：权重

`w(alpha)` 是单子 `alpha` 的存在充实度，也是其本体真值度量。

对任一完整理由截面，固定一个有限总权步数 `M`。若单子 `alpha` 在该截面上实际占有 `k_alpha` 个权步，则把它的权重记作：

```text
w(alpha) := k_alpha : M
Live(w(alpha)) = 1  <=> alpha 为实际单子
Zero(w(alpha)) = 1  <=> alpha 为纯可能单子
```

这里：

- `k_alpha : M` 是有限归一比例记号，不是区间中的实数值；
- `k_alpha` 是从有限总权步词 `W_M` 中划给 `alpha` 的有限占步数，因此不会超过总权步数 `M`；
- `Zero(w(alpha)) = 1` 表示 `k_alpha` 为空占步；
- `Full(w(alpha)) = 1` 表示 `k_alpha` 占满总权步词。

### 定义 7：结构化复合

```text
alpha ⊗ beta
```

表示 `alpha` 与 `beta` 不是简单并列，而是按某连接式共同构成新单子。

### 定义 8：双层复合

结构化复合至少有两层：

1. 物理现实因果层
2. 概念因果层

故复合单子的总权重写为：

```text
w(alpha) = Couple(w_phys(alpha), w_concept(alpha))
```

### 定义 9：原因

若 `alpha != beta` 且 `alpha` 的权重实际参与支撑了 `beta` 的充足理由结构的一部分，则：

```text
alpha -> beta
```

若存在有限因果链连接二者，则：

```text
alpha ⇝ beta
```

### 定义 10：完整理由截面

给定目标单子 `alpha` 与观察角度 `κ`，定义：

```text
Sec(alpha,κ)
```

为从 `κ` 出发、足以构成 `alpha` 充分理由的有限全部对象之整体视图。

合法完整截面必须同时满足：

1. 目标角度确定；
2. 全部正贡献因子已纳入；
3. 全部正贡献路径已纳入。

### 定义 11：路径表

对结果单子 `E` 与背景 `γ`，写其允许回溯路径词序为：

```text
Path(E,γ) := <p_1, p_2, ..., p_m>
```

每条路径 `pj` 都是一个有限关系词：

```text
pj := x_1 rel_1 x_2 rel_2 ... rel_{l-1} x_l
```

其中每个 `rel_t` 只能取 `->`、`=>` 或由连接式诱导出的复合关系。

### 定义 12：路径权重

对路径 `pj`，定义：

```text
mu(pj)     := 允许的排列-复合数
lambda(pj) := 路径上传递的乘法权
ω(pj)      := Couple(mu(pj), lambda(pj))
```

### 定义 13：路径指示词

对理由头部 `R`，它关于 `Path(E,γ)` 的路径指示词写成：

```text
<δ_R(p_1), δ_R(p_2), ..., δ_R(p_m)>

δ_R(pj) = 1, 当且仅当 pj 被 R 吸纳
δ_R(pj) = 0, 其余情形
```

### 定义 14：理由级权重

```text
Ω_{E,γ}(R)
= Pool(<Take(δ_R(p_1),ω(p_1)), ..., Take(δ_R(p_m),ω(p_m))>)
```

这一步把路径级信息压成理由级支持量。

## 4. 截面判据与全局判据

### 定义 15：截面通约判据

对处在同一切面 `τ` 的 `a, b`，定义局部判据：

```text
Com_τ(a,b | E,γ) = 1
```

当且仅当：

```text
P(a | E,γ,τ) = P(b | E,γ,τ)
且 Live(P(a | E,γ,τ)) = 1
```

否则：

```text
Com_τ(a,b | E,γ) = 0
```

这只是切面上的局部判读，不是最终决定。

### 定义 16：切面层级轴的有限语法

对任一观察切面 `τ`，不给它预设无限时间轴，而只给它一个有限层级字：

```text
Ax(τ; u,v,w)
:=
<B[u], B[u-1], ..., B[1], N[1], N[2], ..., N[v], A[1], A[2], ..., A[w]>
```

其中：

- `u` 是当前切面下可回溯的过去层数；
- `v` 是当前切面内被显式分解出来的共时层数；
- `w` 是当前切面下可前瞻或被标记的未来层数。

这三个数都是有限计数值，因此 `Ax(τ;u,v,w)` 是一个有限字，而不是无限轴。

各层符号的读法是：

```text
B[i] : 第 i 个过去层符
N[j] : 第 j 个共时层符
A[k] : 第 k 个未来层符
```

并规定其层级序：

```text
B[u] ≺ ... ≺ B[1] ≺ N[1] ≺ ... ≺ N[v] ≺ A[1] ≺ ... ≺ A[w]
```

这里：

- `B[*]` 表示当前切面可见的过去层；
- `N[*]` 表示当前切面内部已经显式展开的共时层；
- `A[*]` 表示相对于当前切面的未来层。

### 定义 17：节点层符与路径端层

对当前局部因果网络中的每个节点 `x`，定义其层符赋值：

```text
lev_τ(x) := Ax(τ;u,v,w) 中的某一层符
```

也就是说，每个节点在当前切面上只能被赋予一个有限层符。

对路径词

```text
p := x_1 rel_1 x_2 rel_2 ... rel_{l-1} x_l
```

定义：

```text
s_τ(p) := lev_τ(x_1)
t_τ(p) := lev_τ(x_l)
```

于是 `s_τ(p)` 与 `t_τ(p)` 不再是直觉性的“起点层级/终点层级”，而是从有限层级字中抽出来的两个层符。

### 定义 18：路径骨架

对路径词 `p`，定义其其余两个基础描述子：

```text
len(p) : 关系词长度
C(p)   : 连接式骨架
```

其中：

- `len(p)` 只数关系位数；
- `C(p)` 去掉具体节点名，只保留 `->`、`=>`、`⊗` 的排列型。

因此，一个路径词的结构型由四个对象共同给出：

```text
(len(p), C(p), s_τ(p), t_τ(p))
```

### 定义 19：层位归类词

为了在后文中直接调用，给出三类层位判词：

```text
Past_τ(L)   = 1, 当且仅当 L = B[i] 对某个 i 成立
Past_τ(L)   = 0, 其余情形

Now_τ(L)    = 1, 当且仅当 L = N[j] 对某个 j 成立
Now_τ(L)    = 0, 其余情形

Future_τ(L) = 1, 当且仅当 L = A[k] 对某个 k 成立
Future_τ(L) = 0, 其余情形
```

这样：

- “某因子位于未来层”就可写成 `Future_τ(lev_τ(x)) = 1`
- “某因子位于当前共时层”就可写成 `Now_τ(lev_τ(x)) = 1`

### 定义 20：词长一致判据

这属于路径骨架判词。

```text
Len(p_{a,i},p_{b,i}) = 1, 当且仅当 len(p_{a,i}) = len(p_{b,i})
Len(p_{a,i},p_{b,i}) = 0, 其余情形
```

### 定义 21：连接式同型判据

这属于路径骨架判词。

```text
Typ(p_{a,i},p_{b,i}) = 1, 当且仅当 C(p_{a,i}) = C(p_{b,i})
Typ(p_{a,i},p_{b,i}) = 0, 其余情形
```

### 定义 22：层级同位判据

这属于路径骨架判词。

```text
Lay_τ(p_{a,i},p_{b,i}) = 1, 当且仅当 s_τ(p_{a,i})=s_τ(p_{b,i}) 且 t_τ(p_{a,i})=t_τ(p_{b,i})
Lay_τ(p_{a,i},p_{b,i}) = 0, 其余情形
```

它的意思是：两条路径不仅长度和连接式相配，而且在当前切面语法下，它们的起点层符与终点层符也完全一致。

### 定义 23：逐位并行因子

这属于并行判词。

设

```text
Path_E(a) := <p_{a,1}, p_{a,2}, ..., p_{a,n}>
Path_E(b) := <p_{b,1}, p_{b,2}, ..., p_{b,m}>
```

则对每个对应位置 `i` 定义：

```text
Par_i^E(a,b;τ) = All(<Len(p_{a,i},p_{b,i}), Typ(p_{a,i},p_{b,i}), Lay_τ(p_{a,i},p_{b,i})>)
```

它只取 `0` 或 `1`。

### 定义 24：目录同势判据

这属于并行判词。

```text
Card_E(a,b) = 1, 当且仅当 n = m
Card_E(a,b) = 0, 其余情形
```

### 定义 25：全局并行判据

这属于并行判词。

```text
Par_E(a,b;τ) = All(<Card_E(a,b), Par_1^E(a,b;τ), ..., Par_n^E(a,b;τ)>)
```

因此：

```text
Par_E(a,b;τ) = 1
<=> n = m 且对每个 i 都有：
    len(p_{a,i}) = len(p_{b,i}),
    C(p_{a,i}) = C(p_{b,i}),
    s_τ(p_{a,i}) = s_τ(p_{b,i}),
    t_τ(p_{a,i}) = t_τ(p_{b,i})
```

### 定义 26：逐位权重相等因子

这属于权比判词。

```text
Eq_i^E(a,b) = 1, 当且仅当 ω(p_{a,i}) = ω(p_{b,i})
Eq_i^E(a,b) = 0, 其余情形
```

### 定义 27：对应权重相等判据

这属于权比判词。

```text
EqW_E(a,b;τ) = All(<Par_E(a,b;τ), Eq_1^E(a,b), ..., Eq_n^E(a,b)>)
```

### 定义 28：全局通约判据

这属于全局判词。

```text
Com_E(a,b;τ) = All(<Par_E(a,b;τ), EqW_E(a,b;τ)>)
```

于是：

```text
Com_E(a,b;τ) = 1  <=>  a,b 对 E 在当前语法下全局可通约
Com_E(a,b;τ) = 0  <=>  a,b 对 E 在当前语法下全局不可通约
```

这里“全局”仍然指的是结果 `E` 的局部因果网络整体，而不是脱离当前有限语法的绝对无限视角。

### 说明：局部与全局

```text
Com_τ 只反映截面单子上的条件权重读数
Com_E   才反映当前有限层级语法下的决定性判断
```

因此：

- 截面上可以谈概率，因为截面单子上存在权重分布；
- 全局上不谈概率，因为全局决定性结构本身不是概率对象；
- 局部上条件概率相等，不推出全局上必然可通约。

## 5. 公理系统

### 公理 A0：相关闭组权重守恒

若 `G_κ := <g1, ..., gn>` 是同一组单子及其视角单元下相关且闭合的有限权词，则：

```text
Full(Pool(<w(g1), w(g2), ..., w(gn)>)) = 1
```

这里：

- “同一组单子及其视角单元”给出守恒判定的共同单位；
- “相关”表示这些权词彼此属于同一规定链或同一闭合作用域；
- “闭合”只要求理由结构闭合，不要求时间次序闭合；
- 宇宙整体闭合只是该公理的一个特例，而不是唯一适用对象。

### 公理 A1：单子的关系性定义根据

```text
Live(w(alpha)) = 1 => 存在 R 使得 R => alpha
```

并且单子性不能先于该关系被给定。

### 公理 A2：普通单子的外部理由性

若 `alpha != nu`，则：

```text
R => alpha => R 完全由不同于 alpha 的单子构成
```

### 公理 A3：始基单子的闭合规定性

```text
nu => nu
Full(w(nu)) = 1
```

其意义不是孤立自因，而是宇宙整体闭合网络对起点位的自一致规定。

### 公理 A4：因果非自反

```text
alpha -> beta => alpha != beta
alpha -> alpha 永假
```

### 公理 A5：因果非时序

```text
先发生 != 必然是原因
后发生 != 必然不能写入充分理由头部
```

### 公理 A6：多元汇聚的充分规定

任何结果单子 `beta` 的充分理由都形如：

```text
R = alpha1 ⊗ alpha2 ⊗ ... ⊗ alpha_k
R => beta
```

其中各 `alpha_i -> beta` 只是部分支撑。

### 公理 A7：复合封闭性

若有限复合体仍受充足理由律约束，则它构成新单子。

### 公理 A8：真值二值化

```text
Live(w(alpha)) = 1  <=> alpha 真
Zero(w(alpha)) = 1  <=> alpha 假
```

### 公理 A9：合法共时理由头部的内部一致性

若某共时理由头部 `R_τ` 对结果 `E` 合法，则其内部所有正贡献因子两两满足：

```text
Com_E(ri,rj;τ) = 1
```

否则 `R_τ` 不是合法充分理由头部。

## 6. 推理规则

### 规则 R1：规范形规约

```text
R_alpha -> NF_alpha(R_alpha) = Eff(alpha)
```

### 规则 R2：充足理由引入

```text
Eff(alpha) => alpha
```

### 规则 R3：部分原因投影

若 `ri` 在 `R => beta` 中正贡献，则：

```text
ri -> beta
```

### 规则 R4：结构化复合引入

若 `alpha, beta` 在连接式 `C` 下允许复合，则形成：

```text
(alpha ⊗ beta ; C)
```

### 规则 R5：截面展开

沿所有正贡献理由链与正贡献路径递归展开，直到：

1. 不再遗漏正贡献因子；
2. 不再遗漏允许路径；
3. 不再遗漏理由头部规约。

此时：

```text
Sec(E,κ) 完整
```

### 规则 R6：路径过滤

当背景 `γ` 增强时，删除一切与 `γ` 不相容的路径，保留其余路径。

### 规则 R7：理由级权重聚合

```text
Ω_{E,γ}(R)
= Pool(<Take(δ_R(p_1),ω(p_1)), ..., Take(δ_R(p_m),ω(p_m))>)
```

### 规则 R8：由果溯因归一

若候选理由头部为 `R_1, ..., R_n`，则：

```text
P(R_i|E,γ)
= Norm(Ω_{E,γ}(R_i) ; <Ω_{E,γ}(R_1), ..., Ω_{E,γ}(R_n)>)
```

### 规则 R9：闭合退化

若已拿到整体闭合理由截面，且只剩唯一理由头部 `R_*`，则：

```text
Full(P(R_*|E,γ)) = 1
```

否则不得在当前切面强行退化。

## 7. 直接命题

### 命题 1：独立单子不存在

由 A1 与单子定义直接得：

```text
不存在独立单子
```

### 命题 2：单子同一性

```text
Eff(alpha) = Eff(beta) => alpha = beta
```

### 定义 2.1：完整展开出现

对普通单子 `alpha != nu` 与其某个合法充分理由头部 `R => alpha`，记：

```text
Occ(x ; R => alpha) = 1
```

当且仅当在对 `R => alpha` 施行规则 `R5` 的完整递归展开后，单子 `x` 出现在该展开内部。

### 定义 2.2：结构先行关系

对普通单子 `x, alpha`，记：

```text
x ≺ alpha
```

当且仅当存在某个合法充分理由头部 `R => alpha` 使：

```text
Occ(x ; R => alpha) = 1
```

### 引理 2.3：外部理由性的完整展开形式

若 `alpha != nu` 且 `R => alpha` 合法，则：

```text
Occ(x ; R => alpha) = 1 => x != alpha
```

### 证明

由公理 `A2`，`R => alpha` 的理由头部完全由不同于 `alpha` 的单子构成。由规则 `R5`，对 `R => alpha` 的合法使用必须沿全部正贡献理由链递归展开，直到不再遗漏正贡献因子、允许路径与理由头部规约。故只要某个单子 `x` 在该完整展开中出现，它仍属于 `alpha` 的充分理由内部，因而必须满足 `x != alpha`。

### 引理 2.4：结构先行的传递性

```text
x ≺ y 且 y ≺ z => x ≺ z
```

### 证明

`x ≺ y` 表示 `x` 出现在某个合法 `R_y => y` 的完整展开中；`y ≺ z` 表示 `y` 出现在某个合法 `R_z => z` 的完整展开中。由规则 `R5`，对 `z` 的完整展开必须沿全部正贡献理由链继续递归展开，不得在 `y` 处停止。故 `x` 也出现在 `z` 的完整展开中，即 `x ≺ z`。

### 定理 2.5：普通单子的结构自返不可能

若 `alpha != nu`，则：

```text
非(alpha ≺ alpha)
```

### 证明

反设 `alpha ≺ alpha`。则存在某个合法 `R => alpha` 使 `Occ(alpha ; R => alpha) = 1`。这与引理 2.3 的结论

```text
Occ(x ; R => alpha) = 1 => x != alpha
```

在 `x = alpha` 时直接矛盾。故 `alpha ≺ alpha` 不成立。

### 定理 2.6：普通单子的充足理由不可逆返

对任意普通单子 `x, y`，

```text
x ≺ y => 非(y ≺ x)
```

### 证明

反设 `x ≺ y` 且 `y ≺ x`。由引理 2.4 得 `x ≺ x`，与定理 2.5 矛盾。故 `y ≺ x` 不成立。

### 推论 2.7：普通单子上的任意有限结构回路不可能

不存在普通单子词

```text
x_1 ≺ x_2 ≺ ... ≺ x_n ≺ x_1
```

其中 `n >= 2`。

### 证明

由引理 2.4 反复传递，得 `x_1 ≺ x_1`，与定理 2.5 矛盾。

### 推论 2.8：普通单子上的因果链无环

不存在普通单子词

```text
x_1 -> x_2 -> ... -> x_n -> x_1
```

其中 `n >= 2`。

### 证明

由规则 `R3`，每条 `x_i -> x_{i+1}` 都来自某个合法充分理由头部中的正贡献投影，因而推出 `x_i ≺ x_{i+1}`。于是得到一个有限结构回路，与推论 2.7 矛盾。

### 定义 2.9：有限展开子词

对普通单子 `alpha != nu` 的某个完整理由截面 `Sec(alpha,κ)`，取其中任意非空有限单子词 `S`。称 `S` 为 `alpha` 的一个有限展开子词。

### 定理 2.10：有限展开子词的极小元存在

设 `S` 是普通单子 `alpha != nu` 的一个有限展开子词，则存在 `m ∈ S` 使：

```text
对任意 x ∈ S，都有 非(x ≺ m)
```

### 证明

反设对任意 `m ∈ S`，都存在某个 `x ∈ S` 使 `x ≺ m`。任取 `x_1 ∈ S`。由反设可取 `x_2 ∈ S` 使 `x_2 ≺ x_1`；再取 `x_3 ∈ S` 使 `x_3 ≺ x_2`；如此继续，得到一列

```text
x_n ≺ x_{n-1} ≺ ... ≺ x_2 ≺ x_1
```

但 `S` 有限，故其中必有两个位置 `i < j` 满足 `x_i = x_j`。于是得到一个有限结构回路：

```text
x_i ≺ x_{i-1} ≺ ... ≺ x_j = x_i
```

这与推论 2.7 矛盾。故反设不成立，极小元存在。

### 定义 2.11：前沿

对普通单子 `alpha != nu` 的完整理由截面 `Sec(alpha,κ)`，定义其前沿为：

```text
Front(alpha,κ)
:=
< m ∈ Sec(alpha,κ) : 对任意 x ∈ Sec(alpha,κ)，都有 非(x ≺ m) >
```

### 推论 2.12：前沿非空

若 `Sec(alpha,κ)` 完整且 `alpha != nu`，则：

```text
Front(alpha,κ) 非空
```

### 证明

取 `S := Sec(alpha,κ)`。由定义 10，完整理由截面是足以构成 `alpha` 充分理由的有限全部对象之整体视图，故 `S` 为非空有限单子词。由定理 2.10，`S` 含有极小元。全部极小元构成 `Front(alpha,κ)`，故其非空。

### 定义 2.13：理由深度

对普通单子 `alpha != nu` 的完整理由截面 `Sec(alpha,κ)` 中任意单子 `y`，定义：

```text
Depth(alpha ; y,κ)
:=
在 Sec(alpha,κ) 中以 y 终止的全部结构先行链

z0 ≺ z1 ≺ ... ≺ zn = y

的长度 n 的最大值
```

### 定理 2.14：理由深度存在

若 `Sec(alpha,κ)` 完整、`alpha != nu` 且 `y ∈ Sec(alpha,κ)`，则 `Depth(alpha ; y,κ)` 存在且为有限整数。

### 证明

由定义 10 与第 2 节的有限对象约定，`Sec(alpha,κ)` 是有限单子词。由推论 2.7，其中不存在有限结构回路。故以 `y` 终止的结构先行链只能有有限多个，且每条链长度都有限，故其长度集合存在最大值。于是 `Depth(alpha ; y,κ)` 存在且为有限整数。

### 推论 2.15：结构先行严格增加理由深度

若 `x ≺ y`，则：

```text
Depth(alpha ; x,κ) < Depth(alpha ; y,κ)
```

### 证明

取一条以 `x` 终止并达到最大长度的结构先行链：

```text
z0 ≺ z1 ≺ ... ≺ zn = x
```

由 `x ≺ y`，可接成：

```text
z0 ≺ z1 ≺ ... ≺ zn = x ≺ y
```

故存在一条以 `y` 终止的结构先行链，其长度为 `n+1`。于是

```text
Depth(alpha ; y,κ) >= Depth(alpha ; x,κ) + 1
```

从而

```text
Depth(alpha ; x,κ) < Depth(alpha ; y,κ)
```

### 命题 3：截面角度下守恒

若 `Sec(alpha,κ)` 所展开出的权词恰构成某个相关闭组 `G_κ`，则：

```text
Full(Pool(<w(g1), ..., w(gn)>)) = 1
```

因此，截面角度下的守恒并不是来自“截面”这个词本身，而是来自该截面所读出的权词已经构成相关闭组。

### 命题 4：合法充足理由内部必须全局可通约

由 A9 直接得：

```text
R_τ 合法 => 对任意正贡献 ri,rj 都有 Com_E(ri,rj;τ)=1
```

### 命题 5：局部相等不推出全局相等

```text
Com_τ(a,b | E,γ) = 1
不推出
Com_E(a,b;τ) = 1
```

因为 `Com_τ` 只看当前切面条件权重，而 `Com_E` 还要求路径并行与逐位权重相等。

## 9. 不可完全闭合的证明链

### 定义 C1：共时相关项

若两个同切面单子 `a,b` 对同一结果 `E` 都表现出相关支撑，且这种相关性要求内部耦合而不是纯并列，则称 `a,b` 为 `E` 的共时相关项。

### 引理 1：不存在“同时相关的充足理由”伪闭合

若 `a,b` 是 `E` 的共时相关项，且：

```text
Com_E(a,b;τ_now) = 0
```

则：

```text
(a ⊗ b ; C_τ) 不能构成 E 的合法充分理由头部
```

因为：

1. 路径不并行，或
2. 对应权重不相等，或
3. 两者兼而有之。

任一情形都会破坏合法理由头部的全局决定性。

### 引理 2：未来层正贡献因子在当前切面不可消去

若结果 `E` 的某个正贡献因子 `f` 位于当前切面 `τ_now` 的未来层，则：

```text
Future_{τ_now}(lev_{τ_now}(f)) = 1
```

从而：

```text
f 不能写入 `τ_now` 下可完全展开的截面序列
```

故当前切面上不得把它当作已闭合因子。

### 总定理：当前切面的不可完全闭合

设结果 `E` 位于过去层或当下层的观察切面 `τ_now`。若在 `τ_now` 上对 `E` 回溯，则：

1. 若某正贡献因子 `f` 满足 `Future_{τ_now}(lev_{τ_now}(f)) = 1`，则当前切面不能把它完全纳入；
2. 若当前切面中看似存在共时未分解理由，但其相关项满足 `Com_E(*,*;τ_now) = 0`，则它们不是合法充足理由；
3. 因而对 `E` 的当前切面回溯不能完全闭合。

于是：

```text
当前切面的不可完全闭合
= 未来层正贡献理由的切面外在性
  与非法共时伪闭合方案的排除共同给出
```

这说明：

```text
本体上仍有充足理由
但当前切面上不存在把它完全闭合出来的合法步骤
```

## 10. 概率的严格位置

### 定义 P1：由果溯因概率

概率不是全局决定性结构的性质，而是某个截面单子在当前切面下的权重分布读数。

因此，对结果截面单子 `E` 与背景 `γ`，若候选理由头部为 `R_1, ..., R_n`，则：

```text
P(R_i|E,γ)
= Norm(Ω_{E,γ}(R_i) ; <Ω_{E,γ}(R_1), ..., Ω_{E,γ}(R_n)>)
```

这里：

- `P(R_i|E,γ)` 只在截面权词已经被读出时有意义；
- 它描述的是截面单子上的有限归一分布；
- 对全局决定性结构本身，不引入 `P`。

### 定理 4：完整回溯时概率退化

若 `Sec(E,κ)` 已完整且只剩唯一理由头部 `R_*`，则：

```text
Full(P(R_*|E,γ)) = 1
```

### 定理 5：当前切面下概率不可一般退化

若：

1. 存在正贡献因子 `f` 使得 `Future_{τ_now}(lev_{τ_now}(f)) = 1`，或
2. 存在共时相关伪闭合方案且其 `Com_E(*,*;τ_now) = 0`

则：

```text
P(R|E,γ,τ_now) 不得被强行退化为 1
```

也就是说：

- 概率只属于 `τ_now` 这样的具体切面；
- 不属于宇宙整体闭合结构本身。

## 11. 不可全知性

有限认知者不可全知，原因有五：

1. 当前切面只能局部展开全局理由网络；
2. 未闭合回溯会保留多条允许路径；
3. 路径数随层级增长而组合爆炸；
4. 权重流动中的收束会造成区分信息损耗；
5. 未来层理由与非法共时伪闭合共同阻止当前切面闭合。

故：

```text
本体可由充足理由约束
!=
认知者可穷尽该理由全体
```

## 12. 宇宙的闭合结构

### 定义 U1：始基单子

```text
nu => nu
Full(w(nu)) = 1
```

它表示宇宙整体闭合网络对起点位的自一致规定。

### 定义 U2：终结单子

`Theta` 是全部不对称权词达到均衡后的终态，其理由是整个演化史本身。

### 宇宙闭合式

设宇宙全过程任一完整理由截面写为：

```text
Sec_U := <u1, u2, ..., un>
```

则：

```text
Full(w(nu)) = 1
Full(w(Theta)) = 1
Full(Pool(<w(u1), w(u2), ..., w(un)>)) = 1
```

这里的宇宙整体只是在取 `G_U := <u1, ..., un>` 为最大相关闭组时得到的特例。

宇宙是全局节点单子的决定性闭合结构，而宇宙总权守恒只是相关闭组守恒在最大闭组上的实现。

## 13. 完整案例

### 案例 1：湿地面的由果溯因

设：

```text
g := 地面暴露
r := 降雨发生
s := 喷洒启动
E := 地面湿
```

定义：

```text
R_1 := (r ⊗ g ; C_1)
R_2 := (s ⊗ g ; C_2)
R_1 => E
R_2 => E
```

由 R3：

```text
r -> E
g -> E
s -> E
```

取当前背景 `γ_0`，令：

```text
Path(E,γ_0) := <p_1,p_2>
δ_{R_1}(p_1)=1, δ_{R_1}(p_2)=0
δ_{R_2}(p_1)=0, δ_{R_2}(p_2)=1
ω(p_1)=2, ω(p_2)=1
```

则：

```text
Ω_{E,γ_0}(R_1)=2
Ω_{E,γ_0}(R_2)=1
P(R_1|E,γ_0)=2:3
P(R_2|E,γ_0)=1:3
```

若新背景删去 `R_2` 路径，则：

```text
Full(P(R_1|E,γ_1))=1
```

```text
k := 钥匙匹配

m := 监控开启
```

```text
R_O := (k ⊗ t ⊗ a ⊗ m ; C_O)
```

```text
Con_O(m)=0
```

```text
Eff(O)=NF_O(R_O)=(k ⊗ t ⊗ a ; C'_O)
Eff(O) => O
```

### 案例 4：未来层理由与伪共时闭合

```text
f  := 未来确认信号完成
c1 := 共时项 1
c2 := 共时项 2
n  := 当前显示亮起
```

给定：

```text
R_n := (f ⊗ c1 ⊗ c2 ; C_n)
R_n => n
```

若：

```text
Future_{τ_now}(lev_{τ_now}(f)) = 1
Com_E(c1,c2;τ_now)=0
```

则：

1. `f` 在当前切面不可完全纳入；
2. `(c1 ⊗ c2 ; C_sync)` 不是合法共时理由头部；
3. 所以当前切面不能把 `n` 的回溯完全闭合。

## 14. 系统总公式

```text
单子 = 在充足理由关系中被规定出来的有限位点
同一性 = Eff 的规范形唯一性
局部通约 = Com_τ
全局通约 = Com_E(*,*;τ) = All(<Par_E(*,*;τ), EqW_E(*,*;τ)>)
并行判据 = Par_E(*,*;τ) = All(<Card_E, Par_1^E, ..., Par_n^E>)
权重判据 = EqW_E(*,*;τ) = All(<Par_E(*,*;τ), Eq_1^E, ..., Eq_n^E>)
概率 = 对理由级权词作有限归一
不可完全闭合 = 由未来层正贡献理由与非法共时伪闭合排除共同给出
宇宙 = 全局节点单子决定性的闭合理由网络
```
