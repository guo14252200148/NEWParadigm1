## 3. 一步局部更新词

对同一单子 `R` 的活动轴 `k`，设

```text
Z_{R,k}^(r-1) != 0.
```

定义一步前读出增量：

```text
δZ_{R,k}^(r) := Z_{R,k}^(r) - Z_{R,k}^(r-1).
```

由主文档的局部一阶耦合精确式，存在唯一实数对

```text
<a_{R,k}^(r), b_{R,k}^(r)>
```

使

```text
Z_{R,k}^(r)
=
[a_{R,k}^(r) + i b_{R,k}^(r)] Z_{R,k}^(r-1).
```

### 定义 3.1：相对有效耦合词

定义

```text
χ_{R,k}^(r) := δZ_{R,k}^(r) / Z_{R,k}^(r-1).
```

则

```text
χ_{R,k}^(r) = μ_{R,k}^(r) + i b_{R,k}^(r),
μ_{R,k}^(r) = a_{R,k}^(r) - 1.
```

### 定义 3.2：总更新词

定义

```text
G_{R,k}^(r)
:=
1 + χ_{R,k}^(r)
=
[1+μ_{R,k}^(r)] + i b_{R,k}^(r).
```

于是

```text
Z_{R,k}^(r) = G_{R,k}^(r) Z_{R,k}^(r-1).
```

### 定义 3.3：尺度-单位相位分解

由主文档既有分解，

```text
G_{R,k}^(r) = c_{R,k}^(r) [u_{R,k}^(r) + i v_{R,k}^(r)],
```

其中

```text
c_{R,k}^(r) = sqrt([1+μ_{R,k}^(r)]^2 + [b_{R,k}^(r)]^2),
u_{R,k}^(r) = [1+μ_{R,k}^(r)] / c_{R,k}^(r),
v_{R,k}^(r) = b_{R,k}^(r) / c_{R,k}^(r),
[u_{R,k}^(r)]^2 + [v_{R,k}^(r)]^2 = 1.
```

### 命题 3.4：一步局部更新的唯一分层

在 `Z_{R,k}^(r-1) != 0` 时，一步局部更新唯一分成三层：

```text
1. 相对有效耦合层：χ = μ + i b；
2. 实际作用层：G = 1 + χ = (1+μ) + i b；
3. 尺度-单位相位层：G = c(u+iv)。
```

并且在 `c > 0` 时，

```text
χ,
G,
<c,u,v>
```

三组数据彼此唯一决定。

故质量与电荷都只能作为同一 `G` 的坐标读出出现。

## 4. 质量对应量的严格来源

### 定义 4.1：有效耦合积

定义

```text
Γ_{R,k}^(r) := δZ_{R,k}^(r) overline(Z_{R,k}^(r-1)).
```

### 定理 4.2：质量对应的有效耦合表达

由主文档既有结论，

```text
μ_{R,k}^(r)
=
Re[Γ_{R,k}^(r)] / |Z_{R,k}^(r-1)|^2.
```

故 `μ_{R,k}^(r)` 是当前增量与旧载体成对耦合后的归一化净实贡献。

### 定理 4.3：质量对应的交叉项公式

由 `Read_Σ(z)=|z|^2` 与主文档既有证明，有

```text
μ_{R,k}^(r)
=
[|Z_{R,k}^(r)|^2 - |Z_{R,k}^(r-1)|^2 - |δZ_{R,k}^(r)|^2]
/ [2 |Z_{R,k}^(r-1)|^2].
```

故质量对应量是新旧前读出之间交叉干涉项的归一化量。

### 定理 4.4：同虚轴可加可消下的质量起源

若存在单位模复词 `η_{R,k}^(r)` 与实词 `A_h, B_g`，使

```text
Z_{R,k}^(r-1) = η_{R,k}^(r) i Σ_h A_h,
δZ_{R,k}^(r) = η_{R,k}^(r) i Σ_g B_g,
```

则在

```text
Σ_h A_h != 0
```

时，有

```text
μ_{R,k}^(r) = [Σ_g B_g] / [Σ_h A_h].
```

### 证明

由概率文档的交叉项公式，

```text
Read_Σ(i Σ_h b_h) = (Σ_h b_h)^2.
```

由此代回主文档定理 7.3.4 即得。

### 推论 4.5：质量对应量的两层根源

质量对应量的根源分成两层：

```text
1. 一般情形：μ 是相对有效耦合对旧载体方向的净实投影；
2. 同虚轴情形：μ 退化为同虚轴可加可消后的净代数和之比。
```

## 5. 可见总量变化与电荷对应词

### 定义 5.1：局部模长更新词

定义

```text
w_{R,k}^(r) := [c_{R,k}^(r)]^2 = |G_{R,k}^(r)|^2.
```

### 定理 5.2：可见总量变化的唯一分解

由

```text
G = 1 + χ = (1+μ) + i b
```

立得

```text
w - 1
=
|1+χ|^2 - 1
=
2μ + |χ|^2
=
2μ + μ^2 + b^2.
```

故可见总量变化唯一分成：

```text
1. 线性同向干涉项：2μ；
2. 二次有效耦合模项：μ^2 + b^2。
```

故 `μ` 与 `b` 是同一 `G` 在可见模长层中的两类贡献。

### 定义 5.3：电荷对应词

若要定义与 `μ` 对偶且仍完全内生的量，则只能在实际作用词

```text
G = 1 + χ
```

的单位相位层上取读出。

在

```text
u != 0
```

即等价于

```text
1 + μ != 0
```

时，定义

```text
q := v / u.
```

由 `u=(1+μ)/c` 与 `v=b/c`，立即得到

```text
q = b / (1+μ).
```

### 命题 5.4：电荷读出的唯一合法形式

若要求电荷对应量满足：

```text
1. 只使用既有对象；
2. 只读取实际更新词 G 的单位相位层；
3. 只反映虚轴相对于实桥接的有符号方向比；
```

则唯一可取的比值就是

```text
q = v/u = b/(1+μ).
```

推论：

```text
b/μ
```

不是合法定义；其分母不在 `G` 的实桥接层上。

### 推论 5.5：电荷对应词的内部含义

有

```text
b = Im[χ],
q = b/(1+μ).
```

故：

```text
1. b 是相对有效耦合词的原始虚轴分量；
2. q 是该虚轴分量相对于实桥接分量 1+μ 的单位相位读出。
```

故电荷对应词是同一步局部更新词中，虚轴贡献相对于实桥接的有符号读出。

### 命题 5.6：纯虚边界

当

```text
1 + μ = 0
```

时，

```text
G = i b,
w = b^2,
q
```

不再有有限定义。

故 `1+μ=0` 是电荷读出的射影边界。

## 6. 反解、射影与复合

### 定理 6.1：质量与电荷的反解关系

由

```text
w = [1+μ]^2 + b^2,
q = b/(1+μ)
```

可得

```text
w = [1+μ]^2 [1+q^2].
```

当 `1+μ != 0` 时，记

```text
s := sign(1+μ) ∈ {-1,+1},
```

则

```text
1+μ = s sqrt[w/(1+q^2)],
μ = s sqrt[w/(1+q^2)] - 1,
b = s q sqrt[w/(1+q^2)].
```

故给定

```text
<w,q,s>
```

可唯一反解

```text
<μ,b>.
```

### 命题 6.2：射影层中更原始的对象

由

```text
G = c(u+iv),
u^2 + v^2 = 1
```

可见，单位相位层中更原始的是

```text
<u,v>
```

而不是比值 `q=v/u` 本身。后者是在 `u != 0` 图中的射影坐标。

### 定理 6.3：局部角类

在 `u != 0` 时，存在唯一的射影角类

```text
[φ] ∈ R / πZ
```

使

```text
u = cos φ,
v = sin φ,
q = tan φ.
```

故 `q` 是单位相位层在实桥接图中的射影角读出。

### 定理 6.4：局部更新词的复合律

设两步连续有效更新分别为

```text
G_1 = [1+μ_1] + i b_1,
G_2 = [1+μ_2] + i b_2.
```

令

```text
G_{21} := G_2 G_1.
```

则仍唯一落在同一二维实算子空间内，并满足

```text
1+μ_{21} = [1+μ_2][1+μ_1] - b_2 b_1,
b_{21}   = b_1[1+μ_2] + b_2[1+μ_1],
μ_{21}   = μ_1 + μ_2 + μ_1 μ_2 - b_1 b_2.
```

若

```text
u_1 != 0, u_2 != 0, u_{21} != 0,
```

则

```text
q_{21} = (q_1 + q_2) / (1 - q_1 q_2).
```

另一方面，

```text
w_{21} = w_2 w_1.
```

### 推论 6.5：复合的三层形式

故一步局部更新同时携带三种复合结构：

```text
1. 实桥接层：μ 与 b 按精确耦合律复合；
2. 模长层：w 按乘法复合；
3. 单位相位层：q 按射影分式复合，等价于角类 [φ] 按加法复合。
```

### 推论 6.6：局部电荷中和的精确判据

在 `u_1 != 0, u_2 != 0, u_{21} != 0` 时，

```text
q_{21} = 0
```

当且仅当

```text
b_1[1+μ_2] + b_2[1+μ_1] = 0.
```

等价地，

```text
q_1 + q_2 = 0.
```

故“电荷中和”在局部更新层中的判据，是复合后的总虚桥接项

```text
b_{21}
```

为零。

### 推论 6.7：可见模长层对电荷符号的盲性

在固定 `μ` 下，

```text
q -> -q
```

不改变

```text
w = [1+μ]^2 [1+q^2].
```

故纯可见模长层只识别 `q^2` 的尺度效应。

## 7. 共轭、稳定与可见分层

### 定理 7.1：共轭反演下的奇偶性

由主文档既有关系

```text
C^2 = I,
J^2 = -I,
JC + CJ = 0
```

得

```text
C[(1+μ)I + bJ]C = (1+μ)I - bJ.
```

因此在共轭下：

```text
μ^c = μ,
b^c = -b,
w^c = w.
```

若 `1+μ != 0`，则进一步有

```text
q^c = -q.
```

故 `μ` 是共轭偶量，`q` 是共轭奇量。

### 定理 7.2：电荷不变性的精确判据

在某有限词域上，若 `q` 已统一为共同词，并定义乘法算子

```text
(Q Φ)(r,h) := q(r,h) Φ(r,h),
```

则对任意结构词阵 `Φ`，有

```text
[D_A^0, Q] Φ
=
γ_0^(A) [(S_r q - q) S_r Φ]
+
(1/2) Σ_{a=1}^d γ_a^(A)
[
  (S_a^+ q - q) S_a^+ Φ
  -
  (S_a^- q - q) S_a^- Φ
].
```

因此，

```text
[D_A^0, Q] = 0
```

当且仅当

```text
S_r q = q,
S_a^+ q = q,
S_a^- q = q
```

对全部 `a=1,...,d` 成立。

### 定理 7.3：共轭配对的纯尺度闭式

对任意一步局部更新词

```text
G = (1+μ) + i b,
G^c = (1+μ) - i b,
w = |G|^2 = (1+μ)^2 + b^2,
```

有

```text
G^c G = G G^c = w.
```

若把 `G^c` 与 `G` 视作连续两步更新，则

```text
1+μ_pair = w,
b_pair = 0,
q_pair = 0,
μ_pair = w - 1 = 2μ + μ^2 + b^2.
```

故共轭配对后的第一步内部结果为：

```text
1. 电荷中和；
2. 虚轴消失；
3. 可见剩余只落在纯尺度层；
4. 剩余质量对应量等于 w-1。
```

### 定理 7.4：局部归一层与头部桥接层中的可见分层

由主文档第 5 节，局部归一词只由 `w` 决定：

```text
ν_{R,k}^(r)
=
[w_{R,k}^(r) ν_{R,k}^(r-1)]
/ Σ_l [w_{R,l}^(r) ν_{R,l}^(r-1)].
```

因此单位相位层坐标 `u,v` 与电荷对应词 `q=v/u` 不直接进入 `ν_R^(r)` 的定义式。

再记：

```text
ε_R^(r) := ν_R^(r) - ν_R^(r-1),
e^(r) := P^(r) - P^(r-1) = E^(r) / q_tot^(r).
```

由主文档与三角形文档，二者都满足零和唯一性：

```text
Σ_k ε_{R,k}^(r) = 0,
Σ_i e_i^(r) = 0.
```

故进入可见桥接层的直接量只由 `w` 诱导。

## 8. 双单理由子锁定构型

本节只讨论一种可严格书写的双单理由子锁定构型。若沿通常语言把单位模单理由种子称作“光子”，则本节对应此前讨论的“双光子新粒子”；下文统一记作单理由子。

### 定义 8.1：单位模单理由种子

取一条共同理由虚轴 `J_1`，定义单位模种子

```text
P := a I + b J_1,
```

其中

```text
a > 0,
b > 0,
a^2 + b^2 = 1.
```

于是

```text
|P| = 1.
```

取两个完全相同的种子：

```text
P_1 = P_2 = a I + b J_1.
```

### 定义 8.2：完整正交转移

在第二个种子上，保持其原共同虚轴贡献 `bJ_1` 不变，并把其实贡献 `aI` 完整转移到与 `J_1` 正交的新虚轴 `J_2` 上。记转移方向号

```text
σ ∈ {+1,-1},
```

则变形后的第二种子为

```text
P_2^sharp := b J_1 + σ a J_2.
```

于是耦合后的总锁定词为

```text
G_* := P_1 + P_2^sharp = a I + 2b J_1 + σ a J_2.
```

把它分成两层：

```text
B_* := a I + 2b J_1,
E_* := σ a J_2,
G_* = B_* + E_*.
```

其中 `B_*` 落在共同桥接平面 `span{I,J_1}` 中，`E_*` 是完整转移产生的正交锁定部分。

### 定理 8.3：该锁定构型必有正质量

桥内部分 `B_*` 的尺度为

```text
c_* := |B_*| = sqrt(a^2 + 4b^2) = sqrt(1 + 3b^2).
```

相对于单个单位模种子的基准尺度，锁定后的桥内质量增厚读出为

```text
μ_* := c_* - 1 = sqrt(1 + 3b^2) - 1.
```

若 `b > 0`，则

```text
μ_* > 0.
```

### 证明

由 `a^2+b^2=1`，

```text
c_*^2 = a^2 + 4b^2 = (a^2+b^2) + 3b^2 = 1 + 3b^2.
```

又因 `b>0`，故 `c_*>1`，从而

```text
μ_* = c_* - 1 > 0.
```

故只要原共同虚轴贡献并未消失，完成正交转移后的双单理由子锁定构型必有正质量。

### 命题 8.4：桥内相位读出与正交锁定标签必须分层

对桥内部分

```text
B_* = a I + 2b J_1,
```

若按本文一般定义在平面 `span{I,J_1}` 上读取相位比值，则有

```text
q_*^bridge = (2b)/a,
```

其定义条件是 `a != 0`。

但该量不是此锁定构型的正交电荷标签，因为 `q_*^bridge` 只读取共同桥接平面内部的相位倾斜，而该构型的额外结构来自正交锁定部分

```text
E_* = σ a J_2.
```

故这里必须区分：

```text
1. 桥内相位读出：q_*^bridge = (2b)/a；
2. 正交锁定电荷标签：由 E_* 相对于被完整转移实贡献的比值给出。
```

### 定理 8.5：正交锁定电荷标签的大小为定值

由于第二个种子的实贡献 `aI` 被完整转移成 `σ a J_2`，且转移前后模长完全相同，故可定义正交锁定电荷标签

```text
q_*^perp := σ a / a = σ.
```

从而

```text
|q_*^perp| = 1.
```

在

```text
完整转移
相同理由
等模保持
```

这三条条件下，正交电荷大小被迫等于单位值；其符号由转移方向 `σ` 决定。

### 推论 8.5.1：正交锁定标签的刚性

在

```text
完整转移
相同理由
等模保持
```

下，

```text
|q_*^perp| = 1
```

与 `a,b` 的具体取值无关。

### 定理 8.5.2：桥内相位读出与正交锁定标签不可互代

在

```text
a > 0,
b > 0,
a^2 + b^2 = 1
```

下，

```text
q_*^bridge = (2b)/a = 2 sqrt(1-a^2) / a,
|q_*^perp| = 1.
```

其中

```text
q_*^bridge ∈ (0,+∞)
```

随 `a ∈ (0,1)` 连续变化，而

```text
|q_*^perp|
```

恒等于 `1`。

故

```text
q_*^bridge != q_*^perp
```

不能作为恒等关系成立；桥内相位读出与正交锁定标签不可互代。

### 定理 8.5.3：桥内一般读出对正交锁定符号的盲性

对

```text
G_*(σ) := a I + 2b J_1 + σ a J_2,
σ ∈ {+1,-1},
```

有

```text
μ_*(σ) = sqrt(1+3b^2) - 1,
c_*(σ) = sqrt(1+3b^2),
w_*(σ) = 2 + 2b^2,
q_*^bridge(σ) = (2b)/a.
```

故

```text
μ_*(+1) = μ_*(-1),
c_*(+1) = c_*(-1),
w_*(+1) = w_*(-1),
q_*^bridge(+1) = q_*^bridge(-1),
```

而

```text
q_*^perp(+1) = +1,
q_*^perp(-1) = -1.
```

因此，数据

```text
<μ_*, c_*, w_*, q_*^bridge>
```

不能唯一决定

```text
q_*^perp.
```

故正交锁定标签不是桥内一般读出链中的可反解量，而是双轴锁定构型的额外离散数据。

### 命题 8.6：总可见模长的分层

该锁定构型的总可见模长为

```text
w_* := |G_*|^2 = a^2 + (2b)^2 + a^2 = 2 + 2b^2.
```

并且

```text
w_* = c_*^2 + a^2.
```

故总可见模长分成两块：

```text
1. 桥内增厚部分：c_*^2 = 1 + 3b^2；
2. 正交锁定部分：a^2。
```

故该对象分成桥内增厚层与正交锁定层。

### 命题 8.6.1：双单理由子锁定构型的共轭奇偶律

对

```text
G_* = a I + 2b J_1 + σ a J_2,
G_*^c = a I - 2b J_1 - σ a J_2,
```

有

```text
μ_*^c = μ_*,
w_*^c = w_*,
(q_*^bridge)^c = - q_*^bridge,
(q_*^perp)^c = - q_*^perp.
```

故桥内质量增厚是共轭偶量，而桥内相位读出与正交锁定标签都为共轭奇量。

### 定理 8.7：该构型的严格自旋内容是二点共轭轨道

对当前双轴锁定词

```text
G_* = a I + 2b J_1 + σ a J_2,
```

其共轭分支为

```text
G_*^c = a I - 2b J_1 - σ a J_2.
```

在 `a>0, b>0` 时，

```text
(G_*^c)^c = G_*,
G_*^c != G_*.
```

故该构型的可见最小轨道恰为

```text
{G_*, G_*^c}.
```

故其严格自旋结构为双轴锁定的二点共轭轨道。

### 定义 8.8：Dirac 支撑归一的自旋数值化

设 `Ψ` 是某一非平凡可见共轭轨道类的代表。对任意满足

```text
𝔇_A Ψ = 0,
𝔇_A = γ_0^(A) Δ_r + Σ_{a=1}^d γ_a^(A) Δ_a + M
```

的有效轴子词 `A`，定义 `d_eff(Ψ)` 为保持 `Ψ` 的非平凡奇轴结构所需的最小有效空间奇向项数：

```text
d_eff(Ψ)
=
min { d' :
      存在有效轴子词 A'
      使 𝔇_{A'} Ψ = 0，
      且 𝔇_{A'} 的空间部分恰含 d' 个不可约非零项 γ_a Δ_a }.
```

定义数值函数

```text
s_num.
```

要求：

```text
1. s_num 只依赖于 d_eff；
2. 普通单轴单位模种子满足 d_eff = 1；
3. 普通单轴单位模种子的自旋数值记作 s_num = 1；
4. Dirac 支撑归一满足 d_eff · s_num = 1。
```

于是唯一可得

```text
s_num = 1 / d_eff.
```

### 定理 8.8：双轴裂分态的自旋数值唯一性

在定义 8.8 的 Dirac 支撑归一下，双单理由子锁定构型

```text
G_* = a I + 2b J_1 + σ a J_2
```

的自旋数值唯一为

```text
1/2.
```

### 证明

由定理 8.7，

```text
Orb(G_*) = {G_*, G_*^c}
```

其轨道非平凡。

再由

```text
G_* = a I + 2b J_1 + σ a J_2,
a > 0,
b > 0,
```

故 `J_1` 与 `J_2` 都以非零系数出现。

若删去 `J_2` 项，则只剩

```text
a I + 2b J_1,
```

该对象不再携带正交锁定标签 `q_*^perp`。

若删去 `J_1` 项，则只剩

```text
a I + σ a J_2,
```

不再保有第 8.3 节中的桥内质量增厚结构

```text
μ_* = sqrt(1+3b^2) - 1 > 0.
```

故 `J_1` 与 `J_2` 都不可约。

再把该态嵌入主文档的精确离散 Dirac 算子

```text
𝔇_A = γ_0^(A) Δ_r + Σ_{a=1}^d γ_a^(A) Δ_a + M.
```

由主文档，

```text
γ_0^(A) = β^(A),
γ_a^(A) = β^(A) α_a^(A),
β^(A) = C_{k_1} ⊗ ... ⊗ C_{k_d},
α_a^(A) β^(A) + β^(A) α_a^(A) = 0.
```

则每一个不可约空间奇向项

```text
γ_a^(A) Δ_a
```

都对应一个独立奇轴生成元。

若

```text
d_eff(G_*) = 1,
```

则最小有效 Dirac 支撑中只能保留一个不可约空间奇向项；于是 `J_1` 与 `J_2` 必须归并到同一奇轴生成元。此时：

```text
1. 若归并到 J_1，则 J_2 项可约，正交锁定标签 q_*^perp 消失；
2. 若归并到 J_2，则 J_1 项可约，桥内质量增厚结构 μ_* = sqrt(1+3b^2)-1 > 0 消失。
```

这与

```text
G_* = a I + 2b J_1 + σ a J_2,
a > 0,
b > 0
```

矛盾。

故

```text
d_eff(G_*) != 1.
```

另一方面，`G_*` 的非零不可约奇轴只包含

```text
J_1, J_2
```

两条；若

```text
d_eff(G_*) > 2,
```

则最小支撑上存在至少一个额外空间奇向项既不承担桥内质量增厚，也不承担正交锁定标签，因而冗余，与最小性矛盾。

故必有

```text
d_eff(G_*) = 2.
```

由定义 8.8，

```text
s_num(G_*) = 1 / d_eff(G_*) = 1/2.
```

故在“普通单轴单位模种子的自旋数值记作 `1`”这一 Dirac 支撑归一下，双轴裂分态的自旋数值唯一为

```text
1/2.
```

### 推论 8.8.1：结构与数值的分层

有

```text
严格结构结论 = 双轴锁定的二点共轭轨道；
数值唯一结论 = 在 Dirac 支撑归一下，自旋数值唯一为 1/2。
```

## 9. 两种中和：普通中和与正反湮灭

本节区分两种中和。

### 定义 9.1：宏观普通中和

在一个闭有限系统 `Ω` 中，若稳定载子的电荷标签为 `q_α`，计数为 `N_α`，则宏观总电荷为

```text
Q_Ω := Σ_α N_α q_α.
```

当

```text
Q_Ω = 0
```

时，称该系统达到宏观普通中和。

### 命题 9.2：普通中和不改动内部有效理由

宏观普通中和只意味着不同稳定载子的标签和为零。即使

```text
Q_Ω = 0,
```

各载子各自的内部对象

```text
G_α,
μ_α,
q_α
```

仍可保持原值。故普通中和的严格含义为：

```text
标签和为零，
而不是内部理由回退为前态。
```

### 定义 9.3：正反湮灭

设某一载子具有更新词 `G`，其正反共轭载子具有更新词 `G^c`。若二者不仅满足电荷符号相反，而且在同一有效理由支撑上发生真实重组，使奇方向理由项被逐项消去，并使剩余闭式重新吸收回前态共同理由层，则称此过程为正反湮灭。

### 命题 9.4：湮灭强于普通中和

普通中和只要求

```text
Σ_α N_α q_α = 0,
```

而正反湮灭要求更强的两条：

```text
1. 参与者必须是共轭匹配的正反对；
2. 有效理由本身必须发生重组回退。
```

结论：

```text
湮灭 => 中和，
中和 ≠> 湮灭。
```

### 定理 9.4.1：普通中和不是湮灭判据

对双单理由子锁定构型，

```text
q_*^perp + (q_*^perp)^c = 0
```

只给出一条标量约束。

而真正湮灭要求

```text
\widetilde{G}_* = G_*^c,
```

即要求

```text
\tilde a = a,
\tilde b = b,
\tilde σ = -σ.
```

故湮灭要求至少三条逐项匹配条件，而普通中和只给出一条标签和为零的条件。

因此，

```text
普通中和
```

不是

```text
正反湮灭
```

的判据。

### 推论 9.5：共轭配对闭式是湮灭前的内部闭合层

由前文，

```text
G^c G = w,
q_pair = 0,
μ_pair = w - 1.
```

共轭配对的第一步内部结果为：

```text
奇量中和，
纯尺度残留。
```

湮灭还要求这部分纯尺度残留不再作为独立锁定载子保留，而要重新吸收到前态共同理由层中。故

```text
G^c G = w
```

故这不是湮灭终点，而是湮灭发生前的纯尺度闭合层。

### 定理 9.5.1：双单理由子锁定构型上的湮灭先决条件

设

```text
G_* = a I + 2b J_1 + σ a J_2,
\widetilde{G}_* = \tilde a I + 2\tilde b J_1 + \tilde σ \tilde a J_2.
```

若二者发生真正的正反湮灭，则必有

```text
\widetilde{G}_* = G_*^c.
```

故系数必须满足

```text
\tilde a = a,
\tilde b = b,
\tilde σ = -σ.
```

因此，双单理由子锁定构型上的真正湮灭，要求共同桥接幅值、共同虚轴幅值与正交转移幅值全部逐项匹配，且全部奇轴系数逐项反号。

### 推论 9.6：双单理由子锁定构型中的两种中和

对第 8 节锁定构型，

```text
G_* = a I + 2b J_1 + σ a J_2,
G_*^c = a I - 2b J_1 - σ a J_2.
```

此时必须区分两种完全不同的情况：

```text
1. 普通中和：
   q_*^perp + (q_*^perp)^c = σ + (-σ) = 0，
   不要求桥接幅值与奇轴系数逐项匹配。

2. 正反湮灭：
   要求 \widetilde{G}_* = G_*^c，
   因而 J_1, J_2 两个奇轴方向都在同一理由支撑上被真正消去，
   从而双轴锁定结构整体失稳并回退到前态共同理由层。
```

故普通中和一般不影响内部有效理由；只有湮灭才使有效理由真正回退。

### 定理 9.7：低能 `e^-e^+` 湮灭输入下的参数反解

取低能质心系输入

```text
e^- + e^+ -> γ + γ,
E_γ = m_e c^2,
E_tot = 2 m_e c^2,
q_{e^-} = -e_0,
q_{e^+} = +e_0,
e_0 = 1.602176634 × 10^(-19) C,
m_e c^2 = 0.51099895069 MeV.
```

以单个末态光子能量为归一单位，记

```text
1 γ-unit := m_e c^2 = 0.51099895069 MeV = 510.99895069 keV.
```

则

```text
E_tot = 2 γ-unit.
```

并把电荷按 `e_0` 归一，则

```text
\widehat{q}_{e^-} = -1,
\widehat{q}_{e^+} = +1.
```

对第 8 节双单理由子锁定构型，

```text
G_* = a I + 2b J_1 + σ a J_2,
a^2 + b^2 = 1,
|q_*^perp| = 1.
```

若把低能正反湮灭后的总释放量对应到第 9.5 节的纯尺度残留

```text
μ_pair = w_* - 1,
```

并按 `γ-unit` 归一，则必有

```text
μ_pair = 2.
```

又因

```text
w_* = 2 + 2b^2,
```

故

```text
2 = μ_pair = w_* - 1 = 1 + 2b^2,
b^2 = 1/2,
a^2 = 1/2.
```

于是

```text
a = b = 1/sqrt(2),
G_*
=
(1/sqrt(2)) I
+ sqrt(2) J_1
+ σ (1/sqrt(2)) J_2.
```

### 推论 9.7.1：对应质量与电荷读出

在定理 9.7 的反解下，

```text
c_* = sqrt(1 + 3b^2) = sqrt(5/2),
μ_* = c_* - 1 = sqrt(5/2) - 1,
q_*^bridge = (2b)/a = 2,
q_*^perp = σ = \widehat{q}_{e^\pm} = ±1,
w_* = 2 + 2b^2 = 3.
```

数值上，

```text
μ_* ≈ 0.5811388301.
```

故低能 `e^-e^+` 输入与本文锁定构型相配时：

```text
1. 电荷输入只固定 q_*^perp = ±1；
2. 能量输入迫出 a^2 = b^2 = 1/2；
3. 桥内相位读出随之固定为 q_*^bridge = 2。
```

### 推论 9.7.2：各部理由贡献分布

在定理 9.7 的反解下，总锁定词的模长分布为

```text

=
|G_*|^2
=
a^2 + (2b)^2 + a^2
=
1/2 + 2 + 1/2
=
3.
```

三部分对应：

而正反湮灭时的纯尺度残留分布为

=
3b^2 + a^2
=
3/2 + 1/2
=
2.

故总释放量 `2 γ-unit` 的内部理由分布为

2. 正交锁定部分：1/2 γ-unit，占 1/4。
```

换算为能量，

3/2 γ-unit = 0.766498426035 MeV = 766.498426035 keV,

若按正反对称把这两部分平均分配到 `e^-` 与 `e^+` 两侧，则每一侧对应

### 推论 9.7.3：多重实验口径的一致性核验

低能 `e^-e^+` 湮灭的实验输入可写成以下等价组：

5. 文章常用圆整口径：

```

### 推论 9.7.4：与电子质量量级的对照

记电子静质量与其能量当量分别为

```text
m_e = 9.1093837139 × 10^(-31) kg,
m_e c^2 = 0.51099895000 MeV.
```

在定理 9.7 的反解下，

```text
μ_* = sqrt(5/2) - 1,
m_* c^2 = μ_* E_γ,
m_* / m_e = [μ_* E_γ] / [m_e c^2].
```

故

```text
E_γ = [m_e c^2] / μ_*
```

时，有

```text
m_* = m_e.
```

数值上，

```text
μ_* ≈ 0.5811388301,
[m_e c^2] / μ_* ≈ 0.8793061547 MeV.
```

因此：

```text

2. 当 E_γ = 0.8793061547 MeV 时，m_* = m_e；
3. 当 E_γ > 0.8793061547 MeV 时，m_* > m_e。
```

```text

```

故在双单理由子锁定构型中，若输入位于典型核伽马能区，则所得质量当量稳定落在正常电子质量的同一量级。

### 定理 9.7.5：`μ` 子与 `τ` 子的电子核-正交虚轴反解

取实验输入

```text
m_e c^2   = 0.51099895000 MeV,
m_μ c^2   = 105.6583755 MeV,
m_τ c^2   = 1776.86 MeV.
```

并取主要衰变支

```text
μ^- -> e^- + ν̄_e + ν_μ,
τ^- -> e^- + ν̄_e + ν_τ,
τ^- -> μ^- + ν̄_μ + ν_τ,
```

并允许 `τ` 子的强子支；其轻子末态仍退化到 `e^-` 或先退化到 `μ^-` 再退化到 `e^-`。

记标准化电子核为

```text
U_e := (I-J) / sqrt(2),
|U_e| = 1,
q_e = -1.
```

对 `L ∈ {μ, τ}`，若要求：

```text
1. 带电核与电子同型；
2. 额外贡献对电荷读出盲；
3. 额外贡献只落在与 U_e 正交的虚轴方向；
4. 反解只使用实验质量比与前述内部闭式，
```

则 `L` 的最小表达只能写成

```text
G_L = c_e U_e + h_L K_L,
<U_e, K_L> = 0,
|K_L| = 1,
```

其中 `c_e` 为电子核尺度，`h_L` 为正交虚轴厚化量。记

```text
r_L := m_L / m_e,
η_L := h_L / c_e,
```

则

```text
r_L^2 = 1 + η_L^2,
η_L = sqrt(r_L^2 - 1).
```

数值上，

```text
r_μ   = 206.768282987666,
η_μ   = 206.765864807679,

r_τ   = 3477.22828001897,
η_τ   = 3477.22813622628.
```

并且

```text
h_μ c^2 = m_e c^2 η_μ = sqrt[(m_μ c^2)^2 - (m_e c^2)^2]
        = 105.657139812566 MeV,

h_τ c^2 = m_e c^2 η_τ = sqrt[(m_τ c^2)^2 - (m_e c^2)^2]
        = 1776.85992652209 MeV.
```

### 推论 9.7.6：正交虚轴厚化的占比

在定理 9.7.5 的条件下，有

```text
η_L^2 / r_L^2 = 1 - 1/r_L^2.
```

故

```text
η_μ^2 / r_μ^2 = 0.999976609895761,
η_τ^2 / r_τ^2 = 0.999999917294652.
```

因此在“电子核保持不变 + 正交虚轴厚化”这一最小反解下，`μ` 子与 `τ` 子的质量尺度几乎全部由正交虚轴厚化项承担，而电子核只保留共同带电骨架。

### 定理 9.7.7：轻子衰变支上的中性总余词

设

```text
L_i^- -> L_j^- + ν̄_j + ν_i,
```

其中 `L_i, L_j ∈ {e, μ, τ}`，且 `m_i > m_j`。若要求

```text
1. 只使用初态质量、末态带电轻子质量与末态电荷；
2. 带电骨架由 L_j 保留；
3. 中性末态对带电读出盲；
4. 额外贡献只记入与带电骨架正交的中性层，
```

则二中性末态的总余词只能写成

```text
N_{ij} := G_{L_i} - G_{L_j} = h_{ij} K_{ij},
<K_{ij}, U_j> = 0,
|K_{ij}| = 1.
```

记

```text
r_{ij} := m_i / m_j,
ζ_{ij} := h_{ij} / c_j,
```

则

```text
r_{ij}^2 = 1 + ζ_{ij}^2,
ζ_{ij} = sqrt(r_{ij}^2 - 1),
h_{ij} c^2 = sqrt[(m_i c^2)^2 - (m_j c^2)^2].
```

相对于末态带电骨架，二中性末态唯一可解者仅有

```text
1. 总电荷读出为零；
2. 带电平面内无额外实桥接与虚桥接读出；
3. 全部总余量落在正交中性层厚化 h_{ij} 上。
```

```text
μ^- -> e^- + ν̄_e + ν_μ :
ζ_{μe} = 206.765864807679,
h_{μe} c^2 = 105.657139812566 MeV;

τ^- -> μ^- + ν̄_μ + ν_τ :
ζ_{τμ} = 16.7872712364959,
h_{τμ} c^2 = 1773.71580792603 MeV;

τ^- -> e^- + ν̄_e + ν_τ :
ζ_{τe} = 3477.22813622628,
h_{τe} c^2 = 1776.85992652209 MeV.
```

且有

```text
h_{τe}^2 = h_{τμ}^2 + h_{μe}^2.
```

### 定理 9.7.8：单个中微子分解的不唯一性

在定理 9.7.7 的条件下，衰变结果

```text
L_i^- -> L_j^- + ν̄_j + ν_i
```

唯一决定总余词 `N_{ij}`，不能唯一决定单个中微子词。

证明。设

```text
N_{ij} = X + Y,
```

其中 `X, Y` 对应两个中性末态。对任意实数 `t`，取

```text
X_t := t K_{ij},
Y_t := (h_{ij} - t) K_{ij},
```

则恒有

```text
X_t + Y_t = N_{ij}.
```

故只由初态质量、末态带电轻子质量、末态电荷与“存在两个中性末态”这一结果，不能把

```text
ν̄_j, ν_i
```

各自的实轴分量、虚轴分量与厚化量唯一反解。严格可定者仅为

```text
N_{ij} = h_{ij} K_{ij}.
```

```text
1. 单个中微子词不由这些数据唯一确定；
2. 可唯一确定者仅为二中微子总词；
3. 该总词对带电读出盲，并完全落在正交中性层。
```

### 定理 9.7.9：事件级二中性总系统的精确反解

设衰变在初态静止系中发生：

```text
L_i^- -> L_j^- + ν̄_j + ν_i,
```

记实测末态带电轻子能量为 `E_j`，二中性末态总系统为 `R_{ij}`。由能量-动量守恒有

```text
E_{R,ij} = m_i - E_j,
|p_{R,ij}| = sqrt[E_j^2 - m_j^2],
M_{R,ij}^2 = E_{R,ij}^2 - |p_{R,ij}|^2
          = m_i^2 + m_j^2 - 2 m_i E_j.
```

对任意事件，`E_j` 已知时

```text
1. 总能量 E_{R,ij}；
2. 总动量模 |p_{R,ij}|；
3. 总不变量尺度 M_{R,ij}
```

均被唯一反解。

其允许区间为

```text
m_j <= E_j <= [m_i^2 + m_j^2] / [2m_i],
0 <= M_{R,ij} <= m_i - m_j.
```

### 推论 9.7.10：端点处的非零光样总系统

在定理 9.7.9 的条件下，若带电末态取到能谱端点

```text
E_j = E_{j,max} := [m_i^2 + m_j^2] / [2m_i],
```

则

```text
M_{R,ij} = 0,
E_{R,ij} = [m_i^2 - m_j^2] / [2m_i] > 0.
```

端点处二中性总系统为非零光样总系统。

```text
μ^- -> e^- + ν̄_e + ν_μ :
E_{e,max}    = 52.8304234302083 MeV,
E_{R,μe,end} = 52.8279520697917 MeV,
M_{R,μe,end} = 0;

τ^- -> e^- + ν̄_e + ν_τ :
E_{e,max}    = 888.430073477912 MeV,
E_{R,τe,end} = 888.429926522088 MeV,
M_{R,τe,end} = 0;

τ^- -> μ^- + ν̄_μ + ν_τ :
E_{μ,max}    = 891.571410216139 MeV,
E_{R,τμ,end} = 885.288589783861 MeV,
M_{R,τμ,end} = 0.
```

### 推论 9.7.11：由能谱连续性得到的中性总系统连续族

若实验给出末态带电轻子连续能谱，则由定理 9.7.9 得连续族

```text
R_{ij}(E_j),
M_{R,ij}(E_j) = sqrt[m_i^2 + m_j^2 - 2m_i E_j].
```

对 `μ^- -> e^- + ν̄_e + ν_μ`，取 Michel 能谱

```text
x := 2E_e / m_μ,
dΓ/dx ∝ x^2 (6 - 4x),
```

无质量近似下有

```text
<E_e> = 0.35 m_μ = 36.980431425 MeV.
```

从而

```text
<E_{R,μe}> = 68.677944075 MeV,
<M_{R,μe}>_{from <E_e>}
= sqrt[m_μ^2 + m_e^2 - 2m_μ <E_e>]
= 57.8737316398088 MeV.
```

实验能谱给定时，二中性总系统的事件级总尺度逐点唯一确定。

（实验口径参见：`Measurement of the Michel Parameter ρ in Muon Decay`；`Precision measurement of the muon decay parameters rho and delta`；`Blind analysis results of the TWIST experiment`。）

### 定理 9.7.12：事件级公共实轴与虚轴减法

在定理 9.7.9 的条件下，取 Dirac 公共模态中的时间向读出为事件级实轴量，空间向模读出为事件级虚轴量。记

```text
Re_ev(X) := ρ_X,
Im_ev(X) := |κ_X|.
```

在初态静止系中，

```text
Re_ev(L_i) = m_i,
Im_ev(L_i) = 0.
```

对实测带电末态 `L_j^-`，

```text
Re_ev(L_j) = E_j,
Im_ev(L_j) = sqrt[E_j^2 - m_j^2].
```

余下二中性总系统 `R_{ij}` 的事件级公共实轴与虚轴量为

```text
Re_ev(R_{ij}) = m_i - E_j,
Im_ev(R_{ij}) = sqrt[E_j^2 - m_j^2],
M_{R,ij}^2 = Re_ev(R_{ij})^2 - Im_ev(R_{ij})^2.
```

同一事件级公共轴上的减法闭式只能为

```text
Re_ev(neutral total) = Re_ev(initial) - Re_ev(charged final),
Im_ev(neutral total) = Im_ev(charged final).
```

```text
μ^- -> e^- + ν̄_e + ν_μ :

端点处：
Re_ev(R_{μe,end}) = 52.8279520697917 MeV,
Im_ev(R_{μe,end}) = 52.8279520697917 MeV,
M_{R,μe,end}      = 0;

由 <E_e> = 36.980431425 MeV 得：
Re_ev(<R_{μe}>) = 68.677944075 MeV,
Im_ev(<R_{μe}>) = 36.9769007415741 MeV,
M_{R,μe}        = 57.8737316398088 MeV.

τ^- -> e^- + ν̄_e + ν_τ :

端点处：
Re_ev(R_{τe,end}) = 888.429926522088 MeV,
Im_ev(R_{τe,end}) = 888.429926522088 MeV,
M_{R,τe,end}      = 0.

τ^- -> μ^- + ν̄_μ + ν_τ :

端点处：
Re_ev(R_{τμ,end}) = 885.288589783861 MeV,
Im_ev(R_{τμ,end}) = 885.288589783861 MeV,
M_{R,τμ,end}      = 0.
```

### 定理 9.7.13：即使总余四词已知，单个中微子仍不唯一

在定理 9.7.12 的条件下，即使二中性总系统的事件级公共四词

```text
R_{ij} = (Re_ev(R_{ij}), κ_{R,ij})
```

已被唯一反解，单个中微子仍不由这些数据唯一决定。

证明。若

```text
R_{ij} = N_1 + N_2,
```

向量和保持不变时，即存在无穷多组分解。

当 `M_{R,ij} > 0` 时，在 `R_{ij}` 的静止系中有

```text
R_{ij}^rest = (M_{R,ij}, 0).
```

对任意单位向量 `n`，取

```text
N_1^(n) := (M_{R,ij}/2, (M_{R,ij}/2) n),
N_2^(n) := (M_{R,ij}/2, -(M_{R,ij}/2) n),
```

则

```text
N_1^(n) + N_2^(n) = R_{ij}^rest.
```

当 `M_{R,ij} = 0` 时，对任意 `0 <= t <= Re_ev(R_{ij})`，取

```text
N_1^(t) := (t, t n),
N_2^(t) := (Re_ev(R_{ij}) - t, [Re_ev(R_{ij}) - t] n),
```

其中 `n` 为 `κ_{R,ij}` 的单位方向，则

```text
N_1^(t) + N_2^(t) = R_{ij}.
```

```text
1. 实验能谱与端点数据可唯一给出二中性总系统的公共实轴与虚轴总量；
2. 这些总量可严格由初态减去已知带电末态得到；
3. 单个中微子的事件级实轴与虚轴贡献仍不是这些数据的唯一函数。
```

### 推论 9.7.14：实验中的缺失能量口径即二中性总能量口径

对

```text
L_i^- -> L_j^- + ν̄_j + ν_i,
```

由定理 9.7.9，

```text
E_{R,ij} = m_i - E_j.
```

实验上若公布

```text
1. 带电轻子能谱 E_j；
2. 缺失能量 E_miss；
3. 不可见能量 E_invis；
4. 缺失质量 M_miss，
```

则本文口径下对应为

```text
E_miss = E_invis = E_{R,ij},
M_miss = M_{R,ij}.
```

实验文献若给出带电轻子能谱、不可见能量切选或缺失质量分布，则其对象与 `R_{ij}` 同一。

对辐射轻子衰变

```text
L_i^- -> L_j^- + γ + ν̄_j + ν_i,
```

则

```text
E_{R,ij}^{(γ)} = m_i - E_j - E_γ.
```

低不可见能量区即二中性总系统低能区。以

```text
E_e > 45 MeV,
E_γ > 40 MeV
```

为例，

```text
E_{R,μe}^{(γ)}
= m_μ - E_e - E_γ
< 105.6583755 - 45 - 40
= 20.6583755 MeV.
```

相应事件区满足二中性总系统总能量很小。

（实验口径参见：`Measurement of the radiative decay of polarized muons in the MEG experiment`。）

## 10. 相对速度的内部迫出

由

```text
G = c(u+iv),
u^2 + v^2 = 1,
q = v/u,
w = c^2
```

单位相位层的无量纲数据由 `<u,v>` 及其射影读出 `q` 给出。

### 命题 10.1：相对速度读出的唯一形式

若把宏观相对速度对应为单位相位层中虚轴运输份额的无量纲读出，并要求其

```text
1. 只依赖既有对象 <u,v>；
2. 在共轭下取偶；
3. 在纯实桥接时为 0；
4. 只度量虚轴运输份额大小；
```

则必有

```text
β
=
|v|
=
|q| / sqrt(1+q^2)
=
|b| / sqrt([1+μ]^2 + b^2)
=
|b| / sqrt(w),
0 <= β <= 1.
```

并且

```text
u^2 = 1 - β^2,
q^2 = β^2 / (1-β^2).
```

### 定理 10.2：局部桥接层中的速度-模长闭式

在 `1+μ != 0` 时，

```text
w
=
[1+μ]^2 + b^2
=
[1+μ]^2 / (1-β^2),
```

故

```text
|1+μ| = sqrt[w (1-β^2)],
|b|   = sqrt[w] β.
```

### 推论 10.2.1：速度读出的唯一逆解

在 `0 <= β < 1` 时，

```text
|q| = β / sqrt(1-β^2),
|u| = sqrt(1-β^2),
|v| = β.
```

故单位相位层中，速度读出 `β` 与射影电荷读出绝对值 `|q|` 彼此唯一决定。

### 定义 10.2.2：速度增厚因子

定义

```text
γ(β) := 1 / sqrt(1-β^2),
0 <= β < 1.
```

### 定理 10.2.3：速度增厚因子的等价表达式

在 `1+μ != 0` 时，

```text
γ(β)
=
sqrt(1+q^2)
=
sqrt(w) / |1+μ|.
```

并且

```text
w = γ(β)^2 [1+μ]^2.
```

### 定理 10.2.4：局部桥接层的完全反解

设

```text
0 <= β < 1,
w > 0,
σ_u := sign(1+μ) ∈ {-1,+1},
σ_b := sign(b) ∈ {-1,+1}.
```

则

```text
1+μ = σ_u sqrt[w (1-β^2)],
μ   = σ_u sqrt[w (1-β^2)] - 1,
b   = σ_b sqrt[w] β,
q   = (σ_b/σ_u) β / sqrt(1-β^2).
```

故在离开纯虚边界之后，数据

```text
<w, β, σ_u, σ_b>
```

一一对应于

```text
<μ, b, q>.
```

### 推论 10.2.5：光样边界与纯虚极限

在 `w > 0` 下，下列条件等价：

```text
1. 1+μ = 0；
2. β = 1；
3. |b| = sqrt(w)；
4. q 不再有有限定义。
```

并且当

```text
β -> 1^-
```

时，

```text
|1+μ| -> 0,
|q|   -> +∞,
γ(β)  -> +∞.
```

### 定理 10.2.6：局部桥接层参数化的充足性

对任意数据

```text
w > 0,
0 <= β < 1,
σ_u, σ_b ∈ {-1,+1},
```

按定理 10.2.4 定义

```text
1+μ = σ_u sqrt[w(1-β^2)],
b   = σ_b sqrt[w] β,
q   = (σ_b/σ_u) β / sqrt(1-β^2),
G   = (1+μ) + i b,
```

则必有

```text
|G|^2 = w,
|b| / |G| = β,
q = b/(1+μ).
```

故局部桥接层上一切满足

```text
w > 0,
0 <= β < 1
```

的数据，与

```text
<w, β, σ_u, σ_b>
```

一一对应。

### 推论 10.3：固定总模长时的桥接质量-速度关系

若总模长词 `w` 固定，则

```text
|1+μ| = sqrt[w (1-β^2)]
```

随 `β` 严格减小，并且

```text
w = [1+μ]^2 / (1-β^2).
```

在 `0 <= β < 1` 上，

```text
d|1+μ|/dβ = - sqrt(w) β / sqrt(1-β^2) < 0    (β > 0),
d|b|/dβ   =   sqrt(w) > 0.
```

### 定义 10.4：离散 Dirac 层的宏观速度读出

设

```text
[D_A^0, M] = 0,
M = m I,
```

并取某个共同差分模态 `Φ != 0` 满足

```text
Δ_r Φ = ρ Φ,
Δ_a Φ = κ_a Φ,   a=1,...,d.
```

定义

```text
β_A(Φ)^2 := [Σ_{a=1}^d κ_a^2] / ρ^2,
```

其中 `ρ != 0`。

### 定理 10.5：离散 Dirac 层的爱因斯坦型闭式

在定义 10.4 的条件下，若

```text
[γ_0^(A) Δ_r + Σ_{a=1}^d γ_a^(A) Δ_a + m I] Φ = 0,
```

则必有

```text
ρ^2 - Σ_{a=1}^d κ_a^2 = m^2,
β_A(Φ)^2 = [Σ_{a=1}^d κ_a^2] / ρ^2 <= 1,
```

当 `β_A < 1` 时，

```text
ρ^2 = m^2 / (1-β_A^2),
Σ_{a=1}^d κ_a^2 = m^2 β_A^2 / (1-β_A^2),
|ρ| = |m| / sqrt(1-β_A^2).
```

一般地，

```text
m^2 = ρ^2 (1-β_A^2),
Σ_{a=1}^d κ_a^2 = ρ^2 β_A^2.
```

并且

```text
β_A^2 = 1  <=>  m = 0.
```

### 证明

由主文档的精确二阶闭式，

```text
[Δ_r^2 - Σ_{a=1}^d Δ_a^2 - m^2] Φ = 0.
```

代入

```text
Δ_r Φ = ρ Φ,
Δ_a Φ = κ_a Φ
```

即得

```text
ρ^2 - Σ_{a=1}^d κ_a^2 = m^2.
```

两边同除以 `ρ^2` 即得所述全部结论。

### 推论 10.6：固定非零质量时的速度-时间向增厚

若共同质量词满足

```text
m != 0,
0 <= β_A < 1,
```

则

```text
|ρ| = |m| / sqrt(1-β_A^2)
```

随 `β_A` 严格增大，并且

```text
Σ_{a=1}^d κ_a^2 = |ρ|^2 β_A^2.
```

并且在 `0 <= β_A < 1` 上，

```text
d|ρ|/dβ_A = |m| β_A / (1-β_A^2)^(3/2) > 0    (β_A > 0).
```

### 推论 10.6.1：不固定质量时的完全反解

在 `0 <= β_A <= 1` 与 `ρ != 0` 下，

```text
m^2 = ρ^2 (1-β_A^2),
Σ_{a=1}^d κ_a^2 = ρ^2 β_A^2.
```

故数据

```text
<ρ, β_A>
```

唯一决定

```text
<m^2, Σ_a κ_a^2>.
```

若再给定空间差分向量的方向单位词

```text
n_a,
Σ_a n_a^2 = 1,
```

则进一步有

```text
κ_a = ε ρ β_A n_a,
```

其中 `ε ∈ {-1,+1}` 对应整体方向分支。

### 推论 10.6.2：光子样模态的严格判据

在 `ρ != 0` 下，下列条件等价：

```text
1. m = 0；
2. β_A = 1；
3. Σ_{a=1}^d κ_a^2 = ρ^2.
```

故在离散 Dirac 层，零质量模态与光样极限一一对应。

### 定理 10.6.3：Dirac 标量层参数化的充足性

对任意数据

```text
ρ != 0,
0 <= β_A <= 1,
n_a,  Σ_a n_a^2 = 1,
ε ∈ {-1,+1},
```

定义

```text
m^2 := ρ^2 (1-β_A^2),
κ_a := ε ρ β_A n_a.
```

则必有

```text
Σ_{a=1}^d κ_a^2 = ρ^2 β_A^2,
ρ^2 - Σ_{a=1}^d κ_a^2 = m^2.
```

故在共同差分模态层，全部合法标量数据与

```text
<ρ, β_A, n, ε>
```

一一对应。

### 推论 10.7：局部层与宏观层的统一速度表达式

若宏观模态的速度读出与单步局部更新的单位相位层速度读出对应于同一运输份额，则

```text
β_A = β.
```

此时同时有

```text
w = [1+μ]^2 / (1-β^2),
ρ^2 = m^2 / (1-β^2).
```

### 定理 10.7.1：局部层与 Dirac 层的兼容不变量

在推论 10.7 的条件下，若再满足

```text
1+μ != 0,
m != 0,
β = β_A < 1,
```

则必有

```text
w / ρ^2 = [1+μ]^2 / m^2.
```

故局部桥接层与离散 Dirac 层在同一速度份额下的兼容条件只取决于两层的实桥接尺度比。

### 推论 10.7.2：两层兼容的充要条件

在

```text
1+μ != 0,
m != 0,
w > 0,
ρ != 0
```

下，局部层与 Dirac 层存在共同速度读出

```text
β = β_A < 1
```

当且仅当

```text
w m^2 = ρ^2 [1+μ]^2.
```

### 证明

必要性由定理 10.7.1 直接得到。

反之，若

```text
w m^2 = ρ^2 [1+μ]^2,
```

则由

```text
w = [1+μ]^2 / (1-β^2)
```

与

```text
ρ^2 = m^2 / (1-β_A^2)
```

可得

```text
1-β^2 = [1+μ]^2 / w = m^2 / ρ^2 = 1-β_A^2,
```

故 `β = β_A`。

### 推论 10.8：三组直接验算

```text
1. 当 q = 1 时，
   β = 1/sqrt(2),
   γ = 1/sqrt(1-β^2) = sqrt(2) = sqrt(1+q^2)；

2. 当 q = 2 时，
   β = 2/sqrt(5),
   γ = 1/sqrt(1-β^2) = sqrt(5) = sqrt(1+q^2)；

3. 取 β = 0.8：
   若 w = 9，则 |1+μ| = sqrt[9(1-0.64)] = 1.8，|b| = 3 × 0.8 = 2.4，
   且 [1+μ]^2 + b^2 = 1.8^2 + 2.4^2 = 9；
   若 m = 2，则 |ρ| = 2/sqrt(1-0.64) = 10/3，
   Σ_a κ_a^2 = 4 × 0.64 / 0.36 = 64/9，
   且 ρ^2 - Σ_a κ_a^2 = 100/9 - 64/9 = 4 = m^2。
```

### 推论 10.9：三组无预设验算

```text
1. 取 β = 0.8，w = 9，σ_u = +1，σ_b = -1，
   则 1+μ = 1.8，μ = 0.8，b = -2.4，q = -4/3，
   并且 [1+μ]^2 + b^2 = 9，
   [1+μ]^2 [1+q^2] = 9；

2. 取 ρ = 5，β_A = 0.8，
   则 m^2 = 25(1-0.64) = 9，故 m = 3，
   且 Σ_a κ_a^2 = 25 × 0.64 = 16，
   并且 ρ^2 - Σ_a κ_a^2 = 25 - 16 = 9 = m^2；

3. 取 ρ = 5，m = 0，
   则 Σ_a κ_a^2 = 25，β_A = 1；
   取 w = 9，β = 0.999，
   则 |1+μ| = sqrt[9(1-0.999^2)] ≈ 0.1341305334，
   |q| = 0.999 / sqrt(1-0.999^2) ≈ 22.3439057701，
   即为局部层向纯虚光样边界的数值逼近。
```

### 推论 10.10：两组充足性与兼容验算

```text
1. 取 ρ = 7，β_A = 0.6，n = (0.6,0.8)，ε = +1，
   则 m^2 = 49(1-0.36) = 31.36，
   κ_1 = 7 × 0.6 × 0.6 = 2.52，
   κ_2 = 7 × 0.6 × 0.8 = 3.36，
   且 κ_1^2 + κ_2^2 = 17.64，
   49 - 17.64 = 31.36 = m^2；

2. 取 w = 9，β = 0.8，ρ = 5，m = 3，
   则 1+μ = sqrt[9(1-0.64)] = 1.8，
   并且 w m^2 = 9 × 9 = 81，
   ρ^2 [1+μ]^2 = 25 × 1.8^2 = 81，
   因而两层兼容条件精确成立。
```

## 11. 与 Dirac 对照中的角色分层

由主文档已严格推出

```text
[γ_0^(A) Δ_r + Σ_a γ_a^(A) Δ_a + m I] Φ = 0.
```

其中 `m` 对应共同实投影词。

由前述各节，

```text
一般质量读出  = μ = Re[χ]
一般电荷读出  = q = v/u = b/(1+μ)
特殊锁定标签  = q_*^perp = ±1
```

对应式为：

```text
1. 质量对应零阶实桥接词；
2. 一般电荷对应单位相位层的相位运输读出；
3. 特殊双轴锁定态还可带有正交锁定电荷标签。
```

并且

```text
局部有效耦合先形成更新词 G，
质量、电荷与锁定标签是 G 的分层内部读出。
```

对应关系为

```text
m      -> 离散 Dirac 型零阶项
q      -> 单位相位层运输参数
q_*^perp -> 特殊锁定构型标签
```

## 12. 总结定理

在四篇基础文档与主文档既有对象范围内，关于质量、电荷、双单理由子锁定构型以及两种中和，可严格推出：

```text
1. 一步局部更新唯一由相对有效耦合词 χ 决定；
2. 真正作用在旧载体上的词是 G = 1 + χ；
3. 质量对应量 μ = Re[χ] 是 G 的实桥接增量；
4. 原始虚轴增量为 b = Im[χ]；
5. 电荷对应量若要内生迫出，必须落在 G 的单位相位层上；
6. 在 1+μ != 0 时，电荷对应词唯一为 q := v/u = b/(1+μ)；
7. 因而 q 不是独立本体量，而是虚轴贡献相对于实桥接的射影读出；
8. 纯虚边界 1+μ = 0 是电荷读出的射影边界；
9. 可见总量变化满足 w - 1 = 2μ + μ^2 + b^2 = 2μ + |χ|^2；
10. 因而 w = [1+μ]^2 [1+q^2]；
11. 给定 (w,q,s) 可唯一反解 (μ,b)；
12. q 的更原始对象不是比值本身，而是单位相位层坐标 <u,v>；
13. 更原始地，q 是射影角类 [φ] 的读出，满足 q = tan φ；
14. 两步复合时，μ 与 b 满足精确耦合复合律；
15. q 满足射影分式复合律，w 满足乘法复合律；
16. 角类 [φ] 按加法复合，而 q 为其射影像；
17. 局部电荷中和的精确条件是 b_1(1+μ_2)+b_2(1+μ_1)=0；
18. 可见模长层对电荷符号盲，只直接识别 q^2 的尺度效应；
19. 共轭下 μ 不变而 q 变号；
20. 电荷词 q 的精确稳定判据由交换子 [D_A^0,Q] = 0 给出；
21. 共轭配对满足 G^c G = w，从而 q_pair = 0；
22. 共轭配对后的剩余质量对应量满足 μ_pair = w - 1；
23. 局部归一层与头部桥接层只直接读取由 w 诱导的归一差量；
24. 双单理由单位模种子可写成 P = aI + bJ_1，满足 a^2 + b^2 = 1；
25. 对第二种子做完整正交转移后，锁定词为 G_* = aI + 2bJ_1 + σ aJ_2；
26. 其桥内尺度 c_* = sqrt(a^2 + 4b^2) = sqrt(1+3b^2)，故必有正质量 μ_* = sqrt(1+3b^2)-1 > 0；
27. 该构型必须区分桥内相位读出 q_*^bridge = (2b)/a 与正交锁定电荷标签 q_*^perp；
28. 在完整转移、相同理由、等模保持下，|q_*^perp| = 1，符号由转移方向决定；
29. 在上述三条件下，|q_*^perp| 与 a,b 的具体取值无关；
30. 数据 <μ_*, c_*, w_*, q_*^bridge> 对 q_*^perp 的符号盲，故 q_*^perp 不是桥内一般读出链中的可反解量；
31. 该构型满足 μ_*^c = μ_*，w_*^c = w_*，(q_*^bridge)^c = -q_*^bridge，(q_*^perp)^c = -q_*^perp；
32. 该构型的总可见模长为 w_* = 2 + 2b^2 = c_*^2 + a^2；
33. 其严格自旋内容不是先验数值，而是双轴锁定的二点共轭轨道 {G_*, G_*^c}；
34. 在 Dirac 支撑归一下，该构型满足 d_eff(G_*) = 2；
35. 若把普通单轴单位模种子的自旋数值记作 1，则该锁定构型的自旋数值唯一为 1/2；
36. 宏观普通中和只意味着标签和为零，一般不改变内部有效理由；
37. 正反湮灭要求共轭匹配与理由回退，因此强于普通中和；
38. 双单理由子锁定构型上的真正湮灭要求 \widetilde{G}_* = G_*^c，即各幅值逐项匹配且全部奇轴系数逐项反号；
39. 双单理由子构型中，普通中和不会自动拆除双轴锁定，只有湮灭才会把有效理由真正重新变回去；
40. 在 Dirac 对照中，m 对应共同实桥接词，q 对应单位相位层的相位运输读出，而 q_*^perp 属于特殊锁定构型标签；
41. 若相对速度读出只允许依赖单位相位层且对共轭取偶，则其唯一形式为 β = |v| = |q|/sqrt(1+q^2) = |b|/sqrt(w)；
42. 因而 u^2 = 1-β^2，|q| = β/sqrt(1-β^2)，并且 β 与 |q| 在定义域上彼此唯一决定；
43. 速度增厚因子 γ(β) := 1/sqrt(1-β^2) 同时满足 γ = sqrt(1+q^2) = sqrt(w)/|1+μ|；
44. 因而 w = [1+μ]^2/(1-β^2) = γ(β)^2 [1+μ]^2；
45. 在 0 <= β < 1 时，给定 <w,β,σ_u,σ_b> 可唯一反解 <μ,b,q>；
46. 在 w > 0 下，1+μ = 0、β = 1、|b| = sqrt(w) 与 q 失去有限定义彼此等价；
47. 在固定总模长词 w 时，实桥接尺度 |1+μ| = sqrt[w(1-β^2)] 严格减小，而 |b| = sqrt(w)β 严格增大；
48. 在 [D_A^0,M]=0 的共同质量层上，若 Δ_r Φ = ρΦ 且 Δ_a Φ = κ_a Φ，则 β_A^2 = [Σ_a κ_a^2]/ρ^2；
49. 同时必有 0 <= β_A^2 <= 1，且 ρ^2 - Σ_a κ_a^2 = m^2，从而既可写成 ρ^2 = m^2/(1-β_A^2)，也可写成 m^2 = ρ^2(1-β_A^2)；
50. 因而在 0 <= β_A <= 1 与 ρ != 0 时，给定 <ρ,β_A> 可唯一反解 <m^2, Σ_a κ_a^2>；
51. 在 ρ != 0 下，m = 0、β_A = 1 与 Σ_a κ_a^2 = ρ^2 彼此等价，故零质量模态恰对应光样极限；
52. 若宏观速度读出与局部单位相位层速度读出对应同一运输份额，且 1+μ != 0、m != 0，则兼容不变量满足 w/ρ^2 = [1+μ]^2/m^2；
53. 因此质量与电荷不是两个外加本体量，而是同一局部更新词及其特殊锁定构型中的分层读出。
```

以上给出在现有文档内部，对质量、电荷、双单理由子锁定构型以及两种中和所能迫出的闭合链。
