# 由果溯因复载体相对转移级数

## 1. 目的

本文处理以下问题：

1. 任意合法因果推断下，相对转移量级数的最小形式是什么。
2. 该级数为何不能先验压成普通单变量函数级数。
3. 该级数在概率层的实值形式是什么。
4. 该级数在前读出层的复载体形式是什么。
5. 两种形式如何严格对应。
6. 等等性质。

全文只用有限词、有限前缀链、有限项运算。

## 3. 前读出载体的必然性

### 定理 3.1：前读出载体必须存在

对任意相关路径-轴对 `(p_j,ξ_k)`，在进入

```text
Read_Σ(z_{j,k})
```

之前，必须存在不同于终点实读出值本身的前读出载体层。

### 证明

由 `K3`，结果权词由正贡献理由汇聚而成。由 `R5`，合法使用必须沿全部正贡献链继续展开，不得遗漏正贡献因子。由普通单子的充足理由不可逆返，已进入结果链的正贡献不能逆返其来源。由 `R9`，在未满足闭合退化条件时，不得把当前截面强行压成终点满值。由 `K4`，相关闭组中的满权不得失落。故在进入终点实读出之前，前读出转移差异既不能被提前读成终点值，也不能无载体失落，因而前读出载体层必须存在。

### 定理 3.2：前读出数值载体不能只取实数

若要求前读出载体仍以数值载体表达，则该载体不能只取实数，必须取复数。

### 证明

非负实提升在固定 `Read_Σ` 后只有唯一提升，因而把前读出载体压成终点实读出。纯实提升在固定 `Read_Σ` 后至多只剩符号差别，而方向符号已由

```text
σ_k(p_j)
```

单独记录，且 `σ_k(p_j)` 只取 `-1, 0, +1` 三值，故纯实提升全部塌缩到方向层。复数前向支撑在固定 `Read_Σ` 后仍保留单位模提升，且该差别不等同于方向记录。故前读出数值载体不能只取实数，必须取复数。

## 4. 前缀精化链

### 定义 4.1：合法补全词

对给定结果 `E` 与背景 `γ`，取一个合法补全词：

```text
w = z_1 z_2 ... z_L
```

其中每个 `z_r` 都表示一次合法补入的前读出正贡献项。

### 定义 4.2：前缀精化链

记：

```text
w^(0) = empty
w^(r) = z_1 z_2 ... z_r,   1 <= r <= L
```

并把

```text
w^(0) -> w^(1) -> ... -> w^(L)
```

称为由 `w` 诱导的前缀精化链。

### 定义 4.3：前缀支持词

对每个前缀 `w^(r)`，记其在候选头部 `Head(E)=<R_1,...,R_n>` 上诱导的支持词为：

```text
q^(r) = <q_1^(r),...,q_n^(r)>
```

并记：

```text
q_tot^(r) := Σ_i q_i^(r)
```

在

```text
q_tot^(r) > 0
```

时，记：

```text
P_i^(r) := q_i^(r) / q_tot^(r)
```

### 命题 4.3.1：前缀概率词的归一化

若

```text
q_tot^(r) > 0
```

则前缀概率词

```text
P^(r) = <P_1^(r),...,P_n^(r)>
```

唯一存在，并且满足：

```text
Σ_i P_i^(r) = 1
```

### 证明

由定义 4.3，

```text
P_i^(r) = q_i^(r) / q_tot^(r)
```

分母为正，故各 `P_i^(r)` 唯一确定。再对 `i=1,...,n` 求和，得：

```text
Σ_i P_i^(r)
=
[1 / q_tot^(r)] Σ_i q_i^(r)
=
q_tot^(r) / q_tot^(r)
=
1
```

### 命题 4.3.1.1：前缀概率分量的闭区间范围

若

```text
q_tot^(r) > 0
```

则对每个 `i`，有：

```text
0 <= P_i^(r) <= 1
```

### 证明

由支持词各分量非负，得

```text
q_i^(r) >= 0
```

并且

```text
q_i^(r) <= Σ_j q_j^(r) = q_tot^(r)
```

两边同除以 `q_tot^(r)` 即得

```text
0 <= P_i^(r) <= 1
```

### 命题 4.3.1.2：非退化前缀概率分量的开区间性质

若

```text
q_tot^(r) > 0
```

且对某个 `i` 满足：

```text
P_i^(r) > 0
```

并且存在某个 `j != i` 使得

```text
P_j^(r) > 0
```

则：

```text
0 < P_i^(r) < 1
```

### 证明

左端不等式由条件成立。又由

```text
Σ_k P_k^(r) = 1
```

与

```text
P_j^(r) > 0
```

从而

```text
Σ_{k != i} P_k^(r) > 0
```

故

```text
P_i^(r) = 1 - Σ_{k != i} P_k^(r) < 1
```

命题成立。

### 命题 4.3.2：可归一化性只由总支撑决定

对任意前缀 `w^(r)`，前缀概率词 `P^(r)` 的存在只由

```text
q_tot^(r)
```

是否为正决定；与该步的

```text
Δ_i^(r), E_i^(r)
```

是否取正、取负或取零无关。

### 证明

由命题 4.3.1，`P^(r)` 的存在当且仅当商式

```text
q_i^(r) / q_tot^(r)
```

可定义，故只取决于 `q_tot^(r)` 是否为正。`Δ_i^(r), E_i^(r)` 都是由前后支持词导出的差量，不进入 `P_i^(r) := q_i^(r) / q_tot^(r)` 的定义，故不决定可归一化性。

### 推论 4.3.3：隐理由不改变可归一化判据

若某前缀支持词包含隐理由贡献，则该前缀概率词仍只由对应支持词的总支撑

```text
q_tot^(r)
```

是否为正决定。

### 证明

命题 4.3.1 与命题 4.3.2 只使用支持词 `q^(r)` 与总支撑 `q_tot^(r)`，不区分该支持词来自显理由还是含隐理由的合法补全。

### 定义 4.4：一步增量与有效信息

对每一步前缀精化

```text
q^(r-1) -> q^(r)
```

定义：

```text
Δ_i^(r)   := q_i^(r) - q_i^(r-1)
Δ_tot^(r) := q_tot^(r) - q_tot^(r-1)
E_i^(r)   := Δ_i^(r) - P_i^(r-1) Δ_tot^(r)
```

若 `Δ_tot^(r) != 0`，再定义：

```text
G_i^(r) := Δ_i^(r) / Δ_tot^(r)
ρ_i^(r) := E_i^(r) / Δ_tot^(r) = G_i^(r) - P_i^(r-1)
λ_r     := Δ_tot^(r) / q_tot^(r)
```

### 命题 4.4.1：一步增量的合法性

对每一步前缀精化，`Δ_i^(r)` 可取正、负或零。该步仍为合法前缀精化，当且仅当：

```text
q_i^(r-1) + Δ_i^(r) >= 0
```

对全部 `i` 成立，并且：

```text
q_tot^(r-1) + Δ_tot^(r) > 0
```

### 证明

由定义 4.4，

```text
q_i^(r) = q_i^(r-1) + Δ_i^(r)
q_tot^(r) = q_tot^(r-1) + Δ_tot^(r)
```

而合法前缀精化要求新前缀支持词各分量非负，且其总支撑为正。故结论成立。

### 命题 4.4.2：一步原始增量的符号补偿必然

1. 若 `Δ_tot^(r) >= 0` 且存在某个 `i` 满足 `Δ_i^(r) < 0`，则存在某个 `j` 满足 `Δ_j^(r) > 0`。
2. 若 `Δ_tot^(r) <= 0` 且存在某个 `i` 满足 `Δ_i^(r) > 0`，则存在某个 `j` 满足 `Δ_j^(r) < 0`。

### 证明

若 `Δ_tot^(r) >= 0` 且存在某个 `i` 满足 `Δ_i^(r) < 0`，但对全部 `j` 都有 `Δ_j^(r) <= 0`，则

```text
Δ_tot^(r) = Σ_j Δ_j^(r) < 0
```

矛盾。第一条成立。第二条同理。

## 5. 实相对转移级数

### 命题 5.1：一步变化公式

对每个 `r` 与 `i`，有：

```text
P_i^(r) - P_i^(r-1) = E_i^(r) / q_tot^(r)
```

若 `Δ_tot^(r) != 0`，则：

```text
P_i^(r) - P_i^(r-1) = λ_r ρ_i^(r)
```

### 证明

第一式即概率文档中的基本差分公式作用于一步前缀精化。第二式由

```text
E_i^(r) = Δ_tot^(r) ρ_i^(r)
```

代入即得。

### 命题 5.1.1：原始增量与一步概率变化的分离判据

对每个 `r` 与 `i`，有：

```text
P_i^(r) > P_i^(r-1)  <=>  Δ_i^(r) > P_i^(r-1) Δ_tot^(r)
P_i^(r) = P_i^(r-1)  <=>  Δ_i^(r) = P_i^(r-1) Δ_tot^(r)
P_i^(r) < P_i^(r-1)  <=>  Δ_i^(r) < P_i^(r-1) Δ_tot^(r)
```

### 证明

由定义 4.4，

```text
E_i^(r) = Δ_i^(r) - P_i^(r-1) Δ_tot^(r)
```

再与命题 5.1 的第一式

```text
P_i^(r) - P_i^(r-1) = E_i^(r) / q_tot^(r)
```

联立即得。

### 推论 5.1.2：原始负增量与一步概率上升相容

对每个 `r` 与 `i`，下列两条成立：

1. 若

```text
Δ_tot^(r) < 0
```

且

```text
P_i^(r-1) Δ_tot^(r) < Δ_i^(r) < 0
```

则

```text
P_i^(r) > P_i^(r-1)
```

2. 若

```text
Δ_tot^(r) > 0
```

且

```text
0 < Δ_i^(r) < P_i^(r-1) Δ_tot^(r)
```

则

```text
P_i^(r) < P_i^(r-1)
```

### 证明

两条都直接由命题 5.1.1 得到。

### 定理 5.2：实相对转移级数

对任意合法补全词 `w = z_1 ... z_L`，有：

```text
P_i^(L) - P_i^(0)
=
Σ_{r=1}^L E_i^(r) / q_tot^(r)
```

若对每个 `r` 都有 `Δ_tot^(r) != 0`，则：

```text
P_i^(L) - P_i^(0)
=
Σ_{r=1}^L λ_r ρ_i^(r)
```

### 证明

对命题 5.1 在 `r=1,...,L` 上逐步求和：

```text
Σ_{r=1}^L (P_i^(r) - P_i^(r-1))
=
P_i^(L) - P_i^(0)
```

右端即得第一式。第二式再由命题 5.1 的第二式逐项代入。

### 命题 5.3：零和性

对每个 `r`，有：

```text
Σ_i E_i^(r) = 0
```

若 `Δ_tot^(r) != 0`，则：

```text
Σ_i ρ_i^(r) = 0
```

### 证明

第一式由 `E_i^(r) := Δ_i^(r) - P_i^(r-1) Δ_tot^(r)` 与

```text
Σ_i P_i^(r-1) = 1
Σ_i Δ_i^(r) = Δ_tot^(r)
```

直接得到。第二式由第一式除以 `Δ_tot^(r)` 得到。

### 命题 5.4：负有效贡献的轴间补偿

对每个 `r`，若

```text
E^(r) != 0
```

则存在正分量当且仅当存在负分量。特别地，若某个 `i` 满足

```text
E_i^(r) < 0
```

则存在某个 `j` 使

```text
E_j^(r) > 0
```

### 证明

由命题 5.3，

```text
Σ_i E_i^(r) = 0
```

若存在负分量而不存在正分量，则全部分量都不大于零，且至少有一个严格小于零，故总和严格小于零，矛盾。反向同理。

### 推论 5.5：实级数是零和有序级数

定理 5.2 中的每一步级数项都落在零和超平面：

```text
<u_1,...,u_n> ,   Σ_i u_i = 0
```

因此该级数不是纯总量级数，而是纯相对转移级数。

## 6. 差分与块形式

### 定义 6.1：一步有效差分词

记：

```text
T_{ij}^(r) := E_i^(r) - E_j^(r)
```

### 命题 6.2：一步有效信息的差分重建

对每个 `r` 与 `i`，有：

```text
E_i^(r) = (1/n) Σ_j T_{ij}^(r)
```

### 证明

这是概率文档中两两有效差分重建公式作用于一步前缀精化的直接结果。

### 推论 6.3：差分级数

对任意合法补全词 `w = z_1 ... z_L`，有：

```text
P_i^(L) - P_i^(0)
=
Σ_{r=1}^L
[
  1 / (n q_tot^(r))
]
Σ_j T_{ij}^(r)
```

### 证明

把命题 6.2 代入定理 5.2 的第一式即可。

### 定义 6.4：块相对转移量

对任意下标块 `A = <a_1,...,a_s>`，定义：

```text
P_A^(r) := Σ_{h=1}^s P_{a_h}^(r)
E_A^(r) := Σ_{h=1}^s E_{a_h}^(r)
```

若 `Δ_tot^(r) != 0`，定义：

```text
ρ_A^(r) := E_A^(r) / Δ_tot^(r)
```

### 定理 6.5：块相对转移级数

对任意块 `A`，有：

```text
P_A^(L) - P_A^(0)
=
Σ_{r=1}^L E_A^(r) / q_tot^(r)
```

若对每个 `r` 都有 `Δ_tot^(r) != 0`，则：

```text
P_A^(L) - P_A^(0)
=
Σ_{r=1}^L λ_r ρ_A^(r)
```

### 证明

把定理 5.2 对块 `A = <a_1,...,a_s>` 中各下标逐项求和即可。

### 定义 6.6：局部正概率层

若某一步 `r` 满足：

```text
P_i^(r-1) > 0
```

对全部 `i` 都成立，则称第 `r` 步处于局部正概率层，并定义该步的零和切空间为：

```text
T^(r-1)
:=
全部满足 Σ_i u_i = 0 的 n 维实词 <u_1,...,u_n>
```

### 命题 6.6.1：局部正概率层的开区间性质

若第 `r` 步处于局部正概率层，并且

```text
n >= 2
```

则对全部 `i`，有：

```text
0 < P_i^(r-1) < 1
```

### 证明

由定义 6.6，先有

```text
P_i^(r-1) > 0
```

对全部 `i` 成立。又由

```text
Σ_i P_i^(r-1) = 1
```

且 `n >= 2`，对每个固定的 `i` 都可取某个 `j != i`。由于

```text
P_j^(r-1) > 0
```

故

```text
Σ_{k != i} P_k^(r-1) > 0
```

从而

```text
P_i^(r-1) = 1 - Σ_{k != i} P_k^(r-1) < 1
```

命题成立。

### 定义 6.7：局部 Fisher 形式

在定义 6.6 的条件下，对任意属于 `T^(r-1)` 的向量

```text
u, v
```

定义：

```text
g_{P^(r-1)}(u,v)
:=
Σ_i u_i v_i / P_i^(r-1)
```

### 命题 6.8：局部 Fisher 形式的正定性

在定义 6.6 的条件下，`g_{P^(r-1)}` 是 `T^(r-1)` 上的对称正定双线性形式。

### 证明

双线性与对称性由定义 6.7 直接得到。对任意非零且属于 `T^(r-1)` 的向量

```text
u
```

有：

```text
g_{P^(r-1)}(u,u)
=
Σ_i u_i^2 / P_i^(r-1)
```

各项都非负，且因 `u != 0` 与 `P_i^(r-1) > 0`，至少一项严格为正，故

```text
g_{P^(r-1)}(u,u) > 0
```

### 命题 6.9：一步相对转移属于零和切空间

记：

```text
δP^(r) := P^(r) - P^(r-1)
```

则：

```text
δP^(r) 属于 T^(r-1)
```

### 证明

由命题 5.1，

```text
δP_i^(r) = P_i^(r) - P_i^(r-1)
```

且

```text
Σ_i P_i^(r) = Σ_i P_i^(r-1) = 1
```

故：

```text
Σ_i δP_i^(r) = 0
```

于是 `δP^(r)` 属于 `T^(r-1)`。

### 定义 6.10：一步局部协向量

在定义 6.6 的条件下，对任意属于 `T^(r-1)` 的向量

```text
v
```

定义：

```text
α^(r)(v)
:=
- Σ_i
[
  δP_i^(r) / P_i^(r-1)
]
v_i
```

### 命题 6.11：一步 Fisher 对偶式

在定义 6.6 的条件下，对任意属于 `T^(r-1)` 的向量

```text
v
```

有：

```text
g_{P^(r-1)}(-δP^(r), v) = α^(r)(v)
```

### 证明

由定义 6.7 与定义 6.10，

```text
g_{P^(r-1)}(-δP^(r), v)
=
Σ_i [(-δP_i^(r)) v_i / P_i^(r-1)]
=
- Σ_i [δP_i^(r) / P_i^(r-1)] v_i
=
α^(r)(v)
```

### 定义 6.12：一步局部自然梯度

在定义 6.6 的条件下，若存在属于 `T^(r-1)` 的向量 `X` 满足：

```text
g_{P^(r-1)}(X, v) = α^(r)(v)
```

对全部属于 `T^(r-1)` 的向量 `v` 都成立，则称 `X` 为 `α^(r)` 在 `P^(r-1)` 处的一步局部自然梯度。

### 命题 6.13：一步局部自然梯度的唯一表示

在定义 6.6 的条件下，定义 6.12 中的一步局部自然梯度存在且唯一，且：

```text
X = -δP^(r)
```

若再满足

```text
Δ_tot^(r) != 0
```

则：

```text
δP^(r) = λ_r ρ^(r)
```

### 证明

由命题 6.8，`g_{P^(r-1)}` 在 `T^(r-1)` 上正定，故这样的 `X` 至多唯一。由命题 6.11，`X := -δP^(r)` 已满足所要求的对偶关系，故唯一向量即为 `-δP^(r)`。

若再满足

```text
Δ_tot^(r) != 0
```

则由命题 5.1 的第二式，


### 命题 6.20.2：概率切投影的像、核与幂等性

对任意合法概率词 `P` 与任意 `n` 维实词 `x`，有：

```text
Σ_i [A_P(x)]_i = 0
```

若

```text
Σ_i x_i = 0
```

则：

```text
A_P(x) = x
```

又有：

```text
A_P(A_P(x)) = A_P(x)
```

并且：

```text
A_P(P) = 0
```

最后：

```text
A_P(x) = 0
```

当且仅当存在实数 `c` 使得

```text
x = c P
```

### 证明

由定义 6.20.1，

```text
Σ_i [A_P(x)]_i
=
Σ_i x_i - (Σ_i P_i)(Σ_j x_j)
```

又由 `P` 是合法概率词，

```text
Σ_i P_i = 1
```

故

```text
Σ_i [A_P(x)]_i = 0
```

若 `Σ_i x_i = 0`，则对每个 `i` 都有

```text
[A_P(x)]_i = x_i
```

故 `A_P(x) = x`。又由

```text
Σ_i P_i = 1
```

得

```text
[A_P(P)]_i = P_i - P_i Σ_j P_j = 0
```

故 `A_P(P) = 0`。再由前面已证

```text
Σ_i [A_P(x)]_i = 0
```

把 `A_P(x)` 代入“零和词被 `A_P` 固定”这一结论，得

```text
A_P(A_P(x)) = A_P(x)
```

最后，若 `A_P(x) = 0`，则对每个 `i` 都有

```text
x_i = P_i Σ_j x_j
```

记

```text
c := Σ_j x_j
```

即得 `x = cP`。反向由 `A_P(P)=0` 立得 `A_P(cP)=0`。

```text
a = 0
```

即可由全部有限差值唯一决定全部顶点值，因而唯一决定全部链段值。最后，在固定 `P_i^(0)` 后，由

```text
A_i^(m) = P_i^(m) - P_i^(0)
```

即得：

```text
P_i^(m) = P_i^(0) + A_i^(m)
```

故三组数据均唯一决定全部 `P_i^(m)`。

### 命题 7.8：四点平直恒等式

对任意

```text
0 <= a < b <= L-1
```

有：

```text
F_i(a+1,b+1) - F_i(a+1,b) - F_i(a,b+1) + F_i(a,b) = 0
```

### 证明

由命题 6.17，

```text
F_i(a,b) = P_i^(b) - P_i^(a)
```

逐项代入：

```text
F_i(a+1,b+1) - F_i(a+1,b) - F_i(a,b+1) + F_i(a,b)
```

即得

```text
[P_i^(b+1)-P_i^(a+1)]
- [P_i^(b)-P_i^(a+1)]
- [P_i^(b+1)-P_i^(a)]
+ [P_i^(b)-P_i^(a)]
= 0
```

### 定义 7.9：长度坐标三角词

对任意

```text
0 <= a < a+ell <= L
```

定义：

```text
L_i(a,ell) := F_i(a,a+ell)
```

### 命题 7.10：长度坐标重建公式

对任意合法指标，均有：

```text
L_i(a,ell) = Σ_{t=0}^{ell-1} L_i(a+t,1)
```

```text
L_i(a,ell) = L_i(0,a+ell) - L_i(0,a)
```

### 证明

第一式由命题 7.4 代入定义 7.9 即得。第二式由命题 7.6 代入定义 7.9 即得。

### 命题 7.11：长度坐标运输恒等式

对任意

```text
0 <= a < a+ell <= L-1
```

有：

```text
L_i(a+1,ell) - L_i(a,ell) = L_i(a+ell,1) - L_i(a,1)
```

### 证明

由定义 7.9，

```text
L_i(a+1,ell) = P_i^(a+ell+1) - P_i^(a+1)
```

```text
L_i(a,ell) = P_i^(a+ell) - P_i^(a)
```

两式相减，得

```text
L_i(a+1,ell) - L_i(a,ell)
=
[P_i^(a+ell+1)-P_i^(a+ell)]
- [P_i^(a+1)-P_i^(a)]
```

再由定义 7.9，

```text
P_i^(a+ell+1)-P_i^(a+ell) = L_i(a+ell,1)
```

```text
P_i^(a+1)-P_i^(a) = L_i(a,1)
```

代入即得。

### 命题 7.12：自由度与唯一决定

固定初始概率词

```text
P^(0)
```

后，信息转移级数三角形由全部一步边界词

```text
e^(1),...,e^(L)
```

唯一决定；并且对每个

```text
1 <= r <= L
```

都有：

```text
Σ_i e_i^(r) = 0
```

故每一步边界词至多只有 `n-1` 个独立实坐标。

### 证明

由命题 7.4，全部一步边界词唯一决定全部链段值，故唯一决定整个三角形。再由命题 7.2 的第三式作用于

```text
F(r-1,r)
```

得

```text
Σ_i e_i^(r) = 0
```

故在 `n` 个实坐标中，至多仅有 `n-1` 个独立实坐标。

### 定理 7.13：实概率层实现判据

固定一个初始概率词

```text
P^(0) = <P_1^(0),...,P_n^(0)>
```

与长度为 `L` 的有限词

```text
e^(1),...,e^(L)
```

其中每个

```text
e^(r) = <e_1^(r),...,e_n^(r)>
```

都是零和实词。对每个

```text
0 <= m <= L
```

定义：

```text
P_i^(m) := P_i^(0) + Σ_{r=1}^m e_i^(r)
```

则下列两条等价：

```text
1. 对全部 m=0,...,L，词 P^(m) 都是合法概率词。
2. 存在且仅存在一个实概率层信息转移级数三角形，使其一步边界词恰为 e^(1),...,e^(L)。
```

### 证明

先证 `1 => 2`。由条件 1，对全部 `m=0,...,L`，

```text
Σ_i P_i^(m) = 1
```

且各坐标非负。故全部 `P^(m)` 都是合法概率词。定义：

```text
F_i(a,b) := P_i^(b) - P_i^(a)
```

则由构造，

```text
F_i(r-1,r) = e_i^(r)
```

故得到一个以 `e^(1),...,e^(L)` 为一步边界词的信息转移级数三角形。其唯一性由命题 7.12 得到。

再证 `2 => 1`。若存在一个以 `e^(1),...,e^(L)` 为一步边界词的信息转移级数三角形，则由定义 7.3 与命题 7.4，

```text
F_i(0,m) = Σ_{r=1}^m e_i^(r)
```

再由定义 7.5 与命题 7.6，

```text
P_i^(m) - P_i^(0) = A_i^(m) = F_i(0,m)
```

故

```text
P_i^(m) = P_i^(0) + Σ_{r=1}^m e_i^(r)
```

故每个 `P^(m)` 都等于该三角形对应链中的概率词，因而都是合法概率词。

## 8. 信息转移级数三角形与局部度量

### 定义 8.1：一步边界切词

对每个

```text
1 <= r <= L
```

定义：

```text
e^(r) := F(r-1,r)
```

即：

```text
e_i^(r) := F_i(r-1,r) = P_i^(r) - P_i^(r-1)
```

### 定义 8.2：一步局部度量

在定义 6.6 的条件下，若 `h^(r)` 是 `T^(r-1)` 上的对称正定双线性形式，则称 `h^(r)` 为第 `r` 步的一步局部度量。

### 定义 8.3：一步局部度量协向量

在定义 8.2 的条件下，对任意属于 `T^(r-1)` 的向量

```text
v
```

定义：

```text
α_h^(r)(v) := - h^(r)(e^(r), v)
```

### 命题 8.4：任意一步局部度量下的自然梯度表示

在定义 8.2 的条件下，存在且仅存在属于 `T^(r-1)` 的向量

```text
X_h^(r)
```

满足：

```text
h^(r)(X_h^(r), v) = α_h^(r)(v)
```

对全部属于 `T^(r-1)` 的向量 `v` 都成立；并且：

```text
X_h^(r) = - e^(r)
```

### 证明

由定义 8.2，`h^(r)` 在 `T^(r-1)` 上正定，故满足所述对偶关系的向量至多唯一。再由定义 8.3，

```text
h^(r)(-e^(r), v) = - h^(r)(e^(r), v) = α_h^(r)(v)
```

故 `X_h^(r) = -e^(r)` 即为所求唯一向量。

### 推论 8.4.1：一步局部自然梯度向量的度量不变性

若 `h^(r), k^(r)` 都是第 `r` 步的一步局部度量，则：

```text
X_h^(r) = X_k^(r) = - e^(r)
```

### 证明

由命题 8.4，

```text
X_h^(r) = - e^(r)
X_k^(r) = - e^(r)
```

故结论成立。

### 推论 8.4.2：一步局部自然梯度的投影表示

在定义 8.2 的条件下，有：

```text
X_h^(r)
=
- [1 / q_tot^(r)] A_{P^(r-1)}(Δ^(r))
```

### 证明

由命题 6.20.3，

```text
E^(r) = A_{P^(r-1)}(Δ^(r))
```

再由命题 5.1 的第一式与定义 8.1，

```text
e^(r) = δP^(r) = E^(r) / q_tot^(r)
```

最后用命题 8.4 的

```text
X_h^(r) = - e^(r)
```

即得。

### 推论 8.5：任意局部度量链表示

若对全部

```text
r = 1,...,L
```

都给定一步局部度量 `h^(r)`，则对任意

```text
0 <= a < b <= L
```

有：

```text
F(a,b) = - Σ_{r=a+1}^b X_h^(r)
```

特别地：

```text
P^(L) - P^(0) = - Σ_{r=1}^L X_h^(r)
```

### 证明

由命题 8.4，

```text
X_h^(r) = - e^(r)
```

再由命题 7.4，

```text
F(a,b) = Σ_{r=a+1}^b e^(r)
```

代入即得。

### 定义 8.6：Euclid 度量

在定义 6.6 的条件下，定义第 `r` 步的 Euclid 度量为：

```text
h_E^(r)(u,v) := Σ_i u_i v_i
```

### 命题 8.7：Euclid 自然梯度表示

在定义 6.6 的条件下，对任意属于 `T^(r-1)` 的向量 `v`，有：

```text
α_E^(r)(v) = - Σ_i e_i^(r) v_i
```

并且对应的一步局部自然梯度为：

```text
X_E^(r) = - e^(r)
```

### 证明

由定义 8.3 与定义 8.6，

```text
α_E^(r)(v) = - h_E^(r)(e^(r),v) = - Σ_i e_i^(r) v_i
```

再由命题 8.4 即得。

### 定义 8.8：Fisher 度量

在定义 6.6 的条件下，定义第 `r` 步的 Fisher 度量为：

```text
h_F^(r) := g_{P^(r-1)}
```

即：

```text
h_F^(r)(u,v) := Σ_i u_i v_i / P_i^(r-1)
```

### 命题 8.9：Fisher 自然梯度表示

在定义 6.6 的条件下，对任意属于 `T^(r-1)` 的向量 `v`，有：

```text
α_F^(r)(v)
=
- Σ_i
[
  e_i^(r) / P_i^(r-1)
]
v_i
```

并且对应的一步局部自然梯度为：

```text
X_F^(r) = - e^(r)
```

### 证明

由定义 8.3 与定义 8.8，

```text
α_F^(r)(v)
=
- h_F^(r)(e^(r),v)
=
- Σ_i
[
  e_i^(r) v_i / P_i^(r-1)
]
```

再由命题 8.4 即得。

### 定义 8.10：Hellinger 度量

在定义 6.6 的条件下，定义第 `r` 步的 Hellinger 度量为：

```text
h_H^(r)(u,v)
:=
(1/4) Σ_i u_i v_i / P_i^(r-1)
```

即：

```text
h_H^(r) = (1/4) h_F^(r)
```

### 命题 8.11：Hellinger 自然梯度表示

在定义 6.6 的条件下，对任意属于 `T^(r-1)` 的向量 `v`，有：

```text
α_H^(r)(v)
=
- (1/4) Σ_i
[
  e_i^(r) / P_i^(r-1)
]
v_i
```

并且对应的一步局部自然梯度为：

```text
X_H^(r) = - e^(r)
```

### 证明

由定义 8.3 与定义 8.10，

```text
α_H^(r)(v)
=
- h_H^(r)(e^(r),v)
=
- (1/4) Σ_i
[
  e_i^(r) v_i / P_i^(r-1)
]
```

再由命题 8.4 即得。

### 命题 8.12：Fisher 与 Hellinger 的局部长度关系

在定义 6.6 的条件下，对每个

```text
1 <= r <= L
```

定义：

```text
N_F^(r) := h_F^(r)(e^(r), e^(r))
```

```text
N_H^(r) := h_H^(r)(e^(r), e^(r))
```

则：

```text
N_H^(r) = (1/4) N_F^(r)
```

并且：

```text
N_F^(r)
+=
[1 / (q_tot^(r))^2]
Σ_i
[
  (E_i^(r))^2 / P_i^(r-1)
]
```

```text
N_H^(r)
+=
[1 / (4 (q_tot^(r))^2)]
Σ_i
[
  (E_i^(r))^2 / P_i^(r-1)
]
```

### 证明

由定义 8.10 与 `N_F^(r), N_H^(r)` 的定义，

```text
N_H^(r)
=
h_H^(r)(e^(r), e^(r))
=
(1/4) h_F^(r)(e^(r), e^(r))
=
(1/4) N_F^(r)
```

再由命题 5.1 的第一式，

```text
e_i^(r) = P_i^(r) - P_i^(r-1) = E_i^(r) / q_tot^(r)
```

代入 `N_F^(r)` 与 `N_H^(r)` 的定义即得余下两式。

### 定义 8.13：有限步平方根边界

对任意两个合法概率词

```text
P = <P_1,...,P_n>
Q = <Q_1,...,Q_n>
```

定义：

```text
B_sqrt(P,Q) := Σ_i (sqrt(Q_i) - sqrt(P_i))^2
```

### 命题 8.14：有限步平方根边界与局部 Hellinger 二次型一般不相等

存在合法概率词 `P, Q`，其中各 `P_i` 都严格为正，使：

```text
B_sqrt(P,Q)
!=
(1/4) Σ_i (Q_i - P_i)^2 / P_i
```

### 证明

取

```text
P = <1/2, 1/2>
Q = <1, 0>
```

则：

```text
B_sqrt(P,Q)
=
(1 - 1/sqrt(2))^2 + (0 - 1/sqrt(2))^2
=
2 - sqrt(2)
```

并且：

```text
(1/4) Σ_i (Q_i - P_i)^2 / P_i
=
(1/4)
[
  (1/2)^2 / (1/2) + (-1/2)^2 / (1/2)
]
=
1/4
```

而

```text
2 - sqrt(2) != 1/4
```

故命题成立。

## 9. 复载体形式

### 定义 9.1：一步复载体词

对第 `r` 步前缀精化，记其全部相关路径-轴复支撑项按固定次序写成有限词：

```text
Z^(r) := <z_1^(r),...,z_{t_r}^(r)>
```

其中每一项都满足：

```text
Read_Σ(z_h^(r)) = |z_h^(r)|^2
```

且这些实读出经量子化、轴层合成与有限归一后，诱导该步的 `E_i^(r)`；若再满足 `Δ_tot^(r) != 0`，则又诱导 `ρ_i^(r)`。

### 定义 9.2：单位模等价

对两个同长度复词

```text
Z = <z_1,...,z_t>
Z' = <z'_1,...,z'_t>
```

记：

```text
Z ~_U Z'
```

当且仅当存在单位模词

```text
U = <u_1,...,u_t>
```

使：

```text
|u_h| = 1
z'_h = u_h z_h
```

对全部 `h` 都成立。

### 命题 9.3：一步实读出不变性

若

```text
Z^(r) ~_U Z'^(r)
```

则：

```text
Read_Σ(z_h^(r)) = Read_Σ(z'_h^(r))
```

对全部 `h` 成立。

### 证明

由

```text
Read_Σ(u z) = |u z|^2 = |u|^2 |z|^2 = |z|^2 = Read_Σ(z)
```

逐项得到。

### 推论 9.4：一步概率读出不变性

若

```text
Z^(r) ~_U Z'^(r)
```

则两者诱导的

```text
m_{j,k}^(r), Q_{i,k}^(r), Q_i^(r), E_i^(r)
```

若再满足

```text
Δ_tot^(r) != 0
```

则又有

```text
ρ_i^(r)
```

全部相同。

### 证明

由命题 9.3，各实读出逐项相同。量子化、轴层合成与有限归一都只依赖实读出、路径吸纳、方向记录与轴符号，故上列各量相同；若再满足 `Δ_tot^(r) != 0`，则 `ρ_i^(r)` 也相同。

### 定义 9.5：复载体有序词

对合法补全词 `w = z_1 ... z_L`，定义其复载体有序词为：

```text
𝒵(w) := <Z^(1),...,Z^(L)>
```

### 定义 9.6：复载体等价

对两个同长度复载体有序词

```text
𝒵(w) = <Z^(1),...,Z^(L)>
𝒵'(w) = <Z'^(1),...,Z'^(L)>
```

记：

```text
𝒵(w) ~_U 𝒵'(w)
```

当且仅当对全部 `r` 都有：

```text
Z^(r) ~_U Z'^(r)
```

### 定理 9.7：复载体有序词决定实相对转移级数

复载体有序词在 `~_U` 等价下，唯一决定定理 5.2 的实相对转移级数。

### 证明

由推论 9.4，每一步复载体词在 `~_U` 下唯一决定该步的

```text
E_i^(r)
```

再由定理 5.2，

```text
P_i^(L) - P_i^(0)
=
Σ_{r=1}^L E_i^(r) / q_tot^(r)
```

故唯一决定整个实相对转移级数。若再满足

```text
Δ_tot^(r) != 0
```

则该步又唯一决定 `ρ_i^(r), λ_r`。

## 10. 复数形式的最小本体

### 定理 10.1：复数形式首先是有序词，不是单个复和值

在当前文档基底下，复数形式的相对转移对象首先是复载体有序词

```text
𝒵(w)
```

而不是某个先验给定的单个复和值。

### 证明

前读出载体的必然性只要求在每一步正贡献精化上保留独立于终点实读出与方向层之外的复载体。现有读出算子固定为

```text
Read_Σ(z) = |z|^2
```

这是逐项模平方读出，不是对整条前缀链的线性和值读出。故现有基底只严格固定了各步复载体项及其有序排列，未固定一个把整条链直接压成单个复和值的附加算子。于是复数形式首先只能写成复载体有序词。

### 推论 10.2：一般不能压成普通单变量函数级数

本文中的相对转移级数一般不能先验压成普通单变量函数级数。

### 证明

该级数的每一步项依赖于前缀背景 `w^(r-1)`，并由该步复载体词 `Z^(r)` 经过非线性读出链诱导。故项值不只依赖自然数阶数，也不只依赖单个标量变量。于是一般不能先验压成普通单变量函数级数。

## 11. 总结定理

在 `充足理由单子论.md` 与 `由果溯因概率论.md` 的既有对象语言下，可直接得到：

1. 正贡献理由沿完整展开链单向进入结果层。
2. 在未满足闭合退化条件时，前读出转移差异必须保留独立载体。
3. 若前读出载体仍以数值载体表达，则该载体不能只取实数，必须取复数。
4. 任意合法补全词都诱导一条有限前缀精化链。
5. 只要某前缀总支撑 `q_tot^(r)` 为正，该前缀概率词就唯一可归一化；该判据对含隐理由的支持词同样成立。并且对任意 `i` 都有：

```text
0 <= P_i^(r) <= 1
```

若再对某个 `i` 满足

```text
P_i^(r) > 0
```

并且存在某个 `j != i` 使得

```text
P_j^(r) > 0
```

则有：

```text
0 < P_i^(r) < 1
```
6. 全部实概率变化精确写成有限有序转移级数：

```text
P_i^(L) - P_i^(0)
=
Σ_{r=1}^L E_i^(r) / q_tot^(r)
```

7. 每一步有效信息都满足零和式：

```text
Σ_i E_i^(r) = 0
```
