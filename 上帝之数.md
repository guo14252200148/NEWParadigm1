# 上帝之数

## 1. 目的

本文在以下三份文档的对象语言内工作：

```text
充足理由单子论.md
由果溯因概率论.md
由果溯因复载体相对转移级数.md
```

目标是对如下对象作正式定义、严格推导并给出闭式公式：

```text
边长为正整数 n, n >= 2 的六面异色 n 阶魔方
在本文模型中的上帝之数
```

本文要证明的不是一个先验命名，而是一条等式：

```text
恢复商
= 局部自然梯度有效量子个数
= HTM 合法状态图中的最大最短路长度
```

其中恢复商定义为：

```text
还原态总实信息与最小实读出之差
--------------------------------
局部自然梯度有效信息实读出均值
```

本文将证明：

```text
God(2) = 11
God(3) = 20
God(n) = 3n^2 - 4,   n >= 4 且 n 为偶数
God(n) = 3n^2 - 7,   n >= 3 且 n 为奇数
```

全文只使用有限对象、有限层数、有限词与有限和式；不引入集合论外壳，不引入连续概率测度，也不调用外部魔方群论定理。

## 3. 魔方对象

### 定义 3.1：合法魔方

固定正整数 `n >= 2`。`n` 阶正方体魔方指：

```text
1. 外形为 n × n × n 的正方体；
2. 六个外表面初始颜色两两不同；
3. 合法状态只由还原态经合法操作得到。
```

### 定义 3.2：合法操作

一次合法操作由三元组

```text
(F,d,t)
```

给出，其中：

```text
F ∈ {U,D,L,R,F,B}
1 <= d <= n-1
t ∈ {1,2,3}
```

其含义为：

```text
以面 F 为外锚，
把厚度为 d 的外连层整体旋转，
旋转角类分别对应 90°、180°、270°。
```

因此：

```text
1. 允许单层外转；
2. 允许宽层转；
3. 不允许只转内部层；
4. 360° 的整数倍不计为有效一步。
```

### 命题 3.2.1：HTM 的合法基本角类只有三类

在本文的计步下，一次合法操作的角类只有：

```text
90°
180°
270°
```

### 证明

HTM 把任一非平凡转动只按角类区分。对一个固定的外连层旋转，`360°` 为恒等，不计入一步；其余只剩 `90°`、`180°`、`270°` 三类。

### 推论 3.2.1.1：HTM 与本文一步合法操作给出同一邻接关系

两个合法状态在本文状态图中相邻，当且仅当其中一个可由另一个施行一次 HTM 合法操作得到。

### 定义 3.2.2：合法状态图

记 `Γ_n` 为 `n` 阶魔方的合法状态图：

```text
顶点 : 全部合法状态
边   : 一次 HTM 合法操作
```

### 定义 3.2.3：最短合法操作距离

对合法状态 `A,B`，记：

```text
dist_n(A,B)
```

为图 `Γ_n` 中从 `A` 到 `B` 的最短边长。

### 定义 3.2.4：群论意义下的上帝之数

记还原态为 `C_0(n)`。定义：

```text
God(n) := max_C dist_n(C, C_0(n))
```

其中最大值在全部合法状态 `C` 上取。

### 定义 3.3：三维拓扑类型

把表面位置写成三维整数坐标 `(x,y,z)`。定义：

```text
t(x,y,z)
:=
落在边界集 {1,n} 中的坐标个数
```

则：

```text
t=1 : 面内
t=2 : 棱上
t=3 : 角顶
```

### 命题 3.4：拓扑类型在合法操作下保持

### 证明

一次合法操作只是某个外连层的刚性旋转。刚性旋转只改变表面位置的排列，不改变一个位置有多少个坐标落在边界集 `{1,n}` 中，因此 `t` 保持不变。

### 定义 3.5：层号

对任一表面位置，取其所在外表面的二维格点坐标 `(a,b)`，定义：

```text
ℓ := 1 + min(a-1, b-1, n-a, n-b)
```

称 `ℓ` 为该位置的层号。

### 命题 3.6：层号在合法操作下保持

### 证明

合法操作在外表面上的作用只是二维方阵格点的刚性旋转。刚性旋转保持格点到边界的最小距离，故层号 `ℓ` 保持不变。

### 推论 3.7：同层拓扑等效群体存在

对固定的 `(t,ℓ)`，全部满足该对指标的表面位置构成一个拓扑等效群体。合法操作只在同一群体内部重排，不会把其送到别的 `(t,ℓ)` 群体。

## 4. 从基底严格推出的条件

本节只做一件事：把魔方几何压入前三份文档的对象语言，从而推出本文后面使用的条件，而不是把这些条件当作未证前提。

### 命题 4.1：名字盲性

若两个贡献对象在路径骨架、层位骨架与逐位权重上完全一致，则它们在本文对象语言中属于同一贡献类。

### 证明

由 `充足理由单子论.md` 中的 `Par_E`、`EqW_E` 与 `Com_E` 定义，只要两对象满足：

```text
路径并行
逐位权重相等
```

便有：

```text
Com_E(a,b;τ)=1
```

再由 `K2`，同一性来自有效理由结构，不来自名字，故结论成立。

### 推论 4.2：同结构同贡献

若两个色块位置具有相同的 `(t,ℓ)`，并且在当前读出下诱导同一类路径-轴骨架，则它们的贡献完全一致。

### 命题 4.3：六色对称

在本文模型中，颜色标签只起占位区分作用，不进入贡献的结构判据。

### 证明

由命题 4.1，贡献只看结构，不看名字。对魔方而言，六种面色是同一几何载体上的六个标签；只要保持结构骨架不变，改换颜色名字不改变 `Par_E`、`EqW_E`、`Com_E` 的值，故颜色标签不进入贡献判据。

### 推论 4.4：等效群体必为六的整数倍，且六色占位相等

每个拓扑等效群体都可写成：

```text
几何位置轨道 × 六色标签
```

故其大小必为 `6` 的整数倍，并且六色在群体中的占位数相等。

### 命题 4.5：信息转移实读出粒度只与层号有关

对固定层号 `ℓ`，存在一个层量子 `ε_ℓ`，使该层全部等效群体的单步实读出都写成 `ε_ℓ` 的整数倍。

### 证明

由 `由果溯因概率论.md`，任一有限读出词都存在最大公共量子单位 `ε_Σ`。对固定层号 `ℓ`，该层只含有限多个等效群体；由命题 4.2 与命题 4.3，这些群体在结构上通约，因此它们的实读出量子可压成同一公共单位。记此公共单位为 `ε_ℓ`。

### 说明 4.6：本文后续使用的条件

至此，后文所需的五个条件已由基底与三维几何推出：

```text

```

## 5. 最小实读出

### 定义 5.1：对向面实读出通道

把六个外表面按三对对向面配对：

```text
(U,D), (L,R), (F,B)
```

对每一对对向面上的对应格点，定义一个实读出通道。

### 命题 5.2：总实通道数

```text
N_all(n) = 3n^2
```

### 证明

每对对向面给出 `n^2` 个对应格点，共三对，故总数为 `3n^2`。

### 命题 5.3：最小保底通道数

```text
N_min(n) = 0,   n 为偶数
N_min(n) = 3,   n 为奇数
```

### 证明

若 `n` 为偶数，则每对对向面都没有中心单点，全部通道都可在同轴反向有效信息相消后转入复域，因此无保底通道。

若 `n` 为奇数，则每对对向面恰有一个中心对应格点。三对对向面共留下 `3` 个中心通道，它们没有同层反向对应点可与之配对，因此不能被完全消去。

### 定义 5.4：最小实读出

还原态总实信息归一为 `1`。定义：

```text
I_min(n) := N_min(n) / N_all(n)
```

### 定理 5.5：最小实读出公式

```text
I_min(n) =
0,      n 为偶数
1/n^2,  n 为奇数
```

并且最大可恢复亏额为：

```text
Δ(n) := 1 - I_min(n)
```

即：

```text
Δ(n) =
1,            n 为偶数
1 - 1/n^2,    n 为奇数
```

### 证明

由定义 5.4 与命题 5.2、命题 5.3，直接代入即可。

### 定义 5.6：活动通道数

定义：

```text
A(n) := N_all(n) - N_min(n)
```

于是：

```text
A(n) =
3n^2,       n 为偶数
3n^2 - 3,   n 为奇数
```

### 命题 5.7：外层递推的通道增量

对一切 `n >= 4`，有：

```text
A(n) = A(n-2) + 12(n-1)
```

### 证明

从 `(n-2)` 阶扩到 `n` 阶时，每对对向面新增的边框格点数为：

```text
n^2 - (n-2)^2 = 4(n-1)
```

共有三对对向面，因此新增活动通道数为 `12(n-1)`。

## 6. 恢复商与有效量子

本节的做法不是再添加外部假设，而是直接把一步 HTM 合法操作在量子化读出层上的商像构造成一个有限商空间，并在这个商空间里定义有效量子基。这样后面的距离、缺额与恢复商都在同一构造中给出，不再额外引入兼容条件。

### 定义 6.1：空间刚性模

对 `n >= 3`，在活动通道词上存在三种不改变真实相对结构的全局刚性重标记：

```text
x 轴参考模
y 轴参考模
z 轴参考模
```

它们改变的是三维参考框架，不改变真实相对读出结构。

### 定义 6.1.1：三组主轴通道族

把全部对向面实读出通道按其法向分成三组：

```text
X 族 : 由 (L,R) 对向面诱导的通道
Y 族 : 由 (D,U) 对向面诱导的通道
Z 族 : 由 (B,F) 对向面诱导的通道
```

任一合法操作都只绕其中某一条主轴发生。

### 命题 6.2：空间刚性模数

```text
R(2) = 0
R(n) = 3,   n >= 3
```

### 证明

`2` 阶只有角顶壳，没有中心锚点。每个可见位置都同时接触三条主轴；在读出层上，`X/Y/Z` 三族不能被内部结构独立地区分，任何整体参考改写都只表现为同一个全局重标记，而不裂成三条彼此独立的刚性模，故 `R(2)=0`。

当 `n >= 3` 时，至少出现一类不是纯角顶的位置：`3` 阶起出现棱上位置，`4` 阶起再出现面内位置。此时三组主轴通道族 `X/Y/Z` 已可由其所在对向面族独立区分；绕 `x/y/z` 三轴的整体参考改写分别只在对应的主轴标记上起作用，三者互不相同，也不可能由另外两者合成，故刚性模恰为三维，即 `R(n)=3`。

### 命题 6.3：概率径向模恰为一维

进入有效信息层时，必然还要再去掉一维纯比例缩放方向。

### 证明

由基底中的

```text
E = A_P(Δ)
A_P(x)=0  <=>  x = cP
```

可知 `A_P` 的核恰为纯比例方向 `span{P}``，故必须去掉且仅去掉一维径向模。

### 定义 6.3.1：一步局部自然梯度结构词

对任意一步 HTM 合法操作所诱导的概率变化，定义其一步局部自然梯度结构词为：

```text
X^(r)
:=
- e^(r)
=
- [1 / q_tot^(r)] A_{P^(r-1)}(Δ^(r))
```

### 命题 6.3.2：一步局部自然梯度结构绝对不变性

对任意一步局部度量 `h^(r)`，其一步局部自然梯度向量都满足：

```text
X_h^(r) = - e^(r)
       = - [1 / q_tot^(r)] A_{P^(r-1)}(Δ^(r))
```

因此，一步信息转移的局部自然梯度结构只由：

```text
1. 一步边界词 e^(r)
2. 概率切投影 A_{P^(r-1)}
3. 一步总支持 q_tot^(r)
```

决定，而不因 Euclid / Fisher / Hellinger 或任意别的一步局部度量的选取而改变。

### 证明

这正是 `由果溯因复载体相对转移级数.md` 中命题 8.4、推论 8.4.1 与推论 8.4.2 的直接合并：

```text
X_h^(r) = - e^(r)
X_h^(r) = - [1 / q_tot^(r)] A_{P^(r-1)}(Δ^(r))
```

故一步局部自然梯度的结构绝对不变。

### 推论 6.3.3：一步合法操作的量子方向由结构唯一决定

若两次一步 HTM 合法操作在商空间中诱导同一个

```text
A_{P^(r-1)}(Δ^(r))
```

的方向，则它们给出的局部自然梯度结构方向相同；若该投影方向不同，则局部自然梯度结构方向不同。

### 证明

由命题 6.3.2，

```text
X_h^(r) = - [1 / q_tot^(r)] A_{P^(r-1)}(Δ^(r))
```

比例因子 `1 / q_tot^(r)` 只改变大小，不改变方向；负号只作整体反向，故方向完全由投影词决定。

### 推论 6.3.4：一步局部自然梯度的非零判据

对任意一步 HTM 合法操作，以下三件事等价：

```text
1. X^(r) != 0
2. A_{P^(r-1)}(Δ^(r)) != 0
3. Δ^(r) 不是 P^(r-1) 的纯比例词
```

### 证明

由定义 6.3.1，

```text
X^(r) = - [1 / q_tot^(r)] A_{P^(r-1)}(Δ^(r))
```

由于一步总支持 `q_tot^(r)` 为正，故：

```text
X^(r) != 0
<=>
A_{P^(r-1)}(Δ^(r)) != 0
```

再由 `由果溯因概率论.md` 与 `由果溯因复载体相对转移级数.md` 中的核判据

```text
A_P(x)=0  <=>  x = cP
```

立得：

```text
A_{P^(r-1)}(Δ^(r)) != 0
<=>
Δ^(r) 不是 P^(r-1) 的纯比例词
```

三者遂等价。

### 定义 6.4：有效量子个数

定义：

```text
M(n) := A(n) - R(n) - 1
```

称 `M(n)` 为 `n` 阶魔方在本文模型中的局部自然梯度有效量子个数。

### 定义 6.4.1：一步商像集与有效商空间

记全部一步 HTM 合法操作所诱导的活动通道增量词之集合为：

```text
Step_n := {Δ(g) : g 为一步 HTM 合法操作}
```

按空间刚性模与概率径向模取商，得有限实词空间：

```text
V_n
:=
Span(Step_n) / <三维刚性模, 概率径向模>
```

并且，由活动通道的定义，每一个活动通道都能被某个一步 HTM 合法操作在该通道上产生非零增量；故 `Span(Step_n)` 正好张成全部活动通道词空间。

### 定义 6.4.2：原始一步量子类

对两个非零一步商像 `v,w ∈ V_n`，定义：

```text
v ~ w
<=>
存在 λ > 0 使 v = λw 或 v = -λw
```

称 `~` 的等价类为原始一步量子类。

### 命题 6.4.3：原始一步量子类有限

### 证明

一步 HTM 合法操作总数有限，故 `Step_n` 有限，`V_n` 中出现的非零一步商像也有限，故原始一步量子类有限。

### 定义 6.4.4：适配有效量子基

从全部非零原始一步量子类中，每类取一个代表元，并规范为该类中“实读出最小的正向一步商像”，按任意固定次序写成：

```text
B_n = <β_1,...,β_s>
```

若这些代表元张成 `V_n` 且彼此线性无关，则称 `B_n` 为 `n` 阶魔方的适配有效量子基。

### 定义 6.4.4.1：原始支撑域

对任意一步 HTM 合法操作 `g`，其三维几何骨架唯一写成：

```text
g = (a,ℓ,σ)
```

其中 `a ∈ {x,y,z}` 是旋转主轴，`ℓ` 是被旋转的外连层号，`σ` 是旋转方向与角度型。定义 `g` 的原始支撑域 `S(a,ℓ)` 为：全部会被该三维层板刚性带动、并在活动通道词上产生非零增量的表面位置轨道之并。

该定义只依赖主轴与层号，不依赖颜色名字、位置名字与具体记号。

### 引理 6.4.4.2：一步合法操作的原始支撑域唯一

任意一步 HTM 合法操作都恰对应一个唯一的原始支撑域 `S(a,ℓ)`。

### 证明

HTM 的一步合法操作只能是“绕一条主轴，对一个外连层作一次非平凡刚性旋转”。主轴 `a` 一经给定，被带动的三维层板便唯一确定；层号 `ℓ` 一经给定，该层板在立方体中的位置也唯一确定；方向与角度型 `σ` 只改变该层板内的循环方式，不改变被带动的层板本身。

故任意一步 HTM 合法操作只可能落在一个唯一的 `S(a,ℓ)` 中。

### 引理 6.4.4.3：不同原始支撑域有不同的三维前沿轨道

若 `(a,ℓ) != (a',ℓ')`，则存在一个表面位置轨道 `O(a,ℓ)`，满足：

```text
O(a,ℓ) ⊂ S(a,ℓ)
O(a,ℓ) ⊄ S(a',ℓ')
```

亦即，不同原始支撑域至少在一个三维前沿轨道上必然可区分。

### 证明

魔方表面位置按拓扑类型只有 `面内 / 棱上 / 角顶` 三类，且层号在合法操作下保持。对固定的 `(a,ℓ)`，被该层板整体带动的位置形成一族同层轨道；其中总能取到一个“前沿轨道” `O(a,ℓ)`，它完整地附着在该层板的外边界上：

```text
外层时取该面的整面轨道；
内嵌外连层时取该层在外表面的边框轨道。
```

这个前沿轨道由主轴 `a` 与层号 `ℓ` 唯一标定。若改换层号，则前沿轨道所在的层距离改变；若改换主轴，则前沿轨道所在的法向族改变。二者都不可能把 `O(a,ℓ)` 作为自身的完整前沿轨道。

故不同原始支撑域必可由某个三维前沿轨道区分。

### 引理 6.4.4.4：跨原始支撑域的一步商像不能线性抵消

设 `v_1,...,v_m ∈ V_n` 是来自两两不同原始支撑域的非零一步商像。若

```text
c_1 v_1 + ... + c_m v_m = 0
```

则必有

```text
c_1 = ... = c_m = 0
```

### 证明

反设存在非平凡线性关系。对每个 `v_j`，取其原始支撑域 `S(a_j,ℓ_j)`。由引理 6.4.4.3，可为其中任一支撑域选取一个只属于它自己的三维前沿轨道 `O(a_j,ℓ_j)`。将上式限制到该前沿轨道上观察：

```text
只有来自 S(a_j,ℓ_j) 的一步商像
在 O(a_j,ℓ_j) 上留下非零变化。
```

其余来自别的原始支撑域的一步商像，在该前沿轨道上的限制全为零。于是该轨道上的系数和只能等于 `c_j v_j` 的限制，不可能被其他项抵消。由于 `v_j` 非零，故必有 `c_j = 0`。

对每个 `j` 重复此论证，得到全部系数都为零，矛盾。故不同原始支撑域的非零一步商像必线性独立。

### 命题 6.4.5：适配有效量子基存在且基数等于 `M(n)`

存在适配有效量子基 `B_n`，并且：

```text
|B_n| = M(n)
```

### 证明

由定义 6.4.1，`Span(Step_n)` 张成全部活动通道词空间，故：

```text
dim(V_n) = A(n) - R(n) - 1 = M(n)
```

又因为 `V_n` 由一步 HTM 合法操作的商像张成，所以全部非零原始一步量子类的代表元必张成 `V_n`。

另一方面，由引理 6.4.4.2，每个非零一步商像都带有唯一的原始支撑域标记；由引理 6.4.4.4，来自不同原始支撑域的非零一步商像在线性上彼此独立。故只需在每个原始一步量子类中取一个代表元，就已得到一组线性无关的向量。

因此可从每个原始一步量子类中取一个代表元，得到一组既张成又线性无关的基 `B_n`。

又由定义 6.4，

```text
M(n) = A(n) - R(n) - 1 = dim(V_n)
```

故：

```text
|B_n| = dim(V_n) = M(n)
```

### 命题 6.4.6：基向量的一步可实现性

对每个 `β_h`，至少存在一个一步 HTM 合法操作，使其在商空间中的像恰为 `β_h` 或 `-β_h`。

### 证明

由定义 6.4.4，`β_h` 本来就是从某个原始一步量子类中选出的代表元，而每个原始一步量子类都来自某个非零一步商像，故至少存在一步 HTM 合法操作，使其商像与 `β_h` 同向或反向共线。

### 命题 6.4.7：单步商像的单基性

对任意一步 HTM 合法操作 `g`，其在商空间中的像不可能写成两个不同基向量的非平凡和。

### 证明

由定义 6.4.2 与定义 6.4.4，一步商像总属于某个唯一的原始一步量子类，因此在基 `B_n` 下必落在某个唯一代表元 `β_h` 的射线上，只可能等于某个正实数倍的 `β_h` 或 `-β_h`，而不可能写成两个不同基向量的非平凡和。

### 定义 6.5：共同量子读出

对固定 `n`，考察全部一步 HTM 合法操作在实读出层上诱导的非零缺额词；由 `由果溯因概率论.md` 的最小损耗单位存在定理，这个有限非零读出词存在最大公共单位。记该单位为：

```text
g_bar(n)
```

称其为 `n` 阶魔方的一步共同量子读出。

### 命题 6.5.1：全部基量子的实读出都等于 `g_bar(n)`

### 证明

由定义 6.5，任一步 HTM 合法操作的非零实缺额都是 `g_bar(n)` 的整数倍。

再由命题 6.4.7，一步商像在适配有效量子基下只落在一个基方向上。故每个基向量 `β_h` 的读出也必是 `g_bar(n)` 的正整数倍。

由第 4 节的名字盲性、六色对称与层剥离结构，同一原始一步量子类只由“主轴、层号、方向”决定，而不再依赖颜色名字或位置名字。故全部原始一步量子类彼此全局通约，并共享同一个最小正读出单位。

若某个 `β_h` 的读出严格大于 `g_bar(n)`，设为 `m g_bar(n)` 且 `m >= 2`，则 `β_h` 所在原始一步量子类的最小正读出还可以进一步压缩到 `g_bar(n)`，这与定义 6.4.4 中“`β_h` 是该类实读出最小的正向一步商像”矛盾。

故每个 `β_h` 的实读出恰等于 `g_bar(n)`。

### 说明 6.5.2：与命题 4.5 的关系

## 8. 外层递推

### 定理 8.1：有效量子递推

对一切 `n >= 4`，有：

```text
M(n) = M(n-2) + 12(n-1)
```

### 证明

由命题 5.7，

```text
A(n)=A(n-2)+12(n-1)
```

对 `n` 与 `n-2` 同奇偶，空间刚性模数 `R(n)` 与 `R(n-2)` 都等于 `3`。概率径向模数始终等于 `1`。因此：

```text
M(n)
= A(n)-3-1
= [A(n-2)+12(n-1)]-3-1
= M(n-2)+12(n-1)
```

### 定理 8.2：有效量子闭式公式

```text
M(2) = 11
M(3) = 20
M(n) = 3n^2 - 4,   n >= 4 且 n 为偶数
M(n) = 3n^2 - 7,   n >= 3 且 n 为奇数
```

### 证明

偶数列由命题 7.1 与定理 8.1 递推：

```text
M(2m)
= 11 + Σ_{r=2}^m 12(2r-1)
= 11 + 12(m^2-1)
= 12m^2 - 1
= 3(2m)^2 - 4
```

奇数列由命题 7.2 与定理 8.1 递推：

```text
M(2m+1)
= 20 + Σ_{r=2}^m 12(2r)
= 20 + 12(m^2+m-2)
= 12m^2 + 12m - 4
= 3(2m+1)^2 - 7
```

证毕。

## 9. 恢复商与上帝之数

### 定义 9.1：缺额词

对任意合法状态 `C`，相对于还原态定义其缺额词：

```text
u(C) = <u_1(C),...,u_{M(n)}(C)>
```

其中：

```text
u_h(C) = 1  当且仅当第 h 个有效量子已从实层转入不可直接读出的复层
u_h(C) = 0  其余情形
```

这里的第 `h` 个有效量子，固定指向定义 6.4.1 中基 `B_n` 的第 `h` 个基向量 `β_h`。

### 命题 9.2：实缺额公式

对任意合法状态 `C`，其相对还原态的总实信息缺额为：

```text
Def(C) = [Σ_h u_h(C)] g_bar(n)
```

### 证明

由命题 6.5.1，全部有效量子具有共同实读出大小 `g_bar(n)`。而缺额词只逐位记录某量子是否已转入不可直接读出的复层，因此总缺额就是缺额位数乘以共同量子大小。

### 命题 9.3：一步合法操作至多改变一个有效量子位

对任意相邻状态 `C -> C'`，有：

```text
Σ_h |u_h(C') - u_h(C)| <= 1
```

### 证明

一步 HTM 合法操作在状态图中只对应一条边；在相对转移级数中，它只对应一个一步边界词 `e^(r)`，并由

```text
X_h^(r) = - e^(r)
```

给出一个一步局部自然梯度量子。再由命题 6.3.2 与推论 6.3.3，一步操作的局部自然梯度结构方向完全由该步的投影词决定；又由命题 6.4.7，一步商像在适配有效量子基下只落在单个基方向上。故它至多改变一个有效量子位，而不能同时跨越两个独立量子位。

### 引理 9.3.1：未饱和位对应未闭合缺额头部

若某个合法状态 `C` 满足

```text
u_h(C) = 0
```

则第 `h` 个基方向对应的那一份共同量子 `g_bar(n)` 仍以独立理由头部的形式保留在当前截面中，尚未并入闭合缺额组。

### 证明

由定义 9.1，`u_h(C)=1` 的意义正是第 `h` 个有效量子已经转入不可直接读出的复层；因此 `u_h(C)=0` 恰表示这一份量子尚未完成该转移，仍停留在实层。又由命题 6.4.7，一步商像在适配基下只落在单个基方向上，故第 `h` 个方向上的余量不会与别的方向混成一个不可分辨的共同头部；由命题 6.5.1，这份余量的大小恰是一个共同量子 `g_bar(n)`。

因此，只要 `u_h(C)=0`，就仍有一个尚未闭合的独立缺额头部存在于当前截面。

### 命题 9.4：最小实读出态的饱和性

对任意最小实读出态 `C_min(n)`，有：

```text
u(C_min(n)) = <1,...,1>
```

即最小实读出态在有效量子基的每一个独立方向上都达到缺额饱和。

### 证明

反设存在某个 `h` 使

```text
u_h(C_min(n)) = 0
```

由引理 9.3.1，第 `h` 个方向上仍有一个大小为 `g_bar(n)` 的独立缺额头部尚未闭合。

现在把全部已经转入复层的缺额头部与这个剩余头部合在一起，看成同一组、相关的缺额闭组候选。由 `充足理由单子论.md` 的 `A0` 与 `K4`，一旦这是一个真正闭合的相关闭组，则其总权必须满足：

```text
Full(Pool(<...>)) = 1
```

也就是说，缺额只能在闭组内部守恒地汇齐，不能无载消失。既然第 `h` 个共同量子仍以独立头部存在，它就还没有并入满权闭组，因此当前切面尚未闭合。

若此时仍把 `C_min(n)` 当作最小实读出终点，就等于宣称这份剩余头部已经可以在当前切面被压平，不再产生后续转移；但 `充足理由单子论.md` 的 `R9` 明言：

```text
只有在整体闭合理由截面已拿到且只剩唯一理由头部时，
才允许退化为 Full(P(R_*|E,γ)) = 1；
否则不得在当前切面强行退化。
```

而现在第 `h` 个方向仍保留独立头部，所以 `R9` 的退化条件并未满足，当前切面不得被视为已经封口。

于是这一个共同量子仍必须允许继续转移。再由命题 6.4.6，存在一步 HTM 合法操作，其商像等于 `β_h` 或 `-β_h`；选择使实读出减少的方向作用于 `C_min(n)`，并由命题 6.4.7 与命题 6.5.1，可知这一步恰好只把第 `h` 个方向上的一个共同量子 `g_bar(n)` 转出，而不会同时引入别的基方向的反向恢复。

于是可得到一个合法状态 `C'`，满足：

```text
I_real(C') = I_real(C_min(n)) - g_bar(n)
```

这与 `C_min(n)` 已是最小实读出态矛盾。故每一位都必须等于 `1`。

### 命题 9.4.1：最小态缺额公式

从而：


### 证明

第一式由命题 9.4。第二式由命题 9.2 立得。

### 命题 9.5：任意状态到还原态的最短距离等于缺额位数

对任意合法状态 `C`，有：

```text
dist_n(C, C_0(n)) = Σ_h u_h(C)
```

### 证明

先证下界。由命题 9.3，一步操作至多改变一个有效量子位，因此从 `u(C)` 回到全零词 `u(C_0(n))=<0,...,0>`，至少需要

```text
Σ_h u_h(C)
```

步。

再证上界。由命题 6.4.6，对每个缺额位 `u_h(C)=1`，都存在一步 HTM 合法操作，其商像等于 `-β_h`。又由命题 6.4.7，这一步在适配基下只作用于第 `h` 个基方向，因此恰好把该位恢复为 `0`，而不必同时跨过别的基位。依次作用于全部缺额位，便得到一条长度为

```text
Σ_h u_h(C)
```

的合法恢复链。故上界成立。

于是：

```text
dist_n(C, C_0(n)) = Σ_h u_h(C)
```

### 定理 9.6：最小实读出态必然是最难还原态

### 证明

对任意合法状态 `C`，其缺额位数满足：

```text

```

由命题 9.5，


```text

```

因此最小实读出态恰达到最大最短距离，故它就是最难还原态。

```text
1. 贡献同一性只由结构决定，不由名字决定。
2. 魔方表面拓扑类型只有 面内 / 棱上 / 角顶 三类。
3. 层号在合法操作下保持，因此魔方可按层剥离为 2 阶核或 3 阶核再外扩。
```
